Inspection Procedure 6: AS&G System Abnormal (Auto Start Not Carried Out) <A/T>
 
 
cautionIf the following motions are carried out during the auto stop even one time, the AS&G-ECU blinks the AS&G display screen in the combination meter and prohibits the auto start. Even if trying to return to the previous status after that, the auto start is prohibited. When restarted, the engine must be started by the key.
  • In case the hood is opened
 
 
FUNCTION
When the driver depresses the brake pedal during the auto stop, the AS&G-ECU should send the engine start request signal to the engine-ECU via CAN bus line. When receiving the signal, the engine-ECU should start the engine.
 
 
COMMENTS ON TROUBLE SYMPTOM
The failure could possibly be caused by the failed AS&G system. The AS&G-ECU, however, can possibly prohibit the auto start, depending on the vehicle status or the ambient conditions.
 
 
PROBABLE CAUSES
  • Failed AS&G-ECU system.
  • Failed diesel fuel system.
  • Failed AS&G-ECU.
 
 
DIAGNOSIS PROCEDURE
STEP 1. Confirm the vehicle status and the ambient conditions
Confirm the vehicle status or the ambient conditions to check whether the auto start should be carried out or not (Refer to ).
Are the conditions for prohibiting the auto start satisfied?
Check end.
Go to Step 2.
 
STEP 2. M.U.T.-III diagnosis code
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Use M.U.T.-III to check if the diagnosis code is set.
Is any diagnosis code set?
Refer to Diagnosis Code Chart .
Go to Step 3
 
STEP 3. Check for diagnosis code of other system
Use M.U.T.-III to check if the diagnosis code is set in the diesel fuel system.
Is any diagnosis code set in the diesel fuel system?
Perform the troubleshooting of the diesel fuel system (Refer to GROUP 13A - Troubleshooting, Inspection Chart for Diagnosis Code ).
Go to Step 4.
 
STEP 4. M.U.T.-III data list
Use M.U.T.-III to check the data list (Refer to ).
  • Item No. 4: Engine Hood
OK:
  • Close (Hood: Closed)
  • Open (Hood: Opened)
noteIf the hood is opened, the AS&G display screen in the combination meter should blink.
Is the check result normal?
Go to Step 5.
Perform the troubleshooting of the hood latch switch circuit (Refer to GROUP 54A - ETACS, Troubleshooting, Input Signal Procedures, Inspection Procedure 1: The hood latch switch signal is not received ).
 
STEP 5. Check the trouble symptom
Does the trouble symptom persist?
Replace the AS&G-ECU.
Intermittent malfunction (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
