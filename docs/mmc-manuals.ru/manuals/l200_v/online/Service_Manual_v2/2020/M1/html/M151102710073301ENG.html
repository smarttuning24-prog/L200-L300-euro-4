51 -EXTERIOR > OUTSIDE MIRROR > TROUBLESHOOTING > VEHICLE SPEED-DEPENDENT UNFOLDING FUNCTION DOES NOT WORK NORMALLY
Inspection Procedure 4: Vehicle speed-dependent unfolding function does not work normally.
 
 
cautionWhenever the ECU is replaced, ensure that the input and output circuits are normal.
 
 
 
 
OPERATION
The ETACS-ECU operates this function in accordance with the input signals below.
  • Vehicle speed signal (engine-ECU)
 
 
COMMENTS ON TROUBLE SYMPTOM
If this function does not work normally, these input signal circuit(s) or the ETACS-ECU may be defective. Also, with a customisation function, the door mirrors may be set to a mode in which the vehicle speed-dependent unfolding function is not activated.
 
 
PROBABLE CAUSES
  • Faulty vehicle speed signal
  • The CAN bus line is defective.
  • The ETACS-ECU may be defective
 
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check the ETACS customisation function.
Use the ETACS customisation function to confirm that "Automatic fold door mirror" is set to "Open Vehicle SPD".
Is the check result normal?
Go to Step 2.
Using the ETACS customisation function, set "Automatic fold door mirror" to "Open Vehicle SPD". (Refer to .)
 
STEP 2. M.U.T.-III CAN bus diagnostics.
Use the M.U.T.-III to diagnose the CAN bus lines.
Is the CAN bus normal?
Go to Step 3.
Repair the CAN bus lines. (Refer to GROUP 54C, Troubleshooting - CAN Bus Diagnostics Table ).
 
STEP 3. M.U.T.-III diagnosis code.
Check if an ETACS-related diagnosis code is set.
Is the diagnosis code set?
Diagnose the ETACS (Refer to GROUP 54A, ETACS - Diagnosis Code Chart ).
Go to Step 4.
 
STEP 4. M.U.T.-III diagnosis code for other systems
Check if a diagnosis code is set in the engine-ECU.
Is the diagnosis code set?
Carry out troubleshooting for the engine-ECU (Refer to GROUP 13A, Troubleshooting - Diagnosis Code Chart ).
Go to Step 5.
 
STEP 5. Check the trouble symptom.
Check that the vehicle speed-dependent unfolding function of the door mirrors works normally.
Is the check result normal?
Intermittent malfunction. Refer to GROUP 00, How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunctions ).
Replace the ETACS-ECU (Refer to GROUP 54A, ETACS - ETACS-ECU ).
