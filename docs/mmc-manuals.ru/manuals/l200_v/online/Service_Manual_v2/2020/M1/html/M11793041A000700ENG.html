ACTION TO BE TAKEN WHEN UREA SCR SYSTEM RELATED PARTS ARE REPLACED
2020 / L200/L200 SPORTERO / EU - Workshop Manual
17 -ENGINE AND EMISSION CONTROL > EMISSION CONTROL <DIESEL> > UREA SCR SYSTEM<4N14> > ACTION TO BE TAKEN WHEN UREA SCR SYSTEM RELATED PARTS ARE REPLACED
 
 
After urea SCR system related parts are replaced, execute each command of the M.U.T.-III as shown in the table below.

note
  • *1: The urea tank level can be reset without using the M.U.T.-III by leaving the vehicle on a level surface with the ignition switch to the "ON" position for approximately 5 minutes. However, if the amount of change in the urea tank level is less than 3 litres, the urea tank level needs to be reset by using the M.U.T.-III.
  • *2: Only when AdBlue™ is completely drained from the urea tank, execute the urea SCR system test 3 times.
 
 
 
Implementation Method
 
 
DCU Learning Value Writing
If the DCU is replaced, follow the procedure below to save the values stored in the old DCU and then write them to the new DCU.
1.
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Connect the M.U.T.-III to the diagnosis connector.
2. Turn the ignition switch to the "ON" position.
3. Select "DCU" from System select Screen of the M.U.T.-III.
4. From DCU Screen, select "Special Function".
5. Select "Write learned value(Changed ECU)" from Special Function Screen.
6. Select "Learned value Read&save" from Write learned value Screen.
7. Press "OK" to store the initial learning value file.
note
  • When the learning value file is appropriately stored, "Learned value Save Complete" is shown on the screen of the M.U.T.-III.
  • The file is named as "LND + date + time" which is based on the time when the file is imported to a personal computer.
8. After turning the ignition switch to the "LOCK" (OFF) position, replace the DCU (Refer to ).
9. Turn the ignition switch to the "ON" position.
10. Select "DCU" from System Select Screen of the M.U.T.-III.
11. From DCU Screen, select "Special Function".
12. Select "Write learned value(Changed ECU)" from Special Function Screen.
13. Select "Saved learned value writing" from Write learned value Screen.
14. Select the initial learning value file stored during Step 7 to write the initial learning value.
note
  • Use only the initial learning value file stored during Step 7 without using any other files.
  • Make sure that the selected file is correct by checking the file name (saved date), then carry out the procedure.
  • When the initial learning value is appropriately written, "Learned value writing Completed." is shown on the screen of the M.U.T.-III.
15. After turning the ignition switch to the "LOCK" (OFF) position, remove the M.U.T.-III.
 
 
 
Urea SCR System Test
1.
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Connect the M.U.T.-III to the diagnosis connector.
2. Turn the ignition switch to the "ON" position (but do not start the engine).
3. Select "DCU" from System Select Screen of the M.U.T.-III.
4. From DCU Screen, select "Special Function".
5. From Special Function Screen, select "Test".
6. Carry out "Item No. 5 SCR system test" from Test Screen of the M.U.T.-III when all the conditions below are satisfied.
  • Data list item No. 3: SCR catalyst temperature is 80°C or less.
  • Data list item No. 5: SCR state machine is not ''Purge''.
  • Data list item No. 7: Urea tank temperature is between 0°C and 50°C.
  • Data list item No. 8: Urea tank level is 42.25 mm or more.
noteIf the test is interrupted, check whether a diagnosis code is set (Refer to ).
7. After turning the ignition switch to the "LOCK" (OFF) position, remove the M.U.T.-III.
 
 
 
Reset (Urea Tank Level)
1.
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Connect the M.U.T.-III to the diagnosis connector.
2. Turn the ignition switch to the "ON" position (but do not start the engine).
3. Select "DCU" from System Select Screen of the M.U.T.-III.
4. From DCU Screen, select "Special Function".
5. From Special Function Screen, select "Reset".
6. Carry out "Item No. 1 Urea tank level" from Reset Screen of the M.U.T.-III.
7. After turning the ignition switch to the "LOCK" (OFF) position, remove the M.U.T.-III.
 
 
 
Initialisation (Each Item)
1.
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Connect the M.U.T.-III to the diagnosis connector.
2. Turn the ignition switch to the "ON" position (but do not start the engine).
3. Select "DCU" from System Select Screen of the M.U.T.-III.
4. From DCU Screen, select "Special Function".
5. From Special Function Screen, select "Initialization".
6. Select the item you want to initialise from Initialization Screen of the M.U.T.-III, and execute it.
7. After turning the ignition switch to the "LOCK" (OFF) position, remove the M.U.T.-III.
