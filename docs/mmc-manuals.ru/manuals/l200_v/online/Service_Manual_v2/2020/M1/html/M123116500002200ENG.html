Code No. P277C: Electro-magnetic oil pump system (Stuck OFF) <Vehicles with AS&G system>
 
 
 
 
OPERATION
The A/T-ECU operates the electro-magnetic oil pump during idling stop to avoid the fluid pressure reduction.
 
 
DIAGNOSIS CODE SET CONDITIONS
When the gear engagement detection condition is met, the diagnosis code No. 277C is set.
Check Condition <Engine flare detection during auto start>
  • Engine is running.
  • Vehicle is driven with the selector lever in the "D" position.
  • Output shaft speed is less than 200 r/min.
  • Input shaft speed sensor is normal.
  • Output shaft speed sensor is normal.
  • Electro-magnetic oil pump system is normal.
Judgement Criterion <Engine flare detection during auto start>
  • The difference between the input shaft speed and output shaft speed, which is multiplied by the 1st gear ratio, is more than 350 r/min.
Check Condition <Gear engagement detection>
  • After the engine flare detection during auto start
  • Input shaft speed sensor is normal.
  • Output shaft speed sensor is normal.
  • Output shaft speed is less than 200 r/min.
  • Not during gear shifting
Judgement Criterion <Gear engagement detection>
  • Abnormal gear ratio is detected.
 
 
PROBABLE CAUSES
  • Malfunction of the valve body assembly (electro-magnetic oil pump)
  • Malfunction of the transmission assembly [clutch No. 1 (C1)]
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check whether the other diagnosis code is set.
Is the diagnosis code set?
Troubleshoot the diagnosis code (Refer to ).
Go to Step 2.
 
STEP 2. Electro-magnetic oil pump check
Check the electro-magnetic oil pump (Refer to ).
Is the check result normal?
Go to Step 3.
Replace the valve body assembly.
 
STEP 3. Check whether the diagnosis code is set again.
Check again if the diagnosis code No. P277C is set.
Is the diagnosis code No. P277C set?
Disassemble the transmission and replace the faulty parts [clutch No. 1 (C1)].
Intermittent malfunction (Refer to GROUP 00 - How to Cope with Intermittent Malfunction ).
