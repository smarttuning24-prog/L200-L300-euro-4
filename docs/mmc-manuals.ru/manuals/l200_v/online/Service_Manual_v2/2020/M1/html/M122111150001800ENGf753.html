Code No. C143A: R/D lock ACT.terminal high VOLT.
Code No. C143B: R/D lock ACT.terminal low VOLT.
Code No. C143C: R/D lock ACT.terminal HI current
Code No. C143D: R/D lock ACT.terminal LO current
Code No. C143E: R/D lock ACT.driven circuit
 
 
 
 
OPERATION
The transfer-ECU controls the rear differential lock solenoid based on ON/OFF signal from the rear differential lock switch.
 
 
DIAGNOSIS CODE SET CONDITIONS
  • When power supply voltage rises to 4.9 V or more, the code No.C143A is set as high voltage.
  • When power supply voltage drops to 4.7 V or less or 1.1 V or less, the code No.C143B is set as low voltage.
  • When power supply current rises to 8.0 A or more or 4.0 A or more, the code No.C143C is set as high current.
  • When power supply current drops to 0.3 A or less, the code No.C143D is set as low current.
  • When power supply voltage rises to 4.9 V or more, the electric current monitoring circuit offset value is 1.6 V or more, or 0.8 V or less, diagnosis code No. C143E will be set.
 
 
PROBABLE CAUSES
  • Malfunction of the rear differential carrier assembly
  • Damaged harness wires and connectors
  • Malfunction of the transfer-ECU
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III diagnosis code
Check if diagnosis code is set in the transfer-ECU.
Is the diagnosis code set?
Troubleshoot the transfer-ECU. Refer to .
Go to Step 2.
 
STEP 2. Check of short to power supply, short to earth, and open circuit in DSL+, DSL- line between transfer-ECU and rear differential lock solenoid connector.
Is the check result normal?
Go to Step 3.
Repair the connector(s) or wiring harness.
 
STEP 3. Retest the system.
Check that the rear differential lock solenoid functions work normally.
Is the check result normal?
Go to Step 4.
Replace the differential carrier assembly (Refer to GROUP 27 - Differential carrier assembly ). Then go to Step 4.
 
STEP 4. Retest the system
Check that the rear differential lock solenoid functions work normally.
Is the check result normal?
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to use Troubleshooting/inspection Service Points - How to Cope with Intermittent Malfunction ).
Replace the transfer-ECU (Refer to ).
