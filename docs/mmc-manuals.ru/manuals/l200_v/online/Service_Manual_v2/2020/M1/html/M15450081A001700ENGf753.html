Code No.U1102 ASC/TCL/ABS timeout/Not equipped
 
 
caution
  • If diagnosis code No.U1102 is set in the ETACS-ECU, diagnose the CAN main bus line.
  • Whenever the ECU is replaced, ensure that the input and output signal circuits are normal.
 
 
TROUBLE JUDGEMENT
If signal from the ASTC-ECU cannot be received for 2500 milliseconds or more, the ETACS-ECU sets diagnosis code No. U1102.
 
 
PROBABLE CAUSES
  • Malfunction of the ASTC-ECU
  • Malfunction of the ETACS-ECU
  • Malfunction of the CAN bus
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III CAN bus diagnostics
Use the M.U.T.-III to diagnose the CAN bus lines.
Is the check result normal?
Go to Step 2.
Repair the CAN bus line (Refer to GROUP 54C - Troubleshooting ).
 
STEP 2. Check the M.U.T.-III Diagnosis code for other systems
Check that the ASTC-ECU set a diagnosis code.
Is the diagnosis code set?
Diagnose the ASTC-ECU (Refer to GROUP 35C - Troubleshooting ).
Go to Step 3.
 
STEP 3. Check the M.U.T.-III Diagnosis code for other systems
Check if diagnosis code No. U1102 is set in SRS-ECU meter.
Is the diagnosis code set?
Go to Step 4.
Go to Step 5.
 
STEP 4. Check whether the diagnosis code is set again.
Check again if the diagnosis code is set in the ETACS-ECU.
(1) Erase the diagnosis code.
(2) Turn the ignition switch from "LOCK" (OFF) position to "ON" position.
(3) On completion, check that the diagnosis code is set again.
Is the diagnosis code set?
Replace the ASTC-ECU (Refer to GROUP 35C - Hydraulic Unit ). After replacing the ECU, carry out the CAN bus diagnostics. Then go to Step 6.
A poor connection, open circuit or other intermittent malfunction is present in the CAN bus lines between the ASTC-ECU and the ETACS-ECU (Refer to GROUP 00, How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunctions ).
 
STEP 5. Check whether the diagnosis code is set again.
Check again if the diagnosis code is set in the ETACS-ECU.
(1) Erase the diagnosis code.
(2) Turn the ignition switch from "LOCK" (OFF) position to "ON" position.
(3) On completion, check that the diagnosis code is not set again.
Is the diagnosis code set?
Replace the ETACS-ECU (Refer to ). After replacing the ECU, carry out the CAN bus diagnostics. Then go to Step 6.
A poor connection, open circuit or other intermittent malfunction is present in the CAN bus lines between the ASTC-ECU and the ETACS-ECU (Refer to GROUP 00, How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunctions ).
 
STEP 6. Check whether the diagnosis code is set again.
Recheck if the diagnosis code is set in the ETACS-ECU.
(1) Erase the diagnosis code.
(2) Turn the ignition switch from "LOCK" (OFF) position to "ON" position.
(3) On completion, check that the diagnosis code is set again.
Is the diagnosis code set?
Return to Step 1.
The procedure is complete.
