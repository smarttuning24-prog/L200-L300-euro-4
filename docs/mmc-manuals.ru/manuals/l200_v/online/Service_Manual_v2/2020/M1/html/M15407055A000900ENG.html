Code No. C2230: FR corner sensor signal lost
 
 
 
 
OPERATION
The clearance warning control unit sets diagnosis code No. C2230 when it does not receive any signal from the front corner clearance sensor (RH) for at least 3 seconds.
note
  • If some or all of diagnosis code No. C2230 to C2235 are set together with this DTC, the power supply or earth wires for the sensor may be open-circuited.
  • Diagnosis code No. C2242 will be set instead of this DTC when the sensor power supply circuit is short to power supply or earth.
 
 
TROUBLE JUDGEMENT
The condition below is satisfied:
  • The front corner clearance sensor (RH) does not receive any signal for at least 3 seconds.
 
 
PROBABLE CAUSES
  • Malfunction of front corner clearance sensor (RH)
  • Malfunction of clearance warning control unit
  • Damaged harness wires and connectors
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check of short to earth, and open circuit in POUS, SFOR and GOUS line between front corner clearance sensor (RH) connector and clearance warning control unit connector
noteTo check electric continuity, always select a resistance range of a multimeter.
Is the check result normal?
Go to Step 2.
Repair the connector(s) or wiring harness.
 
STEP 2. Check the front corner clearance sensor (RH).
Substitute a known good front side sensor.
(1) Substitute a known good front side sensor.
(2) Erase the diagnosis code.
(3) Turn the ignition switch from "LOCK" (OFF) position to "ON" position.
(4) The diagnosis code representing the substituted sensor should be set, but diagnosis code No. C2230 should not be set.
Is the check result normal?
Replace the front corner clearance sensor (RH) (Refer to ).
Go to Step 3.
 
STEP 3. Check whether the diagnosis code is set again.
Check again if the diagnosis code is set in the clearance warning control unit.
(1) Erase the diagnosis code.
(2) Turn the ignition switch from "LOCK" (OFF) position to "ON" position.
(3) Check if diagnosis code is set.
Is the diagnosis code set?
Replace the clearance warning control unit (Refer to ).
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to Cope with Intermittent Malfunction ).
