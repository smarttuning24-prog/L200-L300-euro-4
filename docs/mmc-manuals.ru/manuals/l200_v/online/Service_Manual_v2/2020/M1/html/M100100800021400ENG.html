WIRE COLOUR CODE
 
 
Wire colours are identified by the following colour codes.
CodeWire colourCodeWire colourCodeWire colourCodeWire colour
BBlackGRGreyPPinkSISilver
BEBeigeLBluePUPurpleVViolet
BRBrownLGLight greenRRedWWhite
GGreenOOrangeSBSky blueYYellow
If a cable has two colours, the first of the two colour code characters indicates the basic colour (colour of the cable coating) and the second indicates the marking colour.
 
No.Meaning
1<F>: Flexible wire
<T>: Twisted wire
2Wire size (mm2)*

note
*: No code indicates 0.5 mm2. Cable colour code in parentheses indicates 0.3 mm2.
