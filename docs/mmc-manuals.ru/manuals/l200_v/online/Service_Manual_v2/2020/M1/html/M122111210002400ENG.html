Code No. C1469: R/D lock engage SW
 
 
 
 
OPERATION
When the rear differential lock is activated, the rear differential lock detection switch is turned to ON, connecting the transfer ECU to the earth. With this operation, the transfer ECU detects the rear differential status at the moment.
 
 
DIAGNOSIS CODE SET CONDITIONS
When the OFF signal from the rear differential lock detection switch is input for 15 minutes continuously with the rear differential lock activated, or when the ON signal from the rear differential lock detection switch is input for 15 minutes continuously with the rear differential lock deactivated, diagnosis code No. C1469 is set.
 
 
PROBABLE CAUSES
  • Malfunction of the rear differential carrier assembly
  • Damaged harness wires and connectors
  • Malfunction of the transfer-ECU
