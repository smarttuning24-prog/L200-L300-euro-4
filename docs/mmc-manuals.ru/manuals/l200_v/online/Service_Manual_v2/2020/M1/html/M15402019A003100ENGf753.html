CHECK PROCEDURE FOR EACH MULTI-INFORMATION DISPLAY SCREEN <COLOUR LIQUID CRYSTAL DISPLAY TYPE>
 
 
cautionWhen there are TV towers, substations, or broadcasting stations which emit strong radio waves in proximity, on rare occasions, a warning is displayed on the multi-information display screen for a few seconds. This is caused by the reception of strong radio waves, and there is no functional problem.
 
 
WARNING SCREEN
When malfunctions occur to the vehicle, the following warning screens are displayed. If these screens are not displayed normally or if they continue to be displayed even after the factor is eliminated, take measures according to the action procedure.
ENGINE
Display contentMessageFactorAction procedure
LOW OIL PRESSUREDisplayed when there is a malfunction to the engine oil circulation system.If the warning screen is not displayed normally or if the screen continues to be displayed, check the oil pressure (Refer to GROUP 12 - Oil Pressure Check <4N14> or <4N15>). Then if the oil pressure is normal, perform the troubleshooting for the engine (Refer to GROUP 13A - Trouble Symptom Chart ).
ENGINE OVERHEATING STOP SAFELYDisplayed when overheated.If the warning screen is not displayed normally or if the screen continues to be displayed, perform the troubleshooting for the engine (Refer to GROUP 13A - Trouble Symptom Chart )).
REMOVE WATER IN SEPARATORDisplayed when the water that exceeds the permissible limit is stored in the fuel filter.If the warning screen is not displayed normally or if the screen continues to be displayed, drain water from the fuel filter (Refer to GROUP 13A - On-vehicle Service ).
FUEL
Display contentMessageFactorAction procedure
FUEL SYSTEM SERVICE REQUIREDDisplayed when there is a malfunction to the fuel system.If the warning screen is not displayed normally or if the screen continues to be displayed, check the combination meter diagnosis code.
REFUELDisplayed when the remaining fuel amount is small.If the warning screen is not displayed normally or if the screen continues to be displayed, refuel immediately.
UREA SCR SYSTEM
Display contentMessageFactorAction procedure
AdBlue SYSTEM SERVICE REQUIRED NO RESTART IN 600kmDisplayed when there is a malfunction to the Urea SCR system.If the warning screen is not displayed normally or if the screen continues to be displayed, troubleshoot the Urea SCR system.
AdBlue SYSTEM SERVICE REQUIRED NO RESTARTDisplayed when the vehicle is driven a certain distance with a malfunction to the Urea SCR system occurring.
LOW AdBlue REFILL AdBlueDisplayed when AdBlue™ remaining amount is low.If the warning screen is not displayed normally or if the screen continues to be displayed, refill AdBlue™ immediately.
ENGINE WILL NOT RESTART REFILL AdBlueDisplayed when AdBlue™ remaining amount is quite low.
ENGINE CAN NOT RESTART REFILL AdBlueDisplayed when there is no remaining AdBlue™.
noteAdBlue™: trademark of the Verband der Automobilindustrie e.V. (VDA).

4WD
Display contentMessageFactorAction procedure
TRANSFER 4HLc<=>4LLc NOT AVAILABLEThere is a fault in the A/T. Shifting between a "2H" <=> "4H" <=> "4HLc" selection can be made, but when the drive mode-selector is in "4LLc" the transfer mode cannot be switched.If the warning screen is not displayed normally or if the screen continues to be displayed, check the 4WD diagnosis code (Refer to GROUP 22A - Diagnosis Code Chart <M/T> or GROUP 23C - Diagnosis Code Chart <A/T>).
TRANSFER 4H<=>4L NOT AVAILABLEThere is a fault in the A/T. Shifting between a "2H" <=> "4H" selection can be made, but when the drive mode-selector is in "4L" the transfer mode cannot be switched.
4WD SYSTEM SERVICE REQUIREDDisplayed when there is a malfunction to the electronically-controlled 4WD system.
PRESS BRAKE PEDAL SHIFT TO N POSITIONThe selector lever is shifted from the "N"(NEUTRAL) position to the "D"(DRIVE) position during shifting process between "4HLc" <=> "4LLc" <super select 4WD> or "4H" <=> "4L" <easy select 4WD>.
SPEED LIMITER WILL ACTIVATEDisplayed just before the speed limitation control is activated when the vehicle is driven at the specified speed or more in the transfer mode "4HLc" <super select 4WD> or "4H" <easy select 4WD>.

Display contentMessageFactorAction procedure

NOT AVAILABLE WITH 2H/4HThe Off Road mode will not be available even if the Off Road mode selector is pressed because the drive mode-selector is in "2H" or "4H".

HDC NOT AVAILABLE WITH R/D LOCKThe HDC cannot be set to ON (stand by) because the rear differential lock is ON.
HDC NOT AVAILABLE SHIFT TO 1st GEARDisplayed when the HDC switch is operated while the gear position is not within the operational range for HDC.
