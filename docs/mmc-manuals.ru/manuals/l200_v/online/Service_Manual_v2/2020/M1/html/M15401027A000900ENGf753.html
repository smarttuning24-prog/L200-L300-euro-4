Inspection Procedure 6: The headlamp automatic-shutdown function does not work normally.
 
 
cautionWhenever the ECU is replaced, ensure that the input and output signal circuits are normal.
 
 
COMMENTS ON TROUBLE SYMPTOM
The ETACS-ECU operates this function in accordance with the input signals below.
  • Ignition switch (IG1)
  • Front door switch (driver’s side)
  • Tail lamp switch
  • Headlamp switch
If this function does not work normally, these input signal circuit(s) or the ETACS-ECU may be defective.
 
 
PROBABLE CAUSES
  • Malfunction of the front door switch (driver’s side)
  • Malfunction of the column switch
  • Malfunction of the ETACS-ECU
  • Damaged harness wires and connectors
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check of adjustment details for the headlamp automatic-shutdown
Check that the headlamp automatic-shutdown function is set to "B-spec" using the customise function of M.U.T.-III.
Is the check result normal?
Go to Step 2.
Using the customise function of M.U.T.-III, set the headlamp automatic-shutdown function to "B-spec."
 
STEP 2. M.U.T.-III diagnosis code.
When the ignition switch is turned to the "LOCK" (OFF) position, check that the ETACS-ECU does not set the diagnosis code.
Is the diagnosis code set?
Refer to diagnosis code chart .
Go to Step 3.
 
STEP 3. M.U.T.-III data list
Check the input signals below, which are related to the headlamp automatic-shutdown function.
  • Turn the ignition switch to the "LOCK" (OFF) position.
  • Open the driver’s door.
  • Turn the headlamp switch to the ON position.
 
Item No.Item nameNormal condition
19Driver’s door switchON
30Ignition switch(IG1)OFF
60Headlamp outputON
OK: Normal conditions are displayed for all the items
Are the check result normal?
YES <All the signals are received normally.> : Go to Step 4.
NO <The driver's door switch signal is not received.> : Refer to inspection procedure 5 "The front door switch (driver’s side) signals is not received ."
NO <The ignition switch (IG1) signal is not received.> : Refer to inspection procedure 2 "The ignition switch (IG1) signal is not received ."
NO <The headlamp switch signal is not received.> : Refer to inspection procedure 9 "The column switch signal is not received ."
 
STEP 4. Retest the system.
Check that the headlamp automatic-shutdown function works normally.
Is the check result normal?
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to Cope with Intermittent Malfunction ).
Replace the ETACS-ECU (Refer to ETACS-ECU ).
