What the FCM/LDW/AHB-ECU monitors
  • By monitoring these items, the system recognises how the FCM is controlled.
  • Monitored data can be stored 3 times; Event 1 (the last), Event 2 (the last but one) and Event 3 (the last but two). To store the forth data, oldest Even 3 will be erased.
