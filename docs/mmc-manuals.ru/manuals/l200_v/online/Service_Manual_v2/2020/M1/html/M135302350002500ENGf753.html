Code No. C1297: Neutral position switch circuit <Vehicles with HDC function-M/T>
 
 
caution
  • If there is any problem in the CAN bus lines, an incorrect diagnosis code may be set. Prior to this diagnosis, diagnose the CAN bus lines (Refer to GROUP 54C - CAN Bus Diagnostics Table ).
  • Whenever ECU is replaced, ensure that the CAN bus lines are normal.
  • When the hydraulic unit (integrated with ASTC-ECU) is replaced, always carry out the calibration of the steering wheel sensor and the G and yaw rate sensor (Refer to ).
 
 
OPERATION
ASTC-ECU monitors whether the neutral position switch is ON or OFF.
 
 
DIAGNOSIS CODE SET CONDITIONS
This diagnosis code is set when the neutral position switch signal is assumed to be stuck in ON condition.
 
 
PROBABLE CAUSES
  • Malfunction of the neutral position switch
  • The neutral position switch improperly installed
  • Damaged wiring harness or connector
  • Malfunction of the hydraulic unit assembly (integrated with ASTC-ECU)
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III CAN bus diagnosis
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Use the M.U.T.-III to diagnose the CAN bus lines.
Is the check result normal?
Go to Step 3.
Repair the CAN bus lines (Refer to GROUP 54C - CAN Bus Diagnostics Table ). Then go to Step 2.
 
STEP 2. Diagnosis code recheck after erasing diagnosis code
Check again if the diagnosis code is set.
(1) Turn the ignition switch to "LOCK" (OFF) position.
(2) Move the shift lever to a position other than neutral.
(3) Turn the ignition switch to "ON" position.
(4) Use M.U.T.-III to erase the diagnosis code.
(5) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. C1297 set?
Go to Step 3.
This diagnosis is complete.
 
STEP 3. M.U.T.-III data list
Use M.U.T.-III to check the data list (Refer to Data List Reference Table ).
  • Item No. 48: Neutral SW state (Raw value)
OK:
  • ON (Transmission: Neutral)
  • OFF (Transmission: Other than neutral)
Is the check result normal?
Go to Step 7.
Go to Step 4.
 
STEP 4. Check for neutral position switch installation
Check the neutral position switch installation.
Is the check result normal?
Go to Step 5.
Install the neutral position switch correctly (Refer to GROUP 22C - Transmission, Transmission Disassembly and Reassembly ). Then go to Step 8 .
 
STEP 5. Check of short circuit to power supply in N line between ASTC-ECU connector and neutral position switch connector
Is the check result normal?
Go to Step 6.
Repair the connector(s) or wiring harness. Then go to Step 8.
 
STEP 6. Check Neutral position switch itself
Check neutral position switch itself (Refer to GROUP 22C - Transmission, Transmission Inspection, Neutral Position Switch Check ).
Is the check result normal?
Go to Step 7.
Replace the neutral position switch (Refer to GROUP 22C - Transmission, Transmission Disassembly and Reassembly ). Then go to Step 8 .
 
STEP 7. Diagnosis code recheck after erasing diagnosis code
Check again if the diagnosis code is set.
(1) Turn the ignition switch to "LOCK" (OFF) position.
(2) Move the shift lever to a position other than neutral.
(3) Turn the ignition switch to "ON" position.
(4) Use M.U.T.-III to erase the diagnosis code.
(5) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. C1297 set?
Replace the hydraulic unit assembly (integrated with ASTC-ECU) (Refer to ). Then go to Step 8.
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to Cope with Intermittent Malfunction ).
 
STEP 8. Diagnosis code recheck after erasing diagnosis code
Check again if the diagnosis code is set.
(1) Turn the ignition switch to "LOCK" (OFF) position.
(2) Move the shift lever to a position other than neutral.
(3) Turn the ignition switch to "ON" position.
(4) Use M.U.T.-III to erase the diagnosis code.
(5) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. C1297 set?
Return to Step 1.
The procedure is complete.
