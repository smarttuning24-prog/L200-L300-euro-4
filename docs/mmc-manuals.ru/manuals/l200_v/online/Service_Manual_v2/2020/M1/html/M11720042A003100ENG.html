17 -ENGINE AND EMISSION CONTROL > CRUISE CONTROL > SYMPTOM PROCEDURES > CRUISE CONTROL CANNOT BE SET.
Inspection Procedure 5: Cruise Control cannot be Set (No Response SET - Switch and RES + Switch is Pressed)
 
 
 
 
COMMENTS ON TROUBLE SYMPTOM
The fail-safe function is probably cancelling cruise control system. In this case, checking the cruise control system, diesel fuel system, ASTC system, CAN bus system and A/T system diagnosis codes. The M.U.T.-III can also be used to check if the circuits of each input switch are normal or not by checking the data list.
notePress the cruise control switches one by one securely. Otherwise, the cruise control system may not be started.
 
 
PROBABLE CAUSES
  • Malfunction of CAN bus system.
  • Malfunction of the diesel fuel system.
  • Malfunction of the ASTC system.
  • Malfunction of the A/T system <A/T>.
  • Malfunction of the cruise control switches.
  • Malfunction of the stop lamp switch.
  • Malfunction of the clutch switch <M/T>.
  • Malfunction of the inhibitor switch <A/T>.
  • Malfunction of the engine-ECU.
 
 
DIAGNOSIS PROCEDURE
caution
  • If there is any problem in the CAN bus lines, an incorrect diagnosis code may be set. Prior to this diagnosis, diagnose the CAN bus lines.
  • Before replacing the ECU, ensure that the CAN bus lines is normal.
STEP 1. Check the CAN bus system diagnosis
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Use M.U.T.-III to perform CAN bus diagnosis (Refer to GROUP 54C - Explanation About The M.U.T.-III Can Bus Diagnostics ).
Is the check result satisfactory?
Go to Step 2.
Repair the CAN bus lines (Refer to GROUP 54C - Troubleshooting, Can Bus Diagnostic Table ). Then go to Step 11.
 
STEP 2. Check the diesel fuel system diagnosis code
Use M.U.T.-III to check diesel fuel system diagnosis code (Refer to GROUP 13A - Troubleshooting, Diagnosis Function ).
Is any diagnosis code set?
Repair the diesel fuel system (Refer to GROUP 13A - Troubleshooting, Inspection Chart for Diagnosis Trouble Code ). Then go to Step 11 .
Go to Step 3 <A/T>
Go to Step 4<M/T>.
 
STEP 3. Check the A/T system diagnosis code <A/T>
Use M.U.T.-III to check A/T system diagnosis code (Refer to GROUP 23C - Troubleshooting, Diagnosis Function ).
Is any diagnosis code set?
Repair the A/T system (Refer to GROUP 23C - Troubleshooting, Diagnosis Code Chart ). Then go to Step 11.
Go to Step 4.
 
STEP 4. Check the ASTC system diagnosis code
Use M.U.T.-III to check ASTC system diagnosis code (Refer to GROUP 35C - Troubleshooting, Diagnosis Function ).
Is any diagnosis code set?
Repair the ASTC system (Refer to GROUP 35C - Troubleshooting, Diagnosis Code Chart ). Then go to Step 11.
Go to Step 5.
 
STEP 5. Check the diagnosis code
(1) Use the M.U.T.-III, erase the diagnosis code of the cruise control system.
(2) Turn the ignition switch to the "ON" position, and press the cruise control ON/OFF switch to turn the cruise control system to ON (turn ON the cruise control indicator lamp).
(3) After turning the cruise control system to ON, when 2 minutes or more has elapsed without operating the cruise control switches.
(4) With the cruise control switches not operated, depress the brake pedal for several seconds, and then read the diagnosis code of the cruise control system.
Is any diagnosis code set?
Repair the cruise control system (Refer to ). Then go to Step 11.
Go to Step 6.
 
STEP 6. M.U.T.-III data list
Use M.U.T.-III to check the data list (Refer to ).
  • Item No. 136: Cruise main switch (Cruise control ON/OFF switch)
  • Item No. 137: Cruise cancel switch (CANCEL switch)
  • Item No. 138: Cruise set switch (SET - switch)
  • Item No. 139: Cruise resume switch (RES + switch)
Is the check result normal?
Go to Step 7.
Check the cruise control switch system (Refer to Diagnosis Code Procedures, Code No. P1564: Abnormal voltage at cruise control switch <4N14> or <4N15>). Then go to Step 11.
 
STEP 7. M.U.T.-III data list
Use M.U.T.-III to check the data list (Refer to ).
  • Item No. 134: Normally closed brake switch.
  • Item No. 135: Brake lamp switch.
Is the check result normal?
Go to Step 8 <M/T>
Go to Step 9 <A/T>
Refer to Diagnosis Code Procedures, Code No.P1571: Stop lamp switch system . Then go to Step 11.
 
STEP 8. M.U.T.-III data list <M/T>
Use M.U.T.-III to check the data list (Refer to ).
  • Item No. 131: MT clutch switch.
Is the check result normal?
Go to Step 10.
Refer to Symptom Procedures, Inspection Procedure 2 . Then go to Step 11.
 
STEP 9. M.U.T.-III data list <A/T>
Use M.U.T.-III to check the data list (Refer to ).
  • Item No. 42: Neutral switch
Is the check result normal?
Go to Step 10.
Check the inhibitor switch system [Refer to GROUP 23C - Troubleshooting, Diagnosis Code Procedures Code No. P0707: Inhibitor Switch System (Open Circuit) , Code No. P0708: Inhibitor Switch System (Short Circuit) ]. Then go to Step 11.
 
STEP 10. Check the symptom
Can cruise control be set?
It can be assumed that this malfunction is intermittent (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunction ).
Replace the engine-ECU (Refer to GROUP 13A - Engine-ECU <4N14> or <4N15>). Then go to Step 11.
 
STEP 11. Check the symptom
Can cruise control be set?
The procedure is complete.
Return to Step 1.
