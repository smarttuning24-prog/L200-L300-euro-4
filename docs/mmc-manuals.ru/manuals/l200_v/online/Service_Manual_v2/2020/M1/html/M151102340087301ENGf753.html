Inspection Procedure 5: Ignition-linked fold/unfold function does not work normally.
 
 
cautionWhenever the ECU is replaced, ensure that the input and output circuits are normal.
 
 
 
 
OPERATION
The ETACS-ECU operates this function in accordance with the input signals below.
  • Ignition switch (IG1)
  • Driver's door switch
 
 
COMMENTS ON TROUBLE SYMPTOM
If this function does not work normally, these input signal circuit(s) or the ETACS-ECU may be defective. Also, with the customisation function, the door mirrors may be set to a mode in which the ignition-linked folding/unfolding function is not activated.
 
 
PROBABLE CAUSES
  • The driver's door switch may be defective
  • The ETACS-ECU may be defective
  • Damaged harness wires and connectors
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check the ETACS customisation function.
By the ETACS customisation function, confirm that "Automatic fold door mirror" is set to "Open/Close by IG".
Is the check result normal?
Go to Step 2.
By the ETACS customisation function, set "Automatic fold door mirror" to "Open/Close by IG" (Refer to ).
 
STEP 2. M.U.T.-III data list
Check the input signals which are related to the ignition-linked folding/unfolding function.
  • Driver's door switch: ON
  • Ignition switch: ON

Check that the ignition-linked folding/unfolding function of door mirrors works normally.
