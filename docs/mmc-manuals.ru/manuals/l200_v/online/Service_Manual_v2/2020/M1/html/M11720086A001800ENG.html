Inspection Procedure 10: When The Speed Limiter Switch is Operated, The System is not Turned ON <Vehicles with Speed Limiter>
 
 
 
 
OPERATION
This circuit judges the signals of each switches (speed limiter ON/OFF, cruise control ON/OFF, CANCEL, SET - and RES +) of the cruise control switches. The engine-ECU detects the state of the cruise control switches by sensing the voltages shown below.
  • When all switches are released: 4.7 to 5.0 V
  • When the speed limiter ON/OFF switch is pressed: 0 to 0.5 V <Vehicles with speed limiter>
  • When the cruise control ON/OFF switch is pressed: 0.9 to 1.7 V
  • When the CANCEL switch is pressed: 1.9 to 2.7 V
  • When the SET - switch is pressed: 2.9 to 3.6 V
  • When the RES + switch is pressed: 3.8 to 4.5 V
 
 
COMMENTS ON TROUBLE SYMPTOM
The M.U.T.-III can also be used to check if the circuits of each input switches are normal or not by checking the data list.
notePress the cruise control switches one by one securely. Otherwise, the cruise control system may not be started.
 
 
PROBABLE CAUSES
  • Damaged harness or connector.
  • Malfunction of the cruise control switches.
  • Malfunction of the clock spring.
  • Malfunction of the engine-ECU.
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check the diesel fuel system diagnosis code
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Use M.U.T.-III to check diesel fuel system diagnosis code (Refer to GROUP 13A - Troubleshooting, Diagnosis Function ).
Is any diagnosis code set?
Repair the diesel fuel system (Refer to GROUP 13A - Troubleshooting, Inspection Chart for Diagnosis Code ). Then go to Step 5.
Go to Step 2.
 
STEP 2. M.U.T.-III data list
Use M.U.T.-III to check the data list (Refer to ).
  • Item No. 136: Cruise main switch (Cruise control ON/OFF switch)
  • Item No. 137: Cruise cancel switch (CANCEL switch)
  • Item No. 138: Cruise set switch (SET - switch)
  • Item No. 139: Cruise resume switch (RES + switch)
  • Item No. 460: ASLF main switch (Speed limiter ON/OFF switch)
Is the check result normal?
Go to Step 4.
Go to Step 3.
 
STEP 3. Check the cruise control switches
Refer to .
Is the check result normal?
Go to Step 4.
Replace the cruise control switches (Refer to ). Then go to Step 5.
 
STEP 4. Check the symptom
Is the system turned on by operating the speed limiter switch?
It can be assumed that this malfunction is intermittent (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunction ).
Replace the engine-ECU (Refer to GROUP 13A - Engine-ECU ). Then go to Step 5.
 
STEP 5. Check the symptom
Is the system turned on by operating the speed limiter switch?
The procedure is complete.
Return to Step 1.
