REMOVAL SERVICE POINT
 
 
<<A>> FUEL FILTER CAP REMOVAL
1. Insert a flat-tipped screwdriver into the groove of the fastener nut.
2. Loosen the fastener nut using the flat-tipped screwdriver until it can be loosened by hand.
 
 
 
 
INSTALLATION SERVICE POINTS
 
 
>>A<< FUEL RETURN HOSE B/FUEL RETURN HOSE E/FUEL RETURN HOSE C/FUEL HOSE B/FUEL HOSE C CONNECTION
Securely insert the fuel return hose and fuel hose to the fuel filter or fuel line heater, and install the hose clip in the position shown in the figure.
 
 
 
>>B<< FUEL FILTER CAP INSTALLATION
1. Install the fuel filter cap by aligning the marks on the fuel filter cap and the fuel filter case.
2. Securely tighten the fastener nut until the mark is in the position shown in the figure (until it clicks in place).
