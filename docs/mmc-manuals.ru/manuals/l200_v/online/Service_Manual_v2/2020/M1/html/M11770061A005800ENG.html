Inspection Procedure 8: AS&G System Abnormal (Auto Start Carried Out Under Condition That Brake Pedal is Operated) <A/T>
 
 
 
 
FUNCTION
When the specified conditions are satisfied, the AS&G-ECU should send the engine start request signal to the engine-ECU via CAN bus line despite the driver's determination. When receiving the signal, the engine-ECU should start the engine.
 
 
COMMENTS ON TROUBLE SYMPTOM
The failure could possibly be caused by the failed AS&G system. The AS&G-ECU, however, can possibly carry out the forced auto start, depending on the vehicle status or the ambient conditions.
 
 
PROBABLE CAUSES
  • Failed AS&G-ECU system.
  • Failed diesel fuel system.
  • Failed A/T system.
  • Failed ASTC system.
  • Failed automatic air conditioner system. <Vehicles with auto A/C>
  • Failed AS&G-ECU.
 
 
DIAGNOSIS PROCEDURE
STEP 1. Confirm the vehicle status and the ambient conditions
Confirm the vehicle status or the ambient conditions to check whether the forced auto start should be carried out or not (Refer to ).
Are the conditions for carrying out the forced auto start satisfied?
Check end.
Go to Step 2.
 
STEP 2. M.U.T.-III diagnosis code
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Use M.U.T.-III to check the diagnosis code is set in the following ECUs.
  • AS&G-ECU
  • Engine-ECU
  • A/T-ECU
  • ASTC-ECU
  • A/C-ECU <Vehicles with auto A/C>
Is any diagnosis code set?
YES <Diagnosis code is set in the AS&G-ECU> : Perform the troubleshooting of the AS&G-ECU (Refer to ).
YES <Diagnosis code is set in the engine-ECU> : Perform the troubleshooting of the engine-ECU (Refer to GROUP 13A - Troubleshooting, Inspection Chart for Diagnosis Code ).
YES <Diagnosis code is set in the A/T-ECU> : Perform the troubleshooting of the transfer-ECU (Refer to GROUP 23C - Troubleshooting <A/T>, Diagnosis Code Chart ).
YES <Diagnosis code is set in the ASTC-ECU> : Perform the troubleshooting of the ASTC-ECU (Refer to GROUP 35C - Troubleshooting, Diagnosis Code Chart ).
YES <Diagnosis code is set in the A/C-ECU> : Perform the troubleshooting of the A/C-ECU (Refer to GROUP 55B - Troubleshooting, Diagnosis Code Chart ).
Go to Step 3.
 
STEP 3. M.U.T.-III data list
Use M.U.T.-III to check the data list (Refer to ).
  • Item No. 17: Vehicle speed
Is the check result normal?
Go to Step 4.
Perform the troubleshooting of the engine-ECU (Refer to GROUP 13A - Troubleshooting, Diagnosis Code Procedures, Diagnosis Code No. P0500: Vehicle Speed Signal Low Input ).
 
STEP 4. M.U.T.-III data list
Use M.U.T.-III to check the data list (Refer to ).
  • Item No. 44: G sensor signal(+:decel,-:accel)
OK:
  • -0.04 to 0.04 m/s2 (Stops the vehicle in a horizontal state).
  • The value changes in the range between -1 to 1 m/s2 (When driving).
Is the check result normal?
Go to Step 5.
Perform the troubleshooting of the ASTC-ECU (Refer to GROUP 35C - Troubleshooting, Diagnosis Code Procedures, Diagnosis Code No. C1377: G and Yaw Rate Sensor Error ).
 
STEP 5. Check for data list of other system
Use M.U.T.-III to check the ASTC-ECU data list (Refer to GROUP 35C - Troubleshooting, Data List Reference Table ).
  • Item No. 05: Master cylinder pressure
OK:
  • Increases by the amount of the brake pedal depression (The brake pedal is depressed).
  • -3 to 3 bar (The brake pedal is released).
Is the check result normal?
Go to Step 6.
Perform the troubleshooting of the ASTC-ECU (Refer to GROUP 35C - Troubleshooting, Diagnosis Code Procedures, Diagnosis Code No. C121D: Master Cylinder Pressure Sensor Malfunction ).
 
STEP 6. M.U.T.-III data list
Use M.U.T.-III to check the data list (Refer to ).
  • Item No. 18: Accelerator Pedal Position
OK:
  • 0% (Release the accelerator pedal).
  • 98% or more (Depress the accelerator pedal fully).
Is the check result normal?
Go to Step 7.
Perform the troubleshooting of the accelerator pedal position sensor [Refer to GROUP 13A - Troubleshooting, Diagnosis Code Procedures, Diagnosis Code No. P2138: Accelerator Pedal Position Sensor (Main and Sub) Range/Performance Problem ].
 
STEP 7. M.U.T.-III data list
Use M.U.T.-III to check the data list (Refer to ).
  • Item No. 14: Brake Vacuum sensor
OK: Increases whenever the brake pedal is depressed (vacuum pressure is reduced) (When depress the brake pedal several times).
Is the check result normal?
Go to Step 8.
Check the brake vacuum sensor circuit (Refer to ).
 
STEP 8. Check the trouble symptom
Does the trouble symptom persist?
Replace the AS&G-ECU.
Intermittent malfunction (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
