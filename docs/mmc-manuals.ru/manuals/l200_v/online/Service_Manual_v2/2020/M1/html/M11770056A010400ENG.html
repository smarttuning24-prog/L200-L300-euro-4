Code No. U1210: Brake Lamp Switch Status Error <A/T>
 
 
 
 
TROUBLE JUDGMENT
Check Conditions
  • 2 seconds has passed after the ignition switch is turn to the "ON" position or the engine has started up.
  • The battery voltage is more than 9 V and is less than 16 V.
  • Neither of the following diagnosis codes is set.
    1. Code No. U0121: ASTC-ECU Time-out
    2. Code No. U0141: ETACS-ECU Time-out
Judgement Criterion
  • The brake lamp switch is OFF when the brake master cylinder hydraulic pressure is 1 MPa or more.
 
 
COMMENTS ON TROUBLE SYMPTOM
Current malfunction
  • The failure is possibly caused by malfunction of the ASTC-ECU signal, by malfunction in the ASTC-ECU, of the ETACS-ECU signal, of the brake lamp switch (stuck to OFF), in the AS&G-ECU.
Past malfunction
  • Proceed with troubleshooting focusing on malfunction of the ASTC-ECU signal and malfunction of the brake lamp switch system. For diagnosis procedure, refer to cope with past malfunctions (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
 
 
FAIL-SAFE AND BACKUP FUNCTION
In the normal operation, the auto stop is prohibited. If the malfunction is detected during the auto stop, the auto start is prohibited.
 
 
PROBABLE CAUSES
  • Failed brake master cylinder hydraulic pressure signal that is entered in the AS&G-ECU from the ASTC-ECU via CAN communication.
  • Failed brake lamp switch signal that is entered in the AS&G-ECU from the ETACS-ECU via CAN communication.
  • Failed hydraulic unit assembly (integrated with ASTC-ECU).
  • Failed AS&G-ECU.
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check for diagnosis code of other system
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Use M.U.T.-III to check if the diagnosis code is set in the ASTC-ECU.
Is any diagnosis code set in the ASTC-ECU?
Perform troubleshooting of the ASTC-ECU (Refer to GROUP 35C - Troubleshooting, Diagnosis Code Chart ).
Go to Step 2.
 
STEP 2. M.U.T.-III data list
Use M.U.T.-III to check the data list (Refer to ).
  • Item No. 30: Brake lamp SW
OK:
  • ON (Depressed the brake pedal.)
  • OFF (Released the brake pedal.)
Is the check result normal?
Intermittent malfunction (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
Go to Step 3.
 
STEP 3. Check for data list of other system
Use M.U.T.-III to check the data list of ETACS-ECU (Refer to GROUP 54A - ETACS, Data List Reference Table ).
  • Item No. 108: Stop lamp switch
OK:
  • ON (Depressed the brake pedal)
  • OFF (Released the brake pedal)
Is the check result normal?
Go to Step 5.
Go to Step 4.
 
STEP 4. Check stop lamp switch itself
Check stop lamp switch itself (Refer to GROUP 35A - Brake Pedal, Stop Lamp Switch Check ).
Is the check result normal?
Check the stop lamp switch circuit (Refer to GROUP 54A - ETACS, Input Signal Procedures, Inspection Procedure 10: The Stop Lamp Switch Signal is Not Received ).
Replace the stop lamp switch (Refer to GROUP 35A - Brake Pedal ).
 
STEP 5. Diagnosis code recheck after erasing diagnosis code
Check again if the diagnosis code is set.
(1) Use M.U.T.-III to erase the diagnosis code.
(2) Turn the ignition switch to "LOCK" (OFF) position.
(3) Turn the ignition switch to "ON" position.
(4) Drive the vehicle at 15 km/h or more for 15 minutes.
(5) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. U1210 set?
Replace the AS&G-ECU.
Intermittent malfunction (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
