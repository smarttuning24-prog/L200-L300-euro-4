Code No. U05A4: PM Sensor Received Invalid Data from DCU
 
 
cautionReplace DCU after certainly confirming that the communication circuits are normal.
 
 
TROUBLE JUDGEMENT
Check Conditions
  • Ignition switch is "ON" position.
Judgement Criterion
  • DCU failure input into DCU sent from exhaust muffler PM sensor for 2 seconds.
 
 
COMMENTS ON TROUBLE SYMPTOM
When the exhaust muffler PM sensor receives invalid data from the DCU, the exhaust muffler PM sensor sends a DCU abnormality signal to the DCU.
 
 
PROBABLE CAUSES
  • Failed harness and connector in CAN bus line
  • Failed DCU
  • Failed harness and connector of exhaust muffler PM sensor
  • Failed exhaust muffler PM sensor
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III diagnosis code
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Use M.U.T.-III to check whether any diagnosis code related to the CAN communication other than diagnosis code No. U05A4 is set.
Is any diagnosis code set?
Carry out the relevant troubleshooting for the set diagnosis code (Refer to ).
Go to Step 2.
 
STEP 2. M.U.T.-III diagnosis code
Use M.U.T.-III to check whether any diagnosis code related to the exhaust muffler PM sensor is set.
Is any diagnosis code set?
Carry out the relevant troubleshooting for the set diagnosis code (Refer to ).
Go to Step 3.
 
STEP 3. Diagnosis code recheck after erasing diagnosis code
Check again if the diagnosis code is set.
(1) Use M.U.T.-III to erase the diagnosis code.
(2) Turn the ignition switch to "LOCK" (OFF) position.
(3) Turn the ignition switch to "ON" position.
(4) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. U05A4 set?
Replace the exhaust muffler PM sensor (Refer to ).
Intermittent malfunction in CAN bus line between DCU and exhaust muffler PM sensor (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
