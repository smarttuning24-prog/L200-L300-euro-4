Code No. P0620: Alternator Control Circuit <4N14>
 
 
 
 
OPERATION
The engine-ECU sends request signals, such as starting and stopping of generation and generation voltage, to the alternator via LIN communication. The alternator sends the generation status information to the engine-ECU via LIN communication.
 
 
FUNCTION
The engine-ECU checks the generation status for any abnormality by comparing the battery voltage and alternator generation voltage.
 
 
TROUBLE JUDGEMENT
Check Conditions
  • While the engine is running.
  • After 2 seconds since the ignition switch is turned to the "ON" position.
  • After 2 seconds since the engine speed is 350 r/min or more, or after 2 seconds since the engine has started up.
Judgement Criterion
  • The voltage, which is calculated by subtracting final target generation voltage from the battery backup voltage, is 1.0 V or more.
or
  • The voltage, which is calculated by subtracting battery backup voltage from the final target generation voltage, is 1.3 V or more.
 
 
FAIL-SAFE AND BACKUP FUNCTION
  • Prohibits alternator output suppression control against current consumers (Operates as a normal alternator).
  • The auto stop is prohibited.
  • The small injection quantity learning is prohibited.
 
 
PROBABLE CAUSES
  • Failed battery
  • Failed alternator drive belt
  • Failed connector contact
  • Disconnection, open/short and damage of alternator LNBS circuit
  • Damage of battery backup circuit
  • Failed alternator assembly
  • Failed engine-ECU
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III diagnosis code
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Use M.U.T.-III to check the diagnosis code.
Is diagnosis code No. P0622 or No. P1538 set?
Troubleshoot the diagnosis codes (Refer to <P0622> or <P1538>).
Go to Step 2.
 
STEP 2. Checking engine start conditions
Check whether the user uses the booster cable to start the engine.
noteThis diagnosis code can be set due to the battery voltage drop.
Have you ever used the booster cable to start the engine?
Delete the diagnosis code and finish the check.
Go to Step 3.
 
STEP 3. Check battery
Check battery (Refer to GROUP 54A - Battery, On-vehicle Service, Battery Test ).
Is the check result normal?
Go to Step 4.
Replace the battery.
 
STEP 4. Drive belt tension check
Drive belt tension check (Refer to GROUP 11A - On-vehicle Service, Alternator Drive Belt Tension Check ).
Is the check result normal?
Go to Step 5.
Replace the alternator drive belt.
 
STEP 5. Check harness between alternator connector terminal and engine-ECU connector terminal LNBS
Check output line for open/short circuit and damage.
Is the check result normal?
Go to Step 6.
Repair or replace the connector, or repair the damaged harness wire.
 
STEP 6. Check harness between battery and engine-ECU connector terminal BACK
Check output line for open/short circuit and damage.
Is the check result normal?
Go to Step 7.
Repair or replace the connector, or repair the damaged harness wire.
 
STEP 7. Diagnosis code recheck after erasing diagnosis code
Check again if the diagnosis code is set.
(1) Use M.U.T.-III to erase the diagnosis code.
(2) Turn the ignition switch to "LOCK" (OFF) position.
(3) Turn the ignition switch to "ON" position and start the engine.
(4) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. P0620 set?
Go to Step 8.
Intermittent malfunction (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
 
STEP 8. Diagnosis code recheck after replace the drive belt
After replacing the drive belt, check again if the diagnosis code is set.
(1) Replacing the drive belt.
(2) Use M.U.T.-III to erase the diagnosis code.
(3) Turn the ignition switch to "LOCK" (OFF) position.
(4) Turn the ignition switch to "ON" position and start the engine.
(5) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. P0620 set?
Go to Step 9.
Check is end.
 
STEP 9. Diagnosis code recheck after replace the alternator assembly
After replacing the alternator assembly, check again if the diagnosis code is set.
(1) Replacing the alternator assembly.
(2) Use M.U.T.-III to erase the diagnosis code.
(3) Turn the ignition switch to "LOCK" (OFF) position.
(4) Turn the ignition switch to "ON" position and start the engine.
(5) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. P0620 set?
Replace the engine-ECU.
Check is end.
