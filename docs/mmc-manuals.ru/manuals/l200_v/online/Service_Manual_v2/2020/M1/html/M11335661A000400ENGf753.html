Code No. P1297: Fuel Pressure Relief Coil Short Abnormal <4N14>
 
 
 
 
OPERATION
Refer to Code No. P1296: Fuel Pressure Relief Control Boost Time Out <4N14> .
 
 
FUNCTION
  • Engine-ECU monitors the fuel pressure in the common rail assembly by receiving the signal from the rail pressure sensor.
  • Engine-ECU controls the fuel pressure in the common rail assembly using the fuel pressure relief control valve.
 
 
TROUBLE JUDGEMENT
Check Condition
  • 2 seconds later after the ignition switch has been in "ON" position.
  • Battery positive voltage is 8 to 16 V.
  • Energisation time of the fuel pressure relief control valve is 0.42 microseconds or more.
  • The following diagnosis codes are not set.
    1. Diagnosis code No. P009B: Fuel Pressure Relief Control Circuit/Open
    2. Diagnosis code No. P009C: Fuel Pressure Relief Control Circuit/Low
    3. Diagnosis code No. P009D: Fuel Pressure Relief Control Circuit/High
    4. Diagnosis code No. P126A: Diagnosis IC Internal Clock Abnormal
    5. Diagnosis code No. P126B: Diagnosis IC Communication Error
    6. Diagnosis code No. P1296: Fuel Pressure Relief Control Boost Time Out
Judgement Criterion
  • Overcurrent in the fuel pressure relief control valve control circuit is detected.
 
 
FAIL-SAFE AND BACKUP FUNCTION
  • The rail pressure is controlled in the range between 50 and 100 MPa.
  • The fuel injection amount is restricted (Output restriction).
  • The open angle of the accelerator pedal is restricted.
  • The throttle valve is closed (Fixed opening degree).
  • The EGR control is stopped (fully closed).
  • The DPF regeneration is prohibited.
  • The AS&G control is prohibited.
  • The small injection quantity learning is prohibited.
  • The speed limiter control is prohibited.
  • The cruise control is prohibit.
 
 
PROBABLE CAUSES
  • Harness damage in fuel pressure relief control valve circuit or loose connector contact.
  • Failed common rail assembly (integrated with fuel pressure relief control valve).
  • Failed engine-ECU.
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check fuel pressure relief control valve itself
  • Fuel pressure relief control valve connector check.
  • Check fuel pressure relief control valve itself (Refer to ).
Is the check result normal?
Go to Step 2.
Repair or replace the connector, or replace the common rail assembly (integrated with fuel pressure relief control valve).
 
STEP 2. Check harness between fuel pressure relief control valve connector terminal and engine-ECU connector terminal FRSB, and fuel pressure relief control valve connector terminal and engine-ECU connector terminal FRS
Check power supply line and earthing line for short circuit.
Is the check result normal?
Go to Step 3.
Repair or replace the connector, or repair the damaged harness wire.
 
STEP 3. Diagnosis code recheck after erasing diagnosis code
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Check again if the diagnosis code is set.
(1) Use M.U.T.-III to erase the diagnosis code.
(2) Turn the ignition switch to "LOCK" (OFF) position.
(3) Decelerate the vehicle once by releasing the accelerator pedal when driving the vehicle at 50 km/h or more.
(4) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. P1297 set?
Replace the engine-ECU.
Intermittent malfunction (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
