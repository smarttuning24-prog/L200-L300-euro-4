DISASSEMBLY SERVICE POINTS
 
 
<<A>> BALL BEARING REMOVAL
1. Use special tool remover (MD998801) to support the ball bearing.
2. Press the front output shaft with a press and remove the ball bearing.
 

 
 
<<B>> BALL BEARING REMOVAL
1. Use special tool remover (MD998801) to support the ball bearing.
2. Press the front output shaft with a press and remove the ball bearing.
 
 

 
 
REASSEMBLY SERVICE POINTS
 
 
>>A<< BALL BEARING INSTALLATION
1. Use special tools to install the ball bearing.
  • Installer (100) (MD998813)
  • Installer adapter (38) (MD998818)
2. Press the front output shaft with a press and install the ball bearing.
 

 
 
>>B<< BALL BEARING INSTALLATION
1. Use special tools to install the ball bearing.
  • Installer (100) (MD998813)
  • Installer adapter (38) (MD998818)
2. Press the front output shaft with a press and install the ball bearing.
