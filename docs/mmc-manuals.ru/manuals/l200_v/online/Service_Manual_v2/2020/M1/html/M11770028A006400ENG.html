Inspection Procedure 5: AS&G System Abnormal (Auto Stop Not Carried Out) <M/T>
 
 
cautionIf the following services are performed, the backup data should be deleted. Do not perform these services without carrying out the check described in Step 2.
  • In case the cable is disconnected from the battery terminal
  • In case the connector is disconnected from the engine-ECU
 
 
FUNCTION
If the specified conditions are satisfied, the AS&G-ECU should send the engine stop request signal to the engine-ECU via CAN bus line. When receiving the signal, the engine-ECU should stop the engine.
 
 
COMMENTS ON TROUBLE SYMPTOM
The failure could possibly be caused by the failed AS&G system. The AS&G-ECU, however, can possibly prohibit the auto stop, depending on the vehicle status or the ambient conditions.
 
 
PROBABLE CAUSES
  • Deteriorated battery.
  • Failed AS&G system.
  • Short circuit in starter active signal circuit, or loose connector contact.
  • Failed AS&G-ECU.
 
 
DIAGNOSIS PROCEDURE
STEP 1. Confirm the vehicle status and the ambient conditions
Confirm the vehicle status or the ambient conditions to check whether the auto stop should be carried out or not (Refer to ).
Are the conditions for prohibiting the auto stop satisfied?
Check end.
Go to Step 2.
 
STEP 2. Check for data list of other system
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.

  • Item No. 436: 2nd Last Batt. Deteriorated ODO
  • Item No. 437: 3rd Last Batt. Deteriorated ODO
OK: 0 km (Ignition switch: ON)
Are the all items displayed as "0 km"?
Go to Step 3.
Write the check result. Then go to Step 3.

YES <Diagnosis code is set in the transfer-ECU> : Perform the troubleshooting of the ETACS-ECU (Refer to GROUP 22A - Troubleshooting <V6M5A>, Diagnosis Code Chart ).
YES <Diagnosis code is set in the ASTC-ECU> : Perform the troubleshooting of the ASTC-ECU (Refer to GROUP 35C - Troubleshooting, Diagnosis Code Chart ).
YES <Diagnosis code is set in the combination meter> : Perform the troubleshooting of the combination meter (Refer to GROUP 54A - Combination Meter, Troubleshooting, Diagnosis Code Chart ).
YES <Diagnosis code is set in the ETACS-ECU> : Perform the troubleshooting of the ETACS-ECU (Refer to GROUP 54A - ETACS, Troubleshooting, Diagnosis Code Chart ).
YES <Diagnosis code is set in the A/C-ECU> : Perform the troubleshooting of the A/C-ECU (Refer to GROUP 55B - Troubleshooting, Diagnosis Code Chart ).
Go to Step 4.

noteIf the situation falls under any of the following items, go to "YES" whether or not the check result in Step 2.

STEP 16. Check of short to body earth in NTSW line between starter relay connector and engine-ECU connector
Is the check result normal?

Does the trouble symptom persist?
Replace the AS&G-ECU.
