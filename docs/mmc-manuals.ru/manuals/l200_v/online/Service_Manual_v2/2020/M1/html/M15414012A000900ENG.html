Inspection Procedure 1: None of the tail lamps illuminate.
 
 
cautionBefore replacing the ECU, ensure that the power supply circuit, the earth circuit and the communication circuit are normal.
 
 
 
 
COMMENTS ON TROUBLE SYMPTOM
If all the tail lamps do not illuminate normally, the tail lamp switch input circuit or ETACS-ECU may have a problem.
 
 
PROBABLE CAUSES
  • Malfunction of column switch
  • Malfunction of the ETACS-ECU
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III diagnosis code
Check if diagnosis code is set in the ETACS-ECU.
Is the diagnosis code set?
Troubleshoot the ETACS-ECU (Refer to ).
Go to Step 2.
 
STEP 2. M.U.T.-III data list
Use the ETACS-ECU data list to check the signals related to the tail lamp illumination.
  • Turn the tail lamp switch to the ON position.
 
Item No.Item nameNormal condition
59Tail lamp outputON
OK: Normal condition is displayed.
Is the check result normal?
Go to Step 3.
Troubleshoot the ETACS-ECU (Refer to Inspection Procedure 9 "The column switch signal is not received" ).
 
STEP 3. Retest the system
Check that the tail lamp illuminates normally.
Is the check result normal?
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to use Troubleshooting/inspection Service Points - How to Cope with Intermittent Malfunction ).
Replace the ETACS-ECU (Refer to ).
