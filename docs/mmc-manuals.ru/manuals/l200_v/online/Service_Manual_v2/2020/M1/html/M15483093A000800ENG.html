Code No. U0141: ETACS CAN Timeout
 
 
caution
  • If diagnosis code No. U0141 is set, be sure to diagnose the CAN bus line.
  • When replacing the ECU, always check that the communication circuit is normal.
 
 
TROUBLE JUDGEMENT
If the signal from ETACS-ECU cannot be received, the gateway-ECU (for GW-CAN-C) sets diagnosis code No. U0141.
 
 
JUDGEMENT CRITERIA
With the ignition switch is at "ON" position, diagnosis code No. U1190 is not set, if the communication with ETACS-ECU cannot be established for 2,500 ms or more, the gateway-ECU (for GW-CAN-C) determines that a problem has occurred.
 
 
PROBABLE CAUSES
  • Malfunction of CAN bus line
  • Malfunction of the gateway-ECU (for GW-CAN-C)
  • Malfunction of the ETACS-ECU
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III CAN bus diagnostics
Use the M.U.T.-III to diagnose the CAN bus lines.
Is the check result normal?
Go to Step 2.
Repair the CAN bus line (Refer to ).
 
STEP 2. M.U.T.-III other system diagnosis code
Check that the other ECU, which belongs to the CAN-C (main) bus line and performs CAN communication with the ETACS-ECU, sets diagnosis code No. U0141 (Refer to ).
Is the diagnosis code set?
Go to Step 3.
Go to Step 4.
 
STEP 3. Check whether the diagnosis code is set again.
Recheck if the diagnosis code is set.
(1) Erase the diagnosis code.
(2) Turn the ignition switch from "LOCK" (OFF) position to "ON" position.
(3) On completion, check that the diagnosis code is not set again.
Is the diagnosis code set?
Replace ETACS-ECU (Refer to GROUP 54A - ETACS ).
A poor connection, open circuit or other intermittent malfunction is present in the CAN bus lines between the ETACS-ECU and the gateway-ECU (for GW-CAN-C) (Refer to GROUP 00 - How to Cope with Intermittent Malfunctions ).
 
STEP 4. Check whether the diagnosis code is set again.
Recheck if the diagnosis code is set.
(1) Erase the diagnosis code.
(2) Turn the ignition switch from "LOCK" (OFF) position to "ON" position.
(3) On completion, check that the diagnosis code is not set again.
Is the diagnosis code set?
Replace gateway-ECU (for GW-CAN-C) (Refer to ).
A poor connection, open circuit or other intermittent malfunction is present in the CAN bus lines between the ETACS-ECU and the gateway-ECU (for GW-CAN-C) (Refer to GROUP 00 - How to Cope with Intermittent Malfunctions ).
