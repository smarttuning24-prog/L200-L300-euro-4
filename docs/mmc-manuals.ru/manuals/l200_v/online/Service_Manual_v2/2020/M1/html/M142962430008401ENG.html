Code No. B1166 Special mode
 
 
 
 
DIAGNOSTIC FUNCTION
When KOS&OSS-ECU is in the special operation mode (ECU is new), KOS&OSS-ECU sets diagnosis code No.B1166, and flashes the indicator of engine switch in green.
noteDiagnosis code B1166 will be erased the system will resume normal functionality when the electric steering lock control is activated. To do so, register the chassis number and the keyless operation key code, execute the global coding, and then operate the engine switch to authorise the key.
 
 
JUDGEMENT CRITERIA
Judgement Criterion
  • During special operation mode
 
 
PROBABLE CAUSES
  • Chassis number not written
  • Key code not written
  • Malfunction of KOS&OSS-ECU
  • KOS&OSS-ECU malfunction
  • Malfunction of keyless operation key
  • Malfunction of electric steering lock
 
 
DIAGNOSIS PROCEDURE
1. Check the KOS diagnosis code.
2. Check if KOS&OSS-ECU diagnosis code No.B1160, B1164, B1165, U1195 or U1199 is set.
3. After operating the engine switch from "ON" to "OFF" twice, recheck the diagnosis code of KOS&OSS-ECU.
