REMOVAL SERVICE POINT
 
 
<<A>> DOOR LATCH ASSEMBLY/DOOR LOCK COVER REMOVAL
Remove the door lock cover in the following procedures:
1. Release the upper and lower claws of the door lock cover by using a flat-tipped screwdriver.
2. Pull the cover out a little.
3. With the cover pulled out state, slide it toward cabled side and remove it from the door latch assembly.
