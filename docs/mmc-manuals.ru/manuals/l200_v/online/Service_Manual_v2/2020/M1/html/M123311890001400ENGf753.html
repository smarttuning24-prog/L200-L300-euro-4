DISASSEMBLY
 
 
1. SEAL RING REMOVAL
Remove the 3 seal rings from the forward clutch drum sub-assembly.
 

 
 
2. CLUTCH DISC SET NO. 1 (C1) REMOVAL
1. Using a screwdriver, remove the snap ring from the forward clutch drum sub-assembly.
2. Remove the flange, the 5 discs and the 5 plates from the forward clutch drum sub-assembly.
noteCheck the number and order of flange, discs and plates.
 

 
 
3. FORWARD CLUTCH RETURN SPRING SUB-ASSEMBLY REMOVAL
1.
cautionBe careful not to shorten the return spring too much.
Set special tool spring compressor body (MD998903) on the clutch balancer No. 1, and compress the forward clutch return spring sub-assembly with a shop press.
2.
cautionBe careful not to expand the snap ring too much.
Using snap ring pliers, remove the snap ring from the forward clutch drum sub-assembly.
3. Remove the clutch balancer No. 1 and forward clutch return spring sub-assembly from the forward clutch drum sub-assembly.
 

 
 
4. O-RING REMOVAL
Remove the O-ring from the clutch balancer No. 1.
 

 
 
5. FORWARD CLUTCH PISTON REMOVAL
1. While pushing the forward clutch piston by hand, apply compressed air (392 kPa) into the oil passage as shown in the illustration and remove the forward clutch piston from the forward clutch drum sub-assembly.
2. Remove the O-ring from the forward clutch piston.
 

 
 
6. O-RING REMOVAL
Remove the O-ring from the forward clutch drum sub-assembly.
 

 
 
7. CLUTCH DISC SET NO. 2 (C2) REMOVAL
1. Using a screwdriver, remove the snap ring from the C2 clutch drum sub-assembly.
2. Remove the flange, the 5 discs and the 5 plates from the C2 clutch drum sub-assembly.
noteCheck the number and order of flange, discs and plates.
 

 
 
8. DIRECT CLUTCH RETURN SPRING SUB-ASSEMBLY REMOVAL
1.
cautionBe careful not to shorten the return spring too much.
Set special tool spring compressor (MB991628) on the clutch balancer No. 2, and compress the direct clutch return spring sub-assembly with a shop press.
2.
cautionBe careful not to expand the snap ring too much.
Using snap ring pliers, remove the snap ring from the C2 clutch drum sub-assembly.
3. Remove the clutch balancer No. 2 and direct clutch return spring sub-assembly from the C2 clutch drum sub-assembly.
 

 
 
9. O-RING REMOVAL
Remove the O-ring from the clutch balancer No. 2.
 

 
 
10. DIRECT CLUTCH PISTON REMOVAL
Close 3 holes on the inner side of the direct clutch piston with your fingers, then blow compressed air (392 kPa) through the hole not closed to remove the direct clutch piston from the C2 clutch drum sub-assembly.
 

 
 
11. O-RING REMOVAL
Remove the 2 O-rings from the direct clutch piston.
 
 
 
 
REASSEMBLY
 
 
INSPECTION BEFORE REASSEMBLY
Refer to Forward Clutch and Direct Clutch Inspection .
  • Inspect Clutch Disc Set No. 1 (C1)
  • Check Forward Clutch Return Spring Sub-assembly
  • Inspect Clutch Disc Set No. 2 (C2)
  • Check Direct Clutch Return Spring Sub-assembly
  • Check Forward Clutch Drum Sub-assembly Bushing
  • Check C2 Clutch Drum Sub-assembly Bushing
  • Check Clutch Hub Sub-assembly Bushing

 
 
1. O-RING INSTALLATION
1. Apply ATF to 2 new O-rings.
2. Install them to the direct clutch piston.

2. DIRECT CLUTCH PISTON INSTALLATION
1. Apply ATF to the sliding surface of the C2 clutch drum sub-assembly.
2.
cautionBe careful not to damage the O-rings and the C2 clutch drum sub-assembly and direct clutch piston.
Install the direct clutch piston to the C2 clutch drum sub-assembly.

4. CLUTCH BALANCER NO. 2 INSTALLATION
1.
cautionBe careful not to damage the O-ring on the clutch balancer No. 2.
Install the direct clutch return spring sub-assembly and clutch balancer No. 2 to the C2 clutch drum sub-assembly.
2.
cautionBe careful not to shorten the return spring too much.
Set special tool spring compressor (MB991628) on the clutch balancer No. 2, and compress the direct clutch return spring sub-assembly with a shop press.
3.
cautionBe careful not to expand the snap ring too much.
Using snap ring pliers, install the snap ring in the groove.
 

 
 
5. CLUTCH DISC SET NO. 2 (C2) INSTALLATION
1.
cautionCheck the number and order of flange, discs and plates.
Install the 5 plates, the 5 discs and the flange to the C2 clutch drum sub-assembly as shown in the illustration.
note
  • When measuring the C2 clutch pack clearance, use a flange of 3.5 mm thickness.
  • Install the plates, discs and the flange in the order shown in the illustration.

2.
cautionWhen installing the snap ring, set the end gap of the snap ring as shown in the illustration.
Using a screwdriver, install the snap ring in the groove.
 

 
 
6. CHECK PACK CLEARANCE OF CLUTCH DISC SET NO. 2 (C2)
1. Set the oil pump assembly onto the torque converter assembly, and then set the C2 clutch drum sub-assembly onto the oil pump assembly.
2.
caution
  • Measure the circumference of the flange at 3 places, and make sure that the average value is within the specified range.
  • If the average value is outside the specified range, check the installation condition.
Place a dial indicator and apply compressed air (196 kPa) as shown in the illustration.
note
  • Use a flange of assembling plate.
  • When using new discs and plates, please use flange of 3.5 mm thickness.
3.
cautionIf the C2 clutch pack clearance is not as specified, select an appropriate clutch flange No. 2.
Select a flange and adjust the pack clearance.
Standard value: 0.5 - 0.8 mm
(C2 clutch pack clearance)
