Inspection Procedure 6: Hunting (Repeated Acceleration and Deceleration) Occurs at the Set Vehicle Speed
 
 
 
 
COMMENTS ON TROUBLE SYMPTOM
  • The wheel speed sensor signal (vehicle speed signal) or the diesel fuel system is suspected.
  • The wheel speed sensor signal (vehicle speed signal) is transmitted from the ASTC-ECU to the engine-ECU via the CAN bus line.
 
 
PROBABLE CAUSES
  • Malfunction of CAN bus system.
  • Malfunction of the ASTC system.
  • Malfunction of the A/T system.
  • Malfunction of the diesel fuel system.
  • Malfunction of the engine-ECU.
 
 
DIAGNOSIS PROCEDURE
caution
  • If there is any problem in the CAN bus lines, an incorrect diagnosis code may be set. Prior to this diagnosis, diagnose the CAN bus lines.
  • Before replacing the ECU, ensure that the CAN bus lines is normal.
STEP 1. Check the CAN bus system diagnosis
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Use M.U.T.-III to perform CAN bus diagnosis (Refer to GROUP 54C - Explanation About The M.U.T.-III Can Bus Diagnostics ).
Is the check result satisfactory?
Go to Step 2.
Repair the CAN bus lines (Refer to GROUP 54C - Troubleshooting, Can Bus Diagnostic Table ). Then go to Step 7.
 
STEP 2. Check the ASTC system diagnosis code
Use M.U.T.-III to check ASTC system diagnosis code (Refer to GROUP 35C - Troubleshooting, Diagnosis Function ).
Is any diagnosis code set?
Repair the ASTC system (Refer to GROUP 35C - Troubleshooting, Diagnosis Code Chart ). Then go to Step 7.
Go to Step 4 <M/T>.
Go to Step 3 <A/T>.
 
STEP 3. Check the A/T system diagnosis code <A/T>
Use M.U.T.-III to check A/T system diagnosis code (Refer to GROUP 23C - Troubleshooting, Diagnosis Function ).
Is any diagnosis code set?
Repair the A/T system (Refer to GROUP 23C - Troubleshooting, Diagnosis Code Chart ). Then go to Step 7.
Go to Step 4.
 
STEP 4. Check the diesel fuel system diagnosis code
Use M.U.T.-III to check diesel fuel system diagnosis code (Refer to GROUP 13A - Troubleshooting, Diagnosis Function ).
Is any diagnosis code set?
Repair the diesel fuel system (Refer to GROUP 13A - Troubleshooting, Inspection Chart for Diagnosis Trouble Code ). Then go to Step 7.
Go to Step 5.
 
STEP 5. M.U.T.-III data list
Use M.U.T.-III to check the data list (Refer to ).
  • Item No. 04: Vehicle speed.
Is the check result normal?
Go to Step 6.
Replace the engine-ECU (Refer to GROUP 13A - Engine-ECU <4N14> or <4N15>). Then go to Step 7.
 
STEP 6. Check the symptom
Does a hunting occur?
Replace the engine-ECU (Refer to GROUP 13A - Engine-ECU <4N14> or <4N15>). Then go to Step 7.
It can be assumed that this malfunction is intermittent (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunction ).
 
STEP 7. Check the symptom
Does a hunting occur?
Return to Step 1.
The procedure is complete.
