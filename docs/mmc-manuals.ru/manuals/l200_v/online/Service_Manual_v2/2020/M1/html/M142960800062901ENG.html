Inspection Procedure 9: Keyless entry hazard answer back function or the room lamp answer back function does not work normally.
 
 
cautionBefore replacing the ECU, ensure that the power supply circuit, the earth circuit and the communication circuit are normal.
 
 
 
 
OPERATION
When the keyless entry function is used, the keyless entry hazard answer back function or room lamp answer back function operate as set by ETACS customisation function (If the flashing count is set to 0 with a customisation function, no answer back function is performed).
 
 
PROBABLE CAUSES
  • Function setting error or no setting with a customisation
  • Malfunction of the turn signal lamp
  • Malfunction of the room lamp
  • Malfunction of ETACS-ECU
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check the hazard warning lamp
Check that the hazard warning lamp illuminate normally.
Is the check result normal?
Go to Step 2.
Refer to GROUP 54A - Hazard Warning Lamp Switch .
 
STEP 2. Check the room lamps operation
Check that the room lamps illuminate normally.
Is the check result normal?
Go to Step 3.
Refer to GROUP 54A - Interior Lamp .
 
STEP 3. Check the keyless entry system operation
Check that the keyless entry system operation normally.
Is the check result normal?
Go to Step 4.
Refer to Inspection Procedure 7: Keyless entry system does not work .
 
STEP 4. Check the customisation
Check that any one of the followings other than "Lock: 0, Unlock: 0" is set for Hazard answer back with a customisation function.
  • Lock: 1, Unlock: 2
  • Lock: 1, Unlock: 0
  • Lock: 0, Unlock: 2
  • Lock: 2, Unlock: 1
  • Lock: 2, Unlock: 0
  • Lock: 0, Unlock: 1
  • Lock: 0, Unlock: 0
noteIf "Lock: 0" or "Unlock: 0" is selected, no answer back function is performed on locking or unlocking.
Is it set to other than "Lock: 0, Unlock: 0"?
Go to Step 5.
Set Hazard answer back to any one other than "Lock: 0, Unlock: 0" with a customisation function (Refer to ).
 
STEP 5. M.U.T.-III diagnosis code
Check if any diagnosis code is set in the KOS&OSS-ECU.
Is the diagnosis code set?
Refer to diagnosis code chart .
Go to Step 6.
 
STEP 6. M.U.T.-III other system diagnosis code
Check if any diagnosis code is set in the ETACS-ECU.
Is the diagnosis code set?
Perform troubleshooting for the ETACS (Refer to GROUP 54A - ETACS - Troubleshooting ).
Go to Step 7.
 
STEP 7. Check of the troubles
Check that the hazard answer back and room lamp answer back functions of the keyless entry operate.
Is the check result normal?
Intermittent malfunction is suspected (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunction ).
Replace ETACS-ECU (Refer to GROUP 54A - ETACS, ETACS-ECU ).
