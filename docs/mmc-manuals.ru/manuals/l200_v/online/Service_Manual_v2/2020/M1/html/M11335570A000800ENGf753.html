Code No. P0515: Battery Temperature Sensor System <4N14>
 
 
 
 
OPERATION
  • Power is supplied by the battery to the battery current sensor.
  • The battery current sensor (integrated with battery temperature sensor) sends a signal to the engine-ECU connector terminal LNBS via the Local Interconnect Network (LIN).
 
 
FUNCTION
  • The battery temperature sensor is incorporated into the battery current sensor and it informs the engine-ECU of battery temperature via the Local Interconnect Network (LIN).
  • In response to the signal, the engine-ECU controls charging and discharging.
 
 
TROUBLE JUDGEMENT
Check Condition
  • After 2 seconds since the ignition switch is turned ON or since the completion of engine start.
  • Battery positive voltage is more than 10 V and is less than 16.5 V.
Judgement Criterion
  • For 2 seconds, the battery temperature information from battery temperature sensor is abnormal.
 
 
FAIL-SAFE AND BACKUP FUNCTION
  • Prohibits alternator output suppression control against current consumers (operates as a normal alternator).
  • The AS&G control is prohibited.
  • The small injection quantity learning is prohibited.
 
 
PROBABLE CAUSES
  • Failed battery current sensor (integrated with battery temperature sensor).
  • Failed engine-ECU
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III diagnosis code
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Use M.U.T.-III to check the diagnosis code.
Is diagnosis code No. P0622 or No. P1538 set?
Troubleshoot the diagnosis codes (Refer to <P0622> or <P1538>).
Go to Step 2.
 
STEP 2. Diagnosis code recheck after erasing diagnosis code
Check again if the diagnosis code is set.
(1) Use M.U.T.-III to erase the diagnosis code.
(2) Turn the ignition switch to "LOCK" (OFF) position.
(3) Turn the ignition switch to "ON" position.
(4) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. P0515 set?
Go to Step 3.
Intermittent malfunction (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
 
STEP 3. Diagnosis code recheck after replace the battery current sensor
After replacing the battery current sensor, check again if the diagnosis code is set.
(1) Replacing the battery current sensor.
(2) Use M.U.T.-III to erase the diagnosis code.
(3) Turn the ignition switch to "LOCK" (OFF) position.
(4) Turn the ignition switch to "ON" position.
(5) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. P0515 set?
Replace the engine-ECU.
The check is end.
