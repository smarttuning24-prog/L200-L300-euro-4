DIAGNOSIS CODE CHART
cautionDuring troubleshooting, a diagnosis code associated with other systems may be set when the ignition switch is turned to the "ON" position with connector(s) disconnected. On the completion of troubleshooting, confirm all systems for diagnosis code(s). If diagnosis code(s) are set, erase them all.
