Code No. U1102: ASC timeout
 
 
caution
  • If diagnosis code No. U1102 is set, diagnose the CAN bus lines.
  • When replacing the ECU, always check that the SRS circuit is normal.
 
 
FUNCTION
If signal from the ASC-ECU cannot be received for 500 milliseconds or more, the SRS-ECU will set diagnosis code No. U1102.
 
 
TROUBLE JUDGEMENT
Because of the CAN-B bus circuit malfunction, if SRS-ECU becomes unable to perform the normal data transmission, SRS-ECU determines that an abnormality is present.
 
 
PROBABLE CAUSES
  • Malfunction of CAN bus line
  • Malfunction of ASC-ECU
  • Malfunction of SRS-ECU
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III CAN bus diagnostics
Use the M.U.T.-III to diagnose the CAN bus lines.
Is the check result normal?
Go to Step 2.
Repair the CAN bus line (Refer to GROUP 54C - Troubleshooting ). On completion, go to Step 5.
 
STEP 2. Check the M.U.T.-III Diagnosis code for other systems
Check whether the ASC-ECU sets a diagnosis code.
Is the diagnosis code set?
Diagnose the ASC-ECU. Refer to GROUP 35C, Troubleshooting .
Go to Step 3.
 
STEP 3. Check whether the diagnosis code is set again.
Check again if the diagnosis code is set in the SRS-ECU.
(1) Erase the diagnosis code.
(2) Turn the ignition switch from "LOCK" (OFF) position to "ON" position.
(3) On completion, check whether the diagnosis code is set again.
Is the diagnosis code set?
Replace the ASC-ECU. After replacing the ECU, carry out the CAN bus diagnostics. Then go to Step 4.
A poor connection, open circuit or other intermittent malfunction is present in the CAN bus lines between the ASC-ECU and the SRS-ECU (Refer to GROUP 00, How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunctions ).
 
STEP 4. Check whether the diagnosis code is set again.
Check again if the diagnosis code is set in the SRS-ECU.
(1) Erase the diagnosis code.
(2) Turn the ignition switch from "LOCK" (OFF) position to "ON" position.
(3) On completion, check that the diagnosis code is not set again.
Is the diagnosis code set?
Replace the SRS-ECU. After replacing the ECU, carry out the CAN bus diagnostics. Then go to Step 5.
A poor connection, open circuit or other intermittent malfunction is present in the CAN bus lines between the ASC-ECU and the SRS-ECU (Refer to GROUP 00, How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunctions ).
 
STEP 5. Check whether the diagnosis code is set again.
Recheck if the diagnosis code is set in the SRS-ECU.
(1) Erase the diagnosis code.
(2) Turn the ignition switch from "LOCK" (OFF) position to "ON" position.
(3) On completion, check whether the diagnosis code is set again.
Is the diagnosis code set?
Return to Step 1.
The procedure is complete.
