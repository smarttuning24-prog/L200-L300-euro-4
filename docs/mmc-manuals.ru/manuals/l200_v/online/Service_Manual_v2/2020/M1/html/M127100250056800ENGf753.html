REMOVAL SERVICE POINTS
 
 
<<A>> AXLE SHAFT ASSEMBLY REMOVAL
Pull the rear axle shaft from axle housing. If the rear axle shaft is hard to remove, use the following special tools.
  • Slide hammer (MB990211)
  • Rear axle shaft puller (MB990241)
 
 
 
<<B>> OIL SEAL REMOVAL
Use the following special tools with hook attached to remove the oil seal.
  • Slide hammer (MB990211)
  • Adapter (MB990212)
 
 
 
 
INSTALLATION SERVICE POINT
 
 
>>A<< OIL SEAL INSTALLATION
Drive the new oil seal into the rear axle housing end by using the following special tools.
  • Installation adapter (MB990932)
  • Installer bar (MB990938)
