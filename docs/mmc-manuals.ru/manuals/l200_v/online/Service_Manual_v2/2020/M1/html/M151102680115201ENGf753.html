Inspection Procedure 1: Electric-folding door mirrors do not work at all.
 
 
cautionWhenever the ECU is replaced, ensure that the input and output signal circuits are normal.
 
 
 
 
OPERATION
The ETACS-ECU operates this function in accordance with the input signals below.
  • Ignition switch (ACC)
  • Remote controlled mirror switch (fold switch)
 
 
COMMENTS ON TROUBLE SYMPTOM
If the system does not work, the input circuit from the Ignition switch (ACC), the remote controlled mirror switch (fold switch), or the ETACS-ECU may be defective.
 
 
PROBABLE CAUSES
  • Defective remote controlled mirror switch
  • The ETACS-ECU may be defective
  • Damaged harness wires and connectors
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III diagnosis code.
Check if an ETACS-related diagnosis code is set.
Is the diagnosis code set?
Diagnose the ETACS (Refer to GROUP 54A, ETACS, Diagnosis Code Chart ).
