Code No. P0299: Turbocharger Underboost Condition
 
 
 
 
FUNCTION
The engine-ECU checks whether the boost pressure is not lower than the specified value.
 
 
TROUBLE JUDGEMENT
Check Condition
  • Engine is running
Judgement Criterion
  • The opening degree of accelerator pedal is 23% for 3 seconds and the boost pressure is not lower than the specified value.
 
 
FAIL-SAFE AND BACKUP FUNCTION
  • The fuel injection amount is restricted (Output restriction).
  • The DPF regeneration is prohibited.
  • The EGR control is stopped (Fully closed).
  • The throttle valve is opened (Electrical discontinuity).
  • The turbocharger control is prohibited.
  • The AS&G control is prohibited.
  • The small injection quantity learning is prohibited.
  • The speed limiter control is prohibited.
  • The cruise control is prohibit.
 
 
PROBABLE CAUSES
  • Intercooler air hose disconnected or damaged
  • Vacuum hose of manifold absolute pressure sensor disconnected or damaged <4N15>
  • Failed manifold absolute pressure sensor
  • Failed engine-ECU
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check on intercooler air hose
Check whether the hose between turbocharger and inlet manifold is disconnected or damaged.
Is the check result normal?
YES <4N14> : Go to Step 3.
YES <4N15> : Go to Step 2.
Repair.
 
STEP 2. Check on vacuum hose of manifold absolute pressure sensor <4N15>
Check whether the vacuum hose of manifold absolute pressure sensor is disconnected or damaged.
Is the check result normal?
Go to Step 3.
Repair.
 
STEP 3. Check turbocharger supercharging pressure
Check turbocharger supercharging pressure (Refer to GROUP 15 - On-vehicle Service, Turbocharger Supercharging Pressure Check ).
Is the check result normal?
Go to Step 4.
Repair.
 
STEP 4. Check on clogged intercooler
noteWhen checking the clogged intercooler, make a visual check or use an air blower.
Check whether the intercooler is being clogged or not.
Is the check result normal?
Go to Step 5.
Replace the intercooler.
 
STEP 5. Check on clogged EGR cooler
noteWhen checking the clogged EGR cooler, make a visual check or use an air blower.
Check whether the EGR cooler is being clogged or not.
Is the check result normal?
Go to Step 6.
Replace the EGR cooler.
 
STEP 6. Check on clogged EGR pipe (A, B)
Check whether the EGR pipe (A, B) is being clogged or not.
Is the check result normal?
Go to Step 7.
Replace the EGR pipe (A, B).
 
STEP 7. Diagnosis code recheck after erasing diagnosis code
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Check again if the diagnosis code is set.
(1) Use M.U.T.-III to erase the diagnosis code.
(2) Turn the ignition switch to "LOCK" (OFF) position.
(3) Turn the ignition switch to "ON" position and start the engine.
(4) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. P0299 set?
Replace the engine-ECU.
Intermittent malfunction (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
