Code No. B1859: e-Call SW
 
 
cautionWhen replacing the ECU, always check that the communication circuit is normal.
 
 
DIAGNOSTIC FUNCTION
  • When the system detects an unfeasible voltage in the right or left LED system for 1 seconds or more, the emergency call control unit sets diagnosis code No. B1859.
    noteWhen diagnosis code No. B1859 is set, the emergency call switch flashes in green.
  • When the emergency call switch is ON for 3 minutes or more, the emergency call control unit sets diagnosis code No. B1859.
    noteWhen diagnosis code No. B1859 is set, the emergency call switch illuminates in red.
 
 
PROBABLE CAUSES
  • Damaged wiring harness or connector(s)
  • Malfunction of emergency call switch
  • Malfunction of emergency call control unit
