Warning Indicator <Vehicles with Mono-colour Liquid Crystal Display Type Combination Meter>
When a problem occurs in the transfer or rear differential lock system, the 2WD/4WD operation indicator lamp (a) or rear differential lock lamp (b) flashes rapidly (twice per second).
When the rear differential lock lamp (b) is flashing rapidly, check the diagnosis output.
 
 
 
Warning Indicator <Vehicles with Colour Liquid Crystal Display Type Combination Meter>
When a problem occurs in the transfer, an icon (A) will be displayed. When a problem occurs in the rear differential lock system, an icon (B) will be displayed.
If any of these icons is displayed, check whether the diagnosis codes are set.

Display items of the freeze frame data are as follows.
