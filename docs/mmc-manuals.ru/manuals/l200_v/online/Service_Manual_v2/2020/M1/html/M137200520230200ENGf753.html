INSTALLATION SERVICE POINTS
 
 
>>A<< POWER STEERING OIL PRESSURE HOSE CONNECTION INSTALLATION
1. Install so that the mating marks are facing rearward.
2. Place the pressure hose stopper to touch the power steering oil pump stopper, and install the pressure hose.
 

 
 
>>B<< POWER STEERING OIL PUMP SUCTION HOSE CONNECTION INSTALLATION
Install the suction hose so that the marking is positioned upward.
