REMOVAL AND INSTALLATION
cautionIf the combination meter needs to be replaced, the current driving distance and elapsed days must be entered into the meter after the replacement in order to be used for service reminder function. Therefore, read "Integrated mileage for reminder," "Integrated days for reminder," "Mileage until Extra reminder," "Months until Extra reminder," and "Current schedule" from the meter before the replacement using the special function of M.U.T.-III, and note them. For the operation method of M.U.T.-III, refer to On-vehicle service . If "Integrated mileage for reminder" or "Integrated days for reminder" cannot be read from the meter using M.U.T.-III, use the following method.
  • As for the driving distance for check warning, use the driving distance displayed on the multi-information display.
  • As for the elapsed days for check warning, calculate the number of elapsed days from the delivery date to the customer (service reminder function start date) and current date.
Pre-removal Operation
  • Instrument Centre Panel Assembly Removal (Refer to GROUP 52A - Instrument Centre Panel ).
  • Outside/Inside Air Selection Damper Control Cable Connection, Air Mixing Damper Control Cable Connection, Mode Selection Damper Control Cable Connection Removal <Manual A/C> (Refer to GROUP 55A - Heater Control Unit ).
Post-installation Operation
  • Outside/Inside Air Selection Damper Control Cable Connection, Air Mixing Damper Control Cable Connection, Mode Selection Damper Control Cable Connection Installation <Manual A/C> (Refer to GROUP 55A - Heater Control Unit ).
  • Instrument Centre Panel Assembly Installation (Refer to GROUP 52A - Instrument Centre Panel ).
