1. INSPECT BRAKE DISC SET NO. 4 (B4)
Check the lining of all discs.
note
  • If the lining is flaking or has changed color, or if it is worn or the print mark is wearing away, replace with a new disc. When replacing, check the contact surfaces between the flange and plate and the disc. If they are scratched or have changed color, replace with new parts.
  • Before replacing with new discs, soak them at least 2 hours in ATF.
 

 
 
2. CHECK PACK CLEARANCE OF BRAKE DISC SET NO. 4 (B4)
1. Make sure that the low/reverse brake piston is pushed fully into the case.
2.
caution
  • Measure the circumference of the low/reverse brake piston at 3 places, and make sure that the average value is within the specified range.
  • If the average value is outside the specified range, reinstall the low/reverse brake piston, the brake reaction sleeve, and the inner brake piston No. 4.
Using vernier calipers, measure as shown in the illustration.
Length A: 32.07 - 32.81 mm
3. Stack the flange, the 7 discs and the 6 plates on top of the surface plate as shown in the illustration.
noteCheck the number and order of discs and plates.
4.
cautionMeasure the facing of the disc at 3 places, and make sure that the average value is within the specified range.
With the disc pressed against the surface plate, measure as shown in the illustration using vernier calipers.
Length B: 29.94 - 30.96 mm
noteMake sure that the measurement load is 3.92 N - 4.9 N.
5. Using vernier calipers, measure the brake flange No. 4 as shown in the illustration.
Length C: 0 - 1.43 mm
noteThe brake flange No. 4 comes in 11 types.
6.
cautionIf the B4 brake pack clearance is not as specified, select an appropriate brake flange No. 4.
Calculate the pack clearance from the measured value and the disc distortion compensation value (0.41 mm).
Standard value: 0.7 - 1.0 mm
(B4 brake pack clearance = A - B - C - 0.41)

3. INSPECT INTERMEDIATE SHAFT
1.
cautionIf the runout is not as specified, replace the intermediate shaft with a new one.
Using a dial indicator, check the intermediate shaft runout.
Standard value: 0.03 mm
2.
cautionIf the outer diameter is not as specified, replace the intermediate shaft with a new one.
Using a micrometer, check the outer diameter of the intermediate shaft at each point shown in the illustration.
Standard value:
A: 32.062 - 32.075 mm
B: 25.962 - 25.975 mm
C: 19.963 - 19.976 mm
 

 
 
4. CHECK ONE-WAY CLUTCH ASSEMBLY NO. 3
Hold the one-way clutch assembly No. 3 in place. Check that the rear planetary ring gear rotates when turned clockwise and does not rotate when turned counterclockwise.
 

 
 
5. CHECK BRAKE PISTON RETURN SPRING SUB-ASSEMBLY NO. 1
cautionIf the free length is smaller than the minimum, replace the brake piston return spring sub-assembly No. 1 with a new one.
Using vernier calipers, measure the free length of the brake piston return spring sub-assembly No. 1.
Standard value: 16.72 mm
 

 
 
6. INSPECT BRAKE DISC SET NO. 1 (B1)
Check the lining of all discs.
note
  • If the lining is flaking or has changed color, or if it is worn or the print mark is wearing away, replace with a new disc. When replacing, check the contact surfaces between the flange and plate and the disc. If they are scratched or have changed color, replace with new parts.
  • Before replacing with new discs, soak them at least 2 hours in ATF.
 

 
 
7. CHECK PACK CLEARANCE OF BRAKE DISC SET NO. 1 (B1)
1. Install the brake piston return spring sub-assembly No. 1 with the brake cylinder No. 1 with brake piston No. 1 to the transmission case.
2.
caution
  • Be careful not to shorten the return spring too much.
  • Do not directly press down the brake piston return spring sub-assembly No. 1 with a screwdriver. Otherwise, the brake piston return spring sub-assembly No. 1 may be deformed. Make sure to press down the snap ring with the screwdriver.
  • Never reuse the deformed brake piston return spring sub-assembly No. 1.
  • Be careful not to damage the transmission case.
Install the snap ring to the transmission case while pressing the snap ring and brake piston return spring sub-assembly No. 1 with screwdrivers.
3.
caution
  • Measure the circumference of the transmission case at 3 places, and make sure that the average value is within the specified range.
  • If the average value is outside the specified range, reinstall the brake cylinder No. 1 with the brake piston No. 1.
With snap ring pulled up on the converter housing side, measure as shown in the illustration using vernier calipers.
Length A: 26.27 - 26.99 mm
4. Stack the flange, the 4 discs, and the 4 plates on top of the surface plate as shown in the illustration.
note
  • Check the number and order of flange, discs and plates.
  • Use a flange of assembling plate.
  • When using new disc and plate, please use flange of 5.3 mm thickness.
5.
cautionMeasure the circumference of the flange at 3 places, and make sure that the average value is within the specified range.
With the flange pressed against the surface plate, measure as shown in the illustration using vernier calipers.
Length B: 22.43 - 23.00 mm
noteMake sure that the measurement load is 3.92 N - 4.9 N.
6.
cautionIf the B1 brake pack clearance is not as specified, select an appropriate brake flange No. 1.
Calculate the pack clearance from the measured value, the disc distortion compensation value (0.24 mm), and the snap ring thickness (3.0 mm).
Standard value: 0.40 - 0.70 mm
(B1 brake pack clearance = A - B - 0.24 - 3.0)
