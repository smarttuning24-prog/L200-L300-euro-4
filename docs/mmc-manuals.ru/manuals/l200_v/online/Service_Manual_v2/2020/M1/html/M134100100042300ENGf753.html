REMOVAL AND INSTALLATION
 
 
caution*: Indicates parts which should be temporarily tightened, and then fully tightened with the vehicle on the earth in the unladen condition.
Pre-installation Operation
  • Support the Axle Housing with a Garage Jack, etc.
Removal steps
1.Bolt (Parking brake cable connection)
>>D<<2.Eye washer
>>D<<3.Shock absorber
4.U-bolt seat
>>C<<5.U-bolts
6.Bump stopper
•Spare tyre (LH only)
7.Shackle plate
>>B<<8.Shackle assembly
9.Rubber bushings
>>B<<10.Bolt
>>A<<11.Rear spring
 
 
INSTALLATION SERVICE POINTS
 
 
>>A<< REAR SPRING INSTALLATION
1.
cautionDo not scratch the rear spring.
Align the rear leaf springs with a plastic hammer.
2. Make sure that the clearance (B) between the rear spring and the silencer is even.
 
 
 
>>B<< BOLT/SHACKLE ASSEMBLY INSTALLATION
1. Install the bolt from the outside toward the inside of vehicle.
2. Install the shackle assembly from the inside toward the outside of vehicle.
 

 
 
>>C<< U-BOLTS INSTALLATION
1. Tighten the U-bolts in the order shown evenly by 10 N·m.
2. Install the u-bolts as shown in the figure.
 

 
 
>>D<< SHOCK ABSORBER/EYE WASHER INSTALLATION
Be sure to install the shock absorber and eye washers (outer and inner) in the specified direction.
