Code No. U0245: NAVI/DISP.AUDIO CAN timeout <Vehicles with smartphone link display audio>
 
 
caution
  • If diagnosis code No. U0245 is set, be sure to diagnose the CAN bus line.
  • Whenever the ECU is replaced, ensure that the input and output signal circuits are normal.
 
 
TROUBLE JUDGEMENT
If signal from the multivision display cannot be received for 2,500 ms or more, the camera-ECU sets diagnosis code No. U0245.
 
 
PROBABLE CAUSES
  • Malfunction of multivision display
  • Malfunction of camera-ECU
  • Malfunction of CAN bus
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III CAN bus diagnostics
Use the M.U.T.-III to diagnose the CAN bus lines.
Is the check result normal?
Go to Step 2.
Repair the CAN bus line (Refer to GROUP 54C - Troubleshooting ) and then go to Step 3.
 
STEP 2. Diagnosis code recheck
Check again if the diagnosis code is set in the camera-ECU.
(1) Erase the diagnosis code.
(2) Turn the ignition switch from the "LOCK" (OFF) position to the "ON" position.
(3) Check if the diagnosis code is set.
Is the diagnosis code set?
Replace the multivision display (Refer to ) and then go to Step 3.
A poor connection, open circuit, or other intermittent malfunction in the CAN bus line between the camera-ECU and the multivision display (Refer to GROUP 00 - How to Cope with Intermittent Malfunction ).
 
STEP 3. Check whether the diagnosis code is set again.
Recheck if the diagnosis code is set.
(1) Erase the diagnosis code.
(2) Turn the ignition switch from the "LOCK" (OFF) position to the "ON" position.
(3) On completion, check that the diagnosis code is not set again.
Is the diagnosis code set?
Replace the camera-ECU (Refer to ).
The procedure is complete.
