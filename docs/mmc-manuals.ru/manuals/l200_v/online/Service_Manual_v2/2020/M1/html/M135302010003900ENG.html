Code No. C1200: FR Wheel Speed Sensor (Signal Open Circuit)
Code No. C1205: FL Wheel Speed Sensor (Signal Open Circuit)
Code No. C1210: RR Wheel Speed Sensor (Signal Open Circuit)
Code No. C1215: RL Wheel Speed Sensor (Signal Open Circuit)
 
 
caution
  • If there is any problem in the CAN bus lines, an incorrect diagnosis code may be set. Prior to this diagnosis, diagnose the CAN bus lines (Refer to GROUP 54C - CAN Bus Diagnostics Table ).
  • Whenever ECU is replaced, ensure that the CAN bus lines are normal.
  • When the hydraulic unit (integrated with ASTC-ECU) is replaced, always carry out the calibration of the steering wheel sensor and the G and yaw rate sensor (Refer to ).
 
 
OPERATION
  • A toothed rotor (for wheel speed detection) generates a voltage pulse as it moves across the pickup field of each wheel speed sensor.
  • The wheel speed sensor transmit the frequency of the pulses to the ASTC-ECU.
  • The ASTC-ECU modulates the amount of braking force individually applied to each wheel cylinder.
 
 
DIAGNOSIS CODE SET CONDITIONS
Diagnosis code No.C1200, C1205, C1210 or C1215 is set when signal is not input due to breakage of the wire of the wheel speed sensor without the output of the diagnosis code No.C1202, C1207, C1212 or C1217.
 
 
PROBABLE CAUSES
Current trouble
  • Damaged wiring harness and connectors
  • Malfunction of the wheel speed sensor
  • Malfunction of the hydraulic unit assembly (integrated with ASTC-ECU)
Past trouble
  • Carry out diagnosis with particular emphasis on wiring harness and connector failures between hydraulic unit assembly (integrated with ASTC-ECU) and the wheel speed sensor. For diagnosis procedures, refer to How to treat past trouble (GROUP 00 - How to Cope with Intermittent Malfunctions ).
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III CAN bus diagnosis.
Use M.U.T.-III to diagnose the CAN bus lines.
Is the check result normal?
Go to Step 3.
Repair the CAN bus lines (Refer to GROUP 54C - CAN Bus Diagnostics Table ). On completion, go to Step 2.
 
STEP 2. Diagnosis code recheck after repairing CAN bus lines.
(1) Erase the diagnosis code.
(2) Turn the ignition switch to the "ON" position.
(3) Drive the vehicle at 20 km/h or more.
noteThe ABS warning lamp does not turn OFF in some cases unless the vehicle runs at 20 km/h or higher.
(4) Check if the diagnosis code is set.
(5) Turn the ignition switch to the "LOCK" (OFF) position.
Is diagnosis code No. C1200, C1205, C1210 or C1215 set?
Go to Step 3.
The procedure is complete.
 
STEP 3. M.U.T.-III diagnosis code.
Check that diagnosis code No. C1202, C1207, C1212 or C1217 is also set.
Is diagnosis code No. C1202, C1207, C1212 or C1217 also set?
Perform the diagnosis for diagnosis code No. C1202, C1207, C1212 or C1217 (Refer to ).
Go to Step 4.

  • (When diagnosis code No. C1215 is set) Item No. 14: RL wheel speed sensor
Is the check result normal?
Go to Step 17.
Go to Step 5.

(3) Ignition switch: "ON"
