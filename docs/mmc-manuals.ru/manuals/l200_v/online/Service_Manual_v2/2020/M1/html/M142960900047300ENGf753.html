Code No.B1A25 Transponder ID unmatched
 
 
caution
  • When replacing the ECU, always check that the communication circuit is normal.
 
 
DIAGNOSTIC FUNCTION
The KOS&OSS-ECU sets diagnosis code No. B1A25 when the received key ID is different from the registered one or the key slot cannot communicate with the key correctly.
 
 
JUDGEMENT CRITERIA
KOS&OSS-ECU determines that the abnormality is present, if the key ID does not match the one registered in the KOS&OSS-ECU.
 
 
PROBABLE CAUSES
  • Malfunction of the keyless operation key
  • Malfunction of KOS&OSS-ECU
  • Key is registered to another vehicle
 
 
DIAGNOSIS PROCEDURE
STEP 1. All of the registered keyless operation key should allow the engine switch to be turned all positions
(1) Prepare all of the registered keyless operation key.
(2) Remove the batteries from all of the keyless operation keys (Refer to ).
(3) Insert one of the keyless operation key into the key slot, after the engine switch to "LOCK" (OFF) position.
(4) Check whether the engine switch can be turned from all positions.
(5) Turn the engine switch to "LOCK" (OFF) position again.
(6) Repeat the same procedure for the other keyless operation key. Check that all of the registered keyless operation key allow the engine switch to be turned all positions.
(7) Install the batteries to all the keyless operation keys.
Is the check result normal?
The engine switch can be turned all positions by operating all of the keyless operation key : Check if the diagnosis code is set. If the diagnosis code is erased, the diagnosis is complete. If not, go to Step 3.
The engine switch cannot be turned all positions by operating all of the keyless operation key : Check whether diagnosis code B2352 is set. If diagnosis code B2352 is set, go to the troubleshooting (Refer to ). Repeat Step 1. If diagnosis code B2352 is not set, go to Step 2.
 
STEP 2. Register the keyless operation key which cannot activate the engine switch again.
Register the keyless operation key which cannot activate the engine switch again (Refer to ).
Is the check result normal?
All of the keyless operation key can be registered again : Go to Step 3.
All of the keyless operation key cannot be registered again : The keyless operation key, which cannot be registered, is defective. Put it aside. Go to Step 3.
 
STEP 3. It should allow the engine switch to be turned all positions, after register all of the keyless operation key again.
(1) Remove the batteries from all of the keyless operation keys (Refer to ).
(2) Insert one of the keyless operation key into the key slot, after the engine switch to "LOCK" (OFF) position.
(3) Check whether the engine switch can be turned all positions.
(4) Turn the engine switch to "LOCK" (OFF) position again.
(5) Repeat the same procedure for the other keyless operation key. Check that all of the registered keyless operation key allow the engine switch to be turned all positions.
(6) Install the batteries to all the keyless operation keys.
Is the check result normal?
The engine switch can be turned all positions by operating all of the keyless operation key : Check if the diagnosis code is set. If the diagnosis code is erased, the diagnosis is complete. If not, go to Step 5.
The engine switch cannot be turned all positions by operating all of the keyless operation key : Go to Step 4.
 
STEP 4. Replace the keyless operation key which cannot activate the engine switch with a new one
(1) Replace the keyless operation key which cannot activate the engine switch with a new one.
noteExclude the keyless operation key which you put aside on Step 2.
(2) Register a new keyless operation key (Refer to ).
(3) Check that all of the registered keyless operation key allow the engine switch to be turned all positions.
(1) Remove the batteries from all of the keyless operation keys (Refer to ).
(2) Insert one of the keyless operation key into the key slot, after the engine switch to "LOCK" (OFF) position.
(3) Check whether the engine switch can be turned all positions.
(4) Turn the engine switch to "LOCK" (OFF) position again.
(5) Repeat the same procedure for the other keyless operation key. Check that all of the registered keyless operation key allow the engine switch to be turned all positions.
(6) Install the batteries to all the keyless operation keys.
Is the check result normal?
The engine switch can be turned all positions by operating all of the keyless operation key : Check if the diagnosis code is set. If the diagnosis code is erased, the diagnosis is complete. If not, go to Step 5.
The engine switch cannot be turned all positions by operating all of the keyless operation key : Go to Step 5.
 
STEP 5. It should allow the engine switch to be turned all positions, after replace the KOS&OSS-ECU, and then register all of the keyless operation key again.
(1) Replace KOS&OSS-ECU and register the ID codes (Refer to ).
(2) Register all of the keyless operation key again (Refer to ).
noteExclude the keyless operation key which you put aside on Step 2.
(3) Check that all of the registered keyless operation key allow the engine switch to be turned all positions.
(1) Remove the batteries from all of the keyless operation keys (Refer to ).
(2) Insert one of the keyless operation key into the key slot, after the engine switch to "LOCK" (OFF) position.
(3) Check whether the engine switch can be turned all positions.
(4) Turn the engine switch to "LOCK" (OFF) position again.
(5) Repeat the same procedure for the other keyless operation key. Check that all of the registered keyless operation key allow the engine switch to be turned all positions.
(6) Install the batteries to all the keyless operation keys.
Is the check result normal?
The engine switch can be turned all positions by operating all of the keyless operation key : Go to Step 6.
The engine switch cannot be turned all positions by operating all of the keyless operation key : Go to Step 1.
 
STEP 6. Check whether the diagnosis code is set again.
(1) Erase the diagnosis code.
(2) Check if the diagnosis code is set.
Is the diagnosis code set?
Go to Step 1.
The diagnosis is complete.
