Code No. P0967: Clutch control solenoid (SL1) system (High current)
 
 
 
 
OPERATION
  • Solenoid closes or opens according to the signals from the A/T-ECU.
  • The A/T-ECU energises or deenergises solenoid, based on input signals such as accelerator pedal position sensor opening angle, inhibitor switch, etc.
 
 
DIAGNOSIS CODE SET CONDITIONS
Check Condition
  • Ignition switch is in the "ON" position.
Judgement Criterion
  • Short circuit in the clutch control solenoid (SL1) system power supply line is detected for 0.5 seconds or more.
 
 
PROBABLE CAUSES
  • Damaged harness wires and connectors
  • Malfunction of the transmission wiring harness
  • Malfunction of the valve body assembly [clutch control solenoid (SL1)]
  • Malfunction of the A/T-ECU
 
 
DIAGNOSIS PROCEDURE
STEP 1. Measure the resistance at the A/T-ECU connector.
(1) Disconnect the A/T-ECU connector, and measure at the wiring harness side.
(2) Check the resistance between the A/T-ECU connector terminals SL1 and SL1G.
OK: 5.0 - 5.6 Ω (A/T fluid temperature 20°C)
(3) Check the resistance between the A/T-ECU connector terminal SL1 and body earth.
OK: No continuity
(4) Check the resistance between the A/T-ECU connector terminal SL1G and body earth.
OK: No continuity
Is the check result normal?
Go to Step 4.
Go to Step 2.
 
STEP 2. Check for short circuit between A/T-ECU connector and A/T control solenoid valve assembly connector.
Check the following lines between A/T-ECU connector and A/T control solenoid valve assembly connector.
  • SL1
  • SL1G
Is the check result normal?
Go to Step 3.
Repair the connector(s) or the wiring harness.
 
STEP 3. Solenoid check
Check the clutch control solenoid (SL1) (Refer to ).
Is the check result normal?
Replace the transmission wiring harness.
Replace the valve body assembly.
 
STEP 4. Check whether the diagnosis code is set again.
Check again if the diagnosis code No. P0967 is set.
Is the diagnosis code No. P0967 set?
Replace the A/T-ECU.
Intermittent malfunction (Refer to GROUP 00 - How to Cope with Intermittent Malfunction ).
