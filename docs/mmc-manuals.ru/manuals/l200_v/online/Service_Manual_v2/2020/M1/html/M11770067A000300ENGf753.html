Code No. U0101: A/T-ECU Time-out <A/T>
 
 
caution
  • If diagnosis code No. U0101 is set in the AS&G-ECU, surely perform CAN bus line diagnosis.
  • Replace ECU after certainly confirming that the communication circuits are normal.
 
 
TROUBLE JUDGEMENT
Check Conditions
  • 2 seconds has passed after the ignition switch is turn to the "ON" position.
  • The battery voltage is more than 10 V and is less than 16 V.
  • After 2 seconds pass from when the battery voltage is more than 10 V.
Judgement Criterion
  • Unable to receive the signals from the A/T-ECU for 4 seconds.
 
 
COMMENTS ON TROUBLE SYMPTOM
Current malfunction
  • The failure is possibly caused by malfunction of the harness and connector in CAN bus line between AS&G-ECU and A/T-ECU, by malfunction in the power supply system of A/T-ECU, in A/T-ECU itself, in AS&G-ECU.
Past malfunction
  • Proceed with troubleshooting focusing on malfunction of the harness and connector in CAN bus line between AS&G-ECU and A/T-ECU and malfunction in the power supply system of A/T-ECU. For diagnosis procedure, refer to how to cope with past malfunctions (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Treat Past Trouble ).
    noteIn the case of past malfunctions, M.U.T.-III CAN bus line diagnosis cannot detect a malfunction even if there is a malfunction in CAN bus. Therefore, check CAN bus line as in the case of response to intermittent malfunctions (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ). It is possible to narrow down areas with a high possibility of malfunctions from diagnosis codes set in each ECU that performs CAN communication (Refer to GROUP 54C - Explanation about the M.U.T.-III CAN Bus Diagnostics ).
 
 
FAIL-SAFE AND BACKUP FUNCTION
In the normal operation, the auto stop is prohibited. If the malfunction is detected during the auto stop, carry out the forcible auto start.
 
 
PROBABLE CAUSES
  • Failed harness and connector.
  • Failed A/T-ECU.
  • Failed AS&G-ECU.
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III CAN bus diagnosis
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Use M.U.T.-III to perform CAN bus line diagnosis.
Is the check result normal?
Go to Step 2.
Repair CAN bus line (Refer to GROUP 54C - Troubleshooting <CAN>, CAN Bus Diagnostics Table ). Then go to Step 6.
 
STEP 2. Check for diagnosis code of other system
Use M.U.T.-III to check if the diagnosis code is set in the A/T-ECU.
Is any diagnosis code set in the A/T-ECU?
Perform troubleshooting of the A/T-ECU (Refer to GROUP 23C - Troubleshooting <A/T>, Diagnosis Code Chart ).
Go to Step 3.
 
STEP 3. Check for diagnosis code of other system
Use M.U.T.-III to check that the other ECU performing CAN communication with A/T-ECU sets diagnosis code No. U0101 (Refer to GROUP 54C - CAN-related Diagnosis Code Table ).
Is diagnosis code No. U0101 set in the other ECU?
Go to Step 4.
Go to Step 5.
 
STEP 4. Diagnosis code recheck after erasing diagnosis code
Check again if the diagnosis code is set.
(1) Use M.U.T.-III to erase the diagnosis code.
(2) Turn the ignition switch to "LOCK" (OFF) position.
(3) Turn the ignition switch to "ON" position.
(4) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. U0101 set?
Replace the A/T-ECU. Then go to Step 6.
Intermittent malfunction in CAN bus line between AS&G-ECU and A/T-ECU (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
 
STEP 5. Diagnosis code recheck after erasing diagnosis code
Check again if the diagnosis code is set.
(1) Use M.U.T.-III to erase the diagnosis code.
(2) Turn the ignition switch to "LOCK" (OFF) position.
(3) Turn the ignition switch to "ON" position.
(4) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. U0101 set?
Replace the AS&G-ECU. Then go to Step 6.
Intermittent malfunction in CAN bus line between AS&G-ECU and A/T-ECU (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
 
STEP 6. Diagnosis code recheck after erasing diagnosis code
Check again if the diagnosis code is set.
(1) Use M.U.T.-III to erase the diagnosis code.
(2) Turn the ignition switch to "LOCK" (OFF) position.
(3) Turn the ignition switch to "ON" position.
(4) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. U0101 set?
Go to Step 1.
Check end.
