WARNING INDICATOR
If a malfunction relating to the A/T system occurs, the symbol (A) will be displayed on the information screen in the multi-information display.
Check if the diagnosis code is set when the symbol (A) continues being displayed in the information screen in the multi-information display.
noteOnly when the inhibitor switch is defective, symbol "A" will flash in the selector lever position indicator.
Symbol (B) (on the multi-information display) depends on the A/T fluid temperature, which A/T fluid temperature sensor informs of. When the symbol (B) is displayed in the information screen in the multi-information display, the A/T fluid temperature is high. (When the A/T fluid temperature sensor 1 detects the A/T fluid temperature of approximately 136°C or higher, symbol (B) is turned on. When it drops below approximately 131°C, symbol (B) is turned off automatically. Meanwhile, when the A/T fluid temperature sensor 2 detects the A/T fluid temperature of approximately 138°C or higher, symbol (B) is turned on. When it drops below approximately 128°C, symbol (B) is turned off automatically.)
noteWhen A/T fluid reaches a high temperature, the shift program for the "D" range will be modified to lower the A/T fluid temperature. It is control to lower temperature of A/T fluid, and this phenomenon is not trouble. If temperature of A/T fluid falls, A/T returns to normal movement.

In the emergency mode, the following controls will be disabled.
  • Learning control
  • Lock-up control
  • Garage shift control
  • Auto stop control <Vehicles with AS&G system>
  • Shift control
  • Multiplex shift control

Emergency mode 2
Emergency mode 2 is the emergency mode activated when the functional failure is detected. Especially for clutch control solenoid (SL1) OFF stuck condition. Before moving the selector lever from the "D" position, 5th gear condition is kept. After moving the selector lever from the "D" position, and the vehicle speed for 3rd gear shifting is satisfied, all solenoids are turned off and the gear is ready to be shifted to 3rd when the selector lever is moved to "D" position. When the neutral condition is detected in the 3rd gear condition, the gear is shifted to 5th.
