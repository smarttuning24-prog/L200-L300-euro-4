DISASSEMBLY SERVICE POINT
 
 
<<A>> OIL TEMPERATURE SENSOR REMOVAL
Using the special tool water temperature sensor wrench (MB992042) to remove the oil temperature sensor.
 
 
 
 
REASSEMBLY SERVICE POINT
 
 
>>A<< OIL TEMPERATURE SENSOR INSTALLATION
1. Apply the sealant over the screw sections on the oil temperature sensor.
Special sealant: MITSUBISHI MOTORS GENUINE Part No. MZ100388 or equivalent
2. Using the special tool water temperature sensor wrench (MB992042) to install the oil temperature sensor.
