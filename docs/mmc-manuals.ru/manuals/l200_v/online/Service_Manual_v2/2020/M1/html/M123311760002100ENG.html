DISASSEMBLY
 
 
1. FR PLANETARY RING GEAR FLANGE SUB-ASSEMBLY REMOVAL
Using needle-nose pliers, remove the snap ring and the FR planetary ring gear flange sub-assembly from the FR planetary ring gear.
 

 
 
2. CENTRE PLANETARY RING GEAR REMOVAL
1. Using a screwdriver, remove the snap ring from the centre planetary ring gear flange with centre planetary ring gear.
2. Remove the centre planetary ring gear from the centre planetary ring gear flange.
 

 
 
3. RR PLANETARY RING GEAR FLANGE WITH RR PLANETARY RING GEAR REMOVAL
1. Remove the RR planetary ring gear flange with RR planetary ring gear and the thrust bearing race from the intermediate shaft
2. Remove the thrust needle roller bearing and the thrust bearing race from the intermediate shaft.
 

 
 
4. RR PLANETARY RING GEAR REMOVAL
1. Using a screwdriver, remove the snap ring from the RR planetary ring gear flange with RR planetary ring gear.
2. Remove the RR planetary ring gear from the RR planetary ring gear flange.
 
 
 
 
REASSEMBLY
 
 
INSPECTION BEFORE REASSEMBLY
Refer to Planetary Gear Inspection .
  • Inspect FR Planetary Ring Gear Flange Sub-assembly
  • Inspect RR Planetary Ring Gear Flange
  • Inspect FR Planetary Gear Assembly
  • Inspect Sun Gear Input Drum
  • Inspect Centre Planetary Gear Assembly
  • Inspect RR Planetary Gear Assembly
  • Inspect Intermediate Shaft

 
 
1. RR PLANETARY RING GEAR FLANGE INSTALLATION
1. Install the RR planetary ring gear flange to the RR planetary ring gear.
2. Using a screwdriver, install the snap ring to the RR planetary ring gear flange with RR planetary ring gear.
 

 
 
2. CENTRE PLANETARY RING GEAR FLANGE INSTALLATION
1. Install the centre planetary ring gear flange to the centre planetary ring gear.
2. Using a screwdriver, install the snap ring to the centre planetary ring gear flange with centre planetary ring gear.
 

 
 
3. FR PLANETARY RING GEAR FLANGE SUB-ASSEMBLY INSTALLATION
Set the snap ring onto the FR planetary ring gear flange sub-assembly. Using needle-nose pliers, insert the snap ring into the FR planetary ring gear while squeezing it, and then expand the snap ring to install it.
