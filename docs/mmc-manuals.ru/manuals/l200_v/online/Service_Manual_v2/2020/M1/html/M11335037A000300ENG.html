Code No. P0089: Suction Control Valve Stuck
 
 
 
 
FUNCTION
Refer to Code No. P0003: Suction Control Valve Open .
 
 
TROUBLE JUDGEMENT
Check Conditions
  • Battery positive voltage is 8 to 16 V.
  • 2 seconds later after the engine has started up.
Judgement Criterion
  • The actual rail pressure remains higher than the objective rail pressure by 10 MPa or more for a certain interval.
 
 
FAIL-SAFE AND BACKUP FUNCTION
  • The rail pressure is controlled in the range between 50 and 100 MPa.
  • The fuel injection amount is restricted (Output restriction).
  • The open angle of the accelerator pedal is restricted.
  • The throttle valve is closed (Fixed opening degree).
  • The EGR control is stopped (Fully closed).
  • The DPF regeneration is prohibited.
  • The AS&G control is prohibited.
  • The small injection quantity learning is prohibited.
  • The speed limiter control is prohibited.
  • The cruise control is prohibit.
 
 
PROBABLE CAUSES
  • Failed suction control valve
  • Failed fuel system
  • Failed fuel pressure regulator of fuel tank pump
  • Use of inappropriate fuel
  • Failed engine-ECU
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III actuator test
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Use M.U.T.-III to perform the actuator test (Refer to ).
  • Item No. 19: Suction control valve
Is the check result normal?
Go to Step 2.
Go to Step 4.
 
STEP 2. Make a check on the following items
noteForeign materials mixed in the suction control valve can possibly cause the stuck suction control valve. Check the followings.
  • Check it for the fuel conditions (mixed foreign materials, smell, colour or viscosity). If the fuel conditions are abnormal, replace the fuel.
  • Check fuel filter itself.
Are the check results normal?
Go to Step 3.
Make a repair or replacement for items whose checking results are abnormal.
 
STEP 3. Diagnosis code recheck after erasing diagnosis code
Check again if the diagnosis code is set.
(1) Use M.U.T.-III to erase the diagnosis code.
(2) Turn the ignition switch to "LOCK" (OFF) position.
(3) Turn the ignition switch to "ON" position and start the engine.
(4) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. P0089 set?
Go to Step 5.
Intermittent malfunction (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
 
STEP 4. Diagnosis code recheck after replace the suction control valve
After replacing the suction control valve, check again if the diagnosis code is set.
(1) Before replacing the suction control valve, check it for the fuel conditions (mixed foreign materials, smell, colour or viscosity). If the fuel conditions are abnormal, replace the fuel.
(2) Replacing the suction control valve.
(3) After the suction control valve, learn the supply pump (Refer to GROUP 00 - Precautions Before Service, What The Common Rail Engine Learns ).
(4) After learning the supply pump assembly, use M.U.T.-III to erase the diagnosis code.
(5) Turn the ignition switch to "LOCK" (OFF) position.
(6) Turn the ignition switch to "ON" position and start the engine.
(7) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. P0089 set?
Replace the engine-ECU.
The check is end.
 
STEP 5. Diagnosis code recheck after replace the fuel pump module assembly
After replacing the fuel pump module assembly, check again if the diagnosis code is set.
(1) The fuel pressure regulator of the fuel tank pump can possibly be failed. Replace the fuel pump module assembly.
(2) Use M.U.T.-III to erase the diagnosis code.
(3) Turn the ignition switch to "LOCK" (OFF) position.
(4) Turn the ignition switch to "ON" position and start the engine.
(5) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. P0089 set?
Replace the engine-ECU.
The check is end.
