Inspection Procedure 11: The Speed Limiter is not Deactivated when the Accelerator Pedal is Depressed Fully <Vehicles with Speed Limiter>
 
 
 
 
COMMENTS ON TROUBLE SYMPTOM
The accelerator pedal position sensor or engine-ECU may also be defective.
 
 
PROBABLE CAUSES
  • Damaged harness or connector.
  • Malfunction of the accelerator pedal position sensor.
  • Malfunction of the engine-ECU.
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check the diesel fuel system diagnosis code
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Use M.U.T.-III to check diesel fuel system diagnosis code (Refer to GROUP 13A - Troubleshooting, Diagnosis Function ).
Is any diagnosis code set?
Repair the diesel fuel system (Refer to GROUP 13A - Troubleshooting, Inspection Chart for Diagnosis Code ). Then go to Step 4.
Go to Step 2.
 
STEP 2. Check the diesel fuel system data list
Use M.U.T.-III to check the diesel fuel system data list (Refer to GROUP 13A - Troubleshooting, Data List Reference Table ).
  • Item No. 9: Accelerator pedal position sensor (main)
OK: Approximately 4.33 V or more (Depress the accelerator pedal fully)
Is the check result normal?
Go to Step 3.
Replace the accelerator pedal assembly (Refer to ). Then go to Step 4.
 
STEP 3. Check the symptom
Is the speed limiter deactivated when the accelerator pedal is depressed fully?
It can be assumed that this malfunction is intermittent (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunction ).
Replace the engine-ECU (Refer to GROUP 13A - Engine-ECU ). Then go to Step 4.
 
STEP 4. Check the symptom
Is the speed limiter deactivated when the accelerator pedal is depressed fully?
The procedure is complete.
Return to Step 1.
