AS&G OFF DISPLAY SCREEN
When the AS&G OFF switch is depressed, the AS&G system should be turned OFF and the AS&G OFF display screen in the multi-information display of the combination meter be turned ON. When the following items in the table, related to the AS&G system, are abnormal, the AS&G OFF display screen should blink. When the AS&G OFF display screen continues to blink, check the output of the diagnosis code. After the ignition switch is turned to the "ON" position, the AS&G OFF display screen should be turned ON for several seconds so that the display can be checked for the normal operation.
noteWhen the engine-ECU or A/T-ECU detects a diagnosis code, the AS&G OFF display screen is also blinked.

noteWhen detecting a malfunction, the AS&G-ECU should store it. When detecting the same malfunction again after the next engine start, the AS&G-ECU should blink the AS&G indicator because the AS&G-ECU judges the malfunction surely occurs. For the items marked with a "*" in the diagnosis code number column, the AS&G OFF indicator blinks on the first detection of the malfunction.
 
 
 
AS&G DISPLAY SCREEN
When the specified conditions are satisfied during the engine operation, the AS&G-ECU should send engine stop request signal to the engine-ECU and the engine-ECU should stop the engine. The AS&G display screen in the multi-information display will come on when the AS&G system is ready to activate or has activated with the engine running.If the following items in the table, related to the AS&G system, are applied, the AS&G display screen should blink. When the AS&G display screen continues to blink, carry out the check according to the applied item. After the ignition switch is turned "ON" position, the AS&G OFF display screen in the multi-information display of the combination meter should be turned ON for several seconds so that the indicator can be checked for the normal operation.
Items to blink AS&G display screen
ItemsInspection contents
The driver’s door is opened.Check the driver’s door is securely closed.
The hood is opened.Check the hood is securely closed. Use the key/engine switch to start the engine because the auto start should be prohibited if the hood is opened during the auto stop.
The driver’s seat belt is unfastened.Check the driver’s seat belt is securely fastened.
The gearshift lever is operated during the auto stop (The combination meter sounds the warning buzzer except the neutral position). <M/T>Check the gearshift lever is in the neutral position.
During auto stop, selector lever is placed on "P" position and brake pedal is released. <A/T>Check the selector lever is in the "P" position. In addition, auto start enable when the brake pedal depressed again or do the selector lever any place other than "P".

INSPECTION WITH M.U.T.-III DATA LIST
Reading data list using M.U.T.-III enables you to check the vehicle body harnesses and components for abnormalities. Sensor input or output to the actuator can be displayed in numerical values or graphs in the data list.
 
 
Diagnosis Procedure
1.
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
After the ignition switch is in "LOCK" (OFF) position, connect the M.U.T.-III to the diagnosis connector.
2. Turn the ignition switch to the "ON" position.
3. Select "AS&G" from system select screen of the M.U.T.-III.
4. Select "Data List" from AS&G screen.
5. Use the data list function to perform the check (Refer to ).
6. If any abnormality is found, inspect and repair vehicle body harnesses and components.
7. After repair, check that input and output have returned to normal.
8. If a diagnosis code which is caused by inspection and repair work is generated, erase the diagnosis code.
9. Perform a test run to check that the malfunction phenomenon is resolved.
 
 
 
FREEZE FRAME DATA
When the AS&G-ECU detects a malfunction and sets a diagnosis code, it also stores a current status of the engine. This function is called "Freeze frame data". By analysing this "Freeze frame data" with the M.U.T.-III, an effective troubleshooting can be performed.The display items of freeze frame data are shown below.
