Code No. U010E: Lost Communication With Reductant Control Module <4N14>
 
 
caution
  • If diagnosis code No. U010E is set in the engine-ECU, surely perform CAN bus line diagnosis.
  • Replace ECU after certainly confirming that the communication circuits are normal.
 
 
TROUBLE JUDGEMENT
Check Condition
  • Ignition switch is in "ON" position.
  • Battery positive voltage is more than 10 V and is less than 16 V.
  • 6 seconds have passed after the above 2 conditions are met.
Judgement Criterion
  • The signals cannot be received from Dosing Control Unit (DCU) for predetermined time.
 
 
FAIL-SAFE AND BACKUP FUNCTION
None.
 
 
COMMENT ON TROUBLE SYMPTOM
Current malfunction
  • The failure is possibly caused by malfunction of the harness and connector in CAN bus line between engine-ECU and DCU, by malfunction in the power supply system of DCU, in DCU itself, in engine-ECU.
Past malfunction
  • Proceed with troubleshooting focusing on malfunction of the harness and connector in CAN bus line between engine-ECU and DCU and malfunction in the power supply system of DCU. For diagnosis procedure, refer to how to cope with past malfunctions (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Treat Past Trouble ).
    noteIn the case of past malfunctions, M.U.T.-III CAN bus line diagnosis cannot detect a malfunction even if there is a malfunction in CAN bus. Therefore, check CAN bus line as in the case of response to intermittent malfunctions (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ). It is possible to narrow down areas with a high possibility of malfunctions from diagnosis codes set in each ECU that performs CAN communication (Refer to GROUP 54C - Explanation about the M.U.T.-III CAN Bus Diagnostics ).
 
 
PROBABLE CAUSES
  • Failed harness and connector
  • Failed DCU
  • Failed engine-ECU
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check "Communication with no ECUs is possible (ENG-LOCAL CAN)"
Check urea SCR system inspection procedure 1: Communication with no ECUs is possible (ENG-LOCAL CAN) [Refer to GROUP 17 - Emission Control <Diesel>, Urea SCR Troubleshooting, Inspection Procedure 1: Communication with no ECUs is possible (ENG-LOCAL CAN) ].
Is the check result normal?
Go to Step 2.
Repair.
 
STEP 2. Diagnosis code recheck after erasing diagnosis code
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Check again if the diagnosis code is set.
(1) Use M.U.T.-III to erase the diagnosis code.
(2) Turn the ignition switch to "LOCK" (OFF) position.
(3) Turn the ignition switch to "ON" position.
(4) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. U010E set?
Go to Step 3.
Intermittent malfunction (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
 
STEP 3. Diagnosis code recheck after replace the DCU
After replacing the DCU, check again if the diagnosis code is set.
(1) Replacing the DCU.
(2) Use M.U.T.-III to erase the diagnosis code.
(3) Turn the ignition switch to "LOCK" (OFF) position.
(4) Turn the ignition switch to "ON" position.
(5) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. U010E set?
Replace the engine-ECU.
The check is end.
