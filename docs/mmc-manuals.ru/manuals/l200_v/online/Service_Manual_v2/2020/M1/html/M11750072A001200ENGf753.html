REMOVAL AND INSTALLATION
 
 
cautionWhen the EGR cooler is replaced, use the M.U.T.-III to initialise the learning value (Refer to GROUP 00 - Precautions Before Service, Initialisation Procedure for Learning Value in Common Rail Engine ).
Pre-removal Operation
  • Under Skid Plate and Engine Room Under Cover Front A and B Removal (Refer to GROUP 51 - Under Cover ).
  • Engine Coolant Draining (Refer to GROUP 14 - On-vehicle Service, Engine Coolant Replacement ).
Post-installation Operation
  • Engine Coolant Refilling (Refer to GROUP 14 - On-vehicle Service, Engine Coolant Replacement ).
  • Under Skid Plate and Engine Room Under Cover Front A and B Installation (Refer to GROUP 51 - Under Cover ).

INSTALLATION SERVICE POINTS
 
 
>>A<< EGR VALVE PIPE GASKET/EGR VALVE PIPE A/EGR VALVE PIPE GASKET/EGR COOLER BYPASS ACTUATOR GASKET/EGR COOLER BYPASS ACTUATOR INSTALLATION
1. Install the EGR valve pipe gasket with its projection being positioned as shown in the figure.
2. Temporarily install the EGR valve pipe A to the cylinder head.
3. Install the EGR valve pipe gasket with its projection being positioned as shown in the figure.
4. Install the EGR cooler bypass actuator gasket.
5. Temporarily install the EGR cooler bypass actuator to the EGR cooler.
6. Check that there is no clearance among each component. Temporarily tighten the bolts, 1, 5 and 7 as shown in the figure.
Temporary tightening torque: 1.3 ± 0.7 N·m
7. Tighten each mounting bolts to the specified torque in the order as shown in the figure.
Tightening torque: 20 ± 2 N·m
 
 
 
>>B<< EGR VALVE PIPE GASKET/EGR VALVE PIPE B/EGR VALVE PIPE GASKET/EGR VALVE GASKET/EGR COOLER BYPASS VALVE INSTALLATION
1. Install the EGR valve pipe gasket with its projection being positioned as shown in the figure.
2. Temporarily install the EGR valve pipe B to the EGR cooler bypass actuator.
3. Install the EGR valve pipe gasket with its projection being positioned as shown in the figure.
4. Install the EGR valve gasket.
5. Temporarily install the EGR valve to the inlet manifold.
6. Check that there is no clearance among each component. Temporarily tighten the bolts, 1, 5 and 7 as shown in the figure.
Temporary tightening torque: 1.3 ± 0.7 N·m
7. Tighten each mounting bolts to the specified torque in the order as shown in the figure.
Tightening torque: 20 ± 2 N·m
