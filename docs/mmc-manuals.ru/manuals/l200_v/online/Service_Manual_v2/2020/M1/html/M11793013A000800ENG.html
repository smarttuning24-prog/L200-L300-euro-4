Urea SCR System Warning Screen/Indicator, Engine Warning Lamp
When the abnormal condition occurs with respect to the items of urea SCR system shown in the table, the warning screen will display, the warning indicator will blink or the engine warning lamp will illuminate. The warning indicator flashes interlocking with the warning screen display. If the warning screen continues to be displayed on the information screen or if the engine warning lamp remains illuminated, check the (provisional) diagnosis code is set.
The warning screen is displayed once the system is determined to be defective, and it will continue to be displayed until the DCU determines that the system is normal. When the diagnosis code is erased by the M.U.T.-III, the engine warning lamp will go off, but the warning screen will not disappear.
In addition, when the warning screen is displayed, the countdown of the travelable distance starts at the same time. If you keep driving with the warning screen displayed until the travelable distance becomes 0 km, the engine restart is prohibited. If the ignition switch is "ON" position, driving can be continued even when the engine restart is prohibited. However, the engine cannot be started once the ignition switch is turned to the "LOCK" (OFF) position.

Inspection with M.U.T.-III Data List and Actuator Teat
Reading data list or performing an actuator test using M.U.T.-III enables you to check the vehicle body harnesses and components for abnormalities. Sensor input or output to the actuator can be displayed in numerical values or graphs in the data list while in the actuator test the actuator can be forced to drive (ON/OFF driving in the specified time).
 
 
Diagnosis procedure
1.
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Connect M.U.T.-III to perform inspection with the data list and actuator test function.
2. If any abnormality is found, inspect and repair vehicle body harnesses and components.
3. After repair, check that input and output have returned to normal.
4. If a failure code which is caused by inspection and repair work is generated, erase the failure code.
5. Disconnect M.U.T.-III and perform a test run to check that the malfunction phenomenon is resolved.
 
 
 
 
Provisional Diagnosis Code Confirmed
If detecting the malfunction during the first drive cycle, the DCU temporarily sets the diagnosis code as the provisional diagnosis code. If detecting the same malfunction during the 2 more drive cycles, the DCU determines that the malfunction exists. The DCU sets the diagnosis code.
On the M.U.T.-III, it is possible to display the set provisional diagnosis code which the DCU had detected during the first drive cycle. This makes it possible to confirm in 1 drive cycle whether the malfunction could happen again after the repair.
 
 
Confirmation method
1.
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
After the ignition switch is in "LOCK" (OFF) position, connect the M.U.T.-III to the diagnosis connector.
2. Turn the ignition switch to the "ON" position.
3. Select "DCU" from System Select Screen of the M.U.T.-III.
4. Select "Special Function" from DCU Screen.
5. Select "Provisional DTCs" from Special Function Screen.
 
 
 
 
How to Release the Engine Restart Prohibition Status
When the vehicle is driven for a predetermined distance after the warning screen of the urea SCR system is displayed on the combination meter, the engine restart is prohibited. When driving the vehicle to make the DCU determine whether the system is normal after carrying out repairs, the engine can be restarted by following the steps below.
noteWhen this operation is executed, the travelable distance is reset to 50 km. Therefore, the command needs to be executed only when the engine restart is prohibited.
 
 
Releasing Procedure
1.
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
After the ignition switch is in "LOCK" (OFF) position, connect the M.U.T.-III to the diagnosis connector.
2. Turn the ignition switch to the "ON" position.
3. Select "MPI/GDI/DIESEL" from System Select Screen of the M.U.T.-III.
4. Select "Special Function" from MPI/GDI/DIESEL Screen.
5. Select "Initialization" from Special Function Screen.
6. Carry out the "Item No. 54: SCR System Repair Completed" from Initialization Screen.
 
 
 
 
Freeze Frame Data
When the DCU detects a malfunction and sets a diagnosis code, it also stores a current status of the urea SCR system.
This function is called "Freeze frame data." By analysing this "Freeze frame data" with the M.U.T.-III, an effective troubleshooting can be performed.
The display items of freeze frame data are shown below.
