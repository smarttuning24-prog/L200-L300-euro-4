Code No. C1809: System temporary not available
 
 
cautionWhenever the ECU is replaced, ensure that the CAN bus lines are normal.
 
 
OPERATION
In very rare case, an exception that the camera image calculation is impossible is detected by the self-diagnosis function of the FCM/LDW/AHB-ECU. In this case, the FCM/LDW/AHB-ECU stops its function temporarily and reset itself internally. As a result, if the FCM/LDW/AHB-ECU judges that there is no problem, it resumes the function as normal. At this time, the diagnosis code No. C1809 is set as a history, but it is not necessary to replace the FCM/LDW/AHB-ECU because this is not a malfunction. When there is a problem left after the internal reset, the FCM/LDW/AHB-ECU stops its function and the diagnosis code No. C1751 is set.
 
 
DIAGNOSIS CODE SET CONDITIONS
Check Condition
  • Ignition switch is in "ON" position.
Judgement Criterion
  • An exception that the camera image calculation is impossible is detected by the self-diagnosis function of the FCM/LDW/AHB-ECU.
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check the M.U.T.-III diagnosis code.
Use M.U.T.-III to check that the diagnosis code No. C1751 is set in FCM/LDW/AHB-ECU.
Is the diagnosis code No. C1751 set?
Perform the troubleshooting for the diagnosis code No. C1751 (Refer to ).
The procedure is complete. This is due to the temporary stop of the FCM/LDW/AHB-ECU control as a result of the self-diagnosis function and there is no malfunction.
