AMMONIA PURGE
2020 / L200/L200 SPORTERO / EU - Workshop Manual
17 -ENGINE AND EMISSION CONTROL > EMISSION CONTROL <DIESEL> > UREA SCR SYSTEM<4N14> > AMMONIA PURGE
1.
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Connect the M.U.T.-III to the diagnosis connector.
2. Start the engine and run at idle.
3. Select "MPI/GDI/DIESEL" from System Select Screen of the M.U.T.-III.
4. From MPI/GDI/DIESEL Screen, select "Special Function".
5. From Special Function Screen, select "Ammonia purge".
6. Carry out the "Item No. 56: Ammonia purge" from Ammonia purge Screen of the M.U.T.-III.
noteThe idle speed increases while ammonia purge is in progress (approximately 2,200 r/min).
7. Return to System select Screen.
8. Select "DCU" from System Select Screen of the M.U.T.-III.
9. From DCU Screen, select "Special Function".
10. From Special Function Screen, select "Ammonia purge".
11. Carry out the "Item No. 1: Stop Urea injection" from Ammonia purge Screen of the M.U.T.-III.
note
  • Do not disconnect the M.U.T.-III from the diagnosis connector during the ammonia purge.
  • When the ammonia purge is completed, the value of the M.U.T.-III data list item No. 76: Estimated ammonia load decreases and smaller value is set.
12. Confirm that the idle operation returns to the normal engine speed, and then turn the ignition switch to the "LOCK" (OFF) position.
noteWhen 30 minutes pass after ammonia purge is performed, the engine speed returns to the normal idle operation.
13. Remove the M.U.T.-III.
