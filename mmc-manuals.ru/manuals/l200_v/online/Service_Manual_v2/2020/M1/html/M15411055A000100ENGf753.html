Code No.B1703: Transponder ID unmatched
2020 / L200/L200 SPORTERO / EU - Workshop Manual
54 -CHASSIS ELECTRICAL > 54A-CHASSIS ELECTRICAL > IMMOBILIZER SYSTEM<VEHICLES WITHOUT KOS> > DIAGNOSIS CODE PROCEDURES > B1703 TRANSPONDER DATA INCONSISTENT
 
 
cautionWhenever the ECU is replaced, ensure that the input and output signal circuits are normal.
 
 
TROUBLE JUDGEMENT
When the key ID that is sent from the transponder does not correspond to the key ID registered in the ETACS-ECU, the ETACS-ECU sets diagnosis code No.B1703.
noteDiagnosis code No.B1703 is always set along with MPI system diagnosis code No.P0513.
 
 
 
PROBABLE CAUSES
  • Ignition key (key ID) is not registered
  • Malfunction of ETACS-ECU
 
 
 
 
DIAGNOSIS PROCEDURE
Register all the ignition key IDs again, and recheck the diagnosis code.
(1) Register all the ignition keys again (Refer to ).
(2) Turn the ignition switch from the "LOCK" (OFF) position to the ON position.
(3) Check that diagnosis code No.B1703 is not set again.
Is diagnosis code No.B1703 set?
Replace the ETACS-ECU (Refer to ). After registering the ID codes, check if the diagnosis code is set.
The diagnosis is complete. If another diagnosis code is set, refer to Diagnosis Code Chart .
