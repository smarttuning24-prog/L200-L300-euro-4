Inspection procedure B-1: Lamp reminder buzzer does not work normally.
 
 
cautionWhenever the ECU is replaced, ensure that the input and output signal circuits are normal.
 
 
COMMENTS ON TROUBLE SYMPTOM
The ETACS-ECU uses the input signals below to send a lamp reminder buzzer request signal to the combination meter.
  • Ignition switch (IG1)
  • Driver's door switch
  • Tail lamp switch
  • Headlamp switch
If the buzzer does not sound properly, the input signal(s) are abnormal or the combination meter or the ETACS-ECU may be defective.
 
 
PROBABLE CAUSES
  • Malfunction of the driver's door switch
  • Malfunction of the column switch
  • Malfunction of the combination meter
  • Malfunction of the ETACS-ECU
  • Damaged harness wires and connectors
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III diagnosis code.
When the ignition switch is turned to the "LOCK" (OFF) position, check that the ETACS-ECU does not set the diagnosis code.
Is the diagnosis code set?
Refer to diagnosis code chart .
Go to Step 2.
 
STEP 2. M.U.T.-III data list
Check the input signals below, which are related to the lamp reminder buzzer function.
  • Turn the ignition switch to the "LOCK" (OFF) position.
  • Open the driver’s door.
  • Turn the tail lamp or headlamp switch to the ON position.
 
Item No.Item nameNormal condition
19Driver's door switchON
30Ignition switch(IG1)OFF
60Headlamp outputAny of items is ON
59Tail lamp output
OK: Normal conditions are displayed for all the items.
Is the check result normal?
YES <All the signals are received normally.> : Go to Step 3.
NO <The driver's door switch signal is not received.> : Refer to inspection procedure 7 "One of the door switch signals is not received ."
NO <The ignition switch (IG1) signal is not received.> : Refer to inspection procedure 2 "The ignition switch (IG1) signal is not received ."
NO <The tail lamp or headlamp switch signal is not received.> : Refer to inspection procedure 9 "The column switch signal is not received ."
 
STEP 3. Check by M.U.T.-III "Special Function"
Using M.U.T.-III, select "Test" from the special function of the combination meter. Execute the following item to check the buzzer.
  • Item 3: Buzzer (AUTO)
Is the check result normal?
Go to Step 4.
Replace the combination meter.
 
STEP 4. Retest the system.
Check that the lamp reminder buzzer works normally.
Is the check result normal?
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to use Troubleshooting/inspection Service Points - How to Cope with Intermittent Malfunction ).
Replace the ETACS-ECU (Refer to ).
