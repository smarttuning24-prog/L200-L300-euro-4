Inspection Procedure 20: You Cannot Enter the Off-road Mode When You Operate the Drive Mode-selector. <Vehicles with HDC Function>
 
 
caution
  • If there is any problem in the CAN bus lines, an incorrect diagnosis code may be set. Prior to this diagnosis, diagnose the CAN bus lines (Refer to GROUP 54C - CAN Bus Diagnostics Table ).
  • Whenever ECU is replaced, ensure that the CAN bus lines are normal.
  • When the hydraulic unit (integrated with ASTC-ECU) is replaced, always carry out the calibration of the steering wheel sensor and the G and yaw rate sensor (Refer to ).
 
 
COMMENTS ON TROUBLE SYMPTOM
When you set the drive mode-selector to "4HLc" or "4LLc", the ASTC-ECU will enter the off-road mode, and "GRAVEL" will appear on the multi-information display in the combination meter.
 
 
PROBABLE CAUSES
  • Damaged wiring harness or connector
  • Malfunction of the drive mode-selector
  • Malfunction of the hydraulic unit assembly (integrated with ASTC-ECU)
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III CAN bus diagnosis
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Use the M.U.T.-III to diagnose the CAN bus lines.
Is the check result normal?
Go to Step 3.
Repair the CAN bus lines (Refer to GROUP 54C - CAN Bus Diagnostics Table ). Then go to Step 2.
 
STEP 2. Retest the system
When you set the drive mode-selector to "4HLc" or "4LLc", "GRAVEL" should appear on the multi-information display in the combination meter.
Is the check result normal?
This diagnosis is complete.
Go to Step 3.
 
STEP 3. M.U.T.-III diagnosis code
Use M.U.T.-III to check the diagnosis code .
Is any diagnosis code set?
Perform the troubleshooting for the diagnosis code (Refer to Diagnosis Code Chart ). Then go to Step 6.
Go to Step 4.
 
STEP 4. Check for data list of other system
Use M.U.T.-III to check the data lists in the transfer-ECU (Refer to GROUP 22A - Troubleshooting, Data List Reference Table <Super Select 4WD> ).
  • Item No. 10: T/F position
OK:
  • 2H (Drive mode-selector position: 2H)
  • 4H (Drive mode-selector position: 4H)
  • 4HLc (Drive mode-selector position: 4HLc)
  • 4LLc (Drive mode-selector position: 4LLc)
Is the check result normal?
Go to Step 5.
Perform the transfer-ECU diagnosis code procedure (Refer to GROUP 22A - Troubleshooting, Code No. C1430: Transfer position switch <Super select 4WD> ). Then go to Step 6.
 
STEP 5. Retest the system
When you set the drive mode-selector to "4HLc" or "4LLc", "GRAVEL" should appear on the multi-information display in the combination meter.
Is the check result normal?
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to Cope with Intermittent Malfunction ).
Replace the hydraulic unit assembly (integrated with ASTC-ECU) (Refer to ). Then go to Step 6 .
 
STEP 6. Retest the system
When you set the drive mode-selector to "4HLc" or "4LLc", "GRAVEL" should appear on the multi-information display in the combination meter.
Is the check result normal?
This diagnosis is complete.
Return to Step 1.
