Code No.U1570 No bus activity error for LIN
 
 
 
 
TROUBLE JUDGEMENT
The ETACS-ECU will set diagnosis code No. U1570 when no communication has been carried out for more than a certain period after it received last valid data.
 
 
PROBABLE CAUSES
  • Malfunction of the ETACS-ECU
  • Malfunction of the slave ECU
  • LIN bus lines shorted to power supply
  • LIN bus lines shorted to earth
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check the following wiring harness.
  • Check of open circuit in LIN line between ETACS-ECU connector and column switch connector.
  • Check of open circuit in LIN line between ETACS-ECU connector and lighting control sensor connector. <Vehicles with auto lamp>
Are all the wiring harness wires in good condition?
Go to Step 2.
Repair the connector(s) or wiring harness.
 
STEP 2. Measure the voltage at the LIN bus lines.
Check the LIN bus lines for short to power supply.
(1) Disconnect all the ECU connectors which are connected to the LIN bus lines.
(2) Turn the ignition switch from "LOCK" (OFF) to "ON" position.
(3) Measure the voltage between column switch connector (wiring harness side: DATA terminal) and earth.
  • The voltmeter needle does not swing.
noteIf the LIN bus lines are short to power supply, the voltmeter indicates a constant value.
Is the check result normal?
Go to Step 3.
Repair the connector(s) or wiring harness.
 
STEP 3. Measure the resistance at the LIN bus lines.
Check the LIN bus lines for short to earth.
(1) Disconnect all the ECU connectors which are connected to the LIN bus lines.
(2) Measure the resistance value between column switch connector (wiring harness side: DATA terminal) and earth.
  • No continuity (1 kΩ or more)
Is the check result normal?
Go to Step 4.
Repair the connector(s) or wiring harness.
 
STEP 4. Check the column switch.
Disconnect column switch connector, and recheck whether diagnosis code No. U1570 is set.
(1) Connect all the relevant ECUs, and disconnect column switch connector only.
(2) Erase the diagnosis code.
(3) Turn the ignition switch from "LOCK" (OFF) to "ON" position.
(4) Check whether diagnosis code No. U1570 is set.
noteIf diagnosis code No. U1570 is not set, diagnosis code No. U1500 is set because the column switch is disconnected. This is normal.
(5) Turn the ignition switch to the "LOCK" (OFF) position.
Is diagnosis code No. U1570 set?
YES <Vehicles with auto lamp> : Reconnect the column switch connector, and go to Step 5.
YES <Vehicles without auto lamp> : Reconnect the column switch connector, and go to Step 6.
Replace the column switch (Refer to GROUP 54A - Column switch ).
 
STEP 5. Check the lighting control sensor.
Disconnect lighting control sensor connector, and recheck whether diagnosis code No. U1570 is set.
(1) Connect all the relevant ECUs, and disconnect lighting control sensor connector only.
(2) Erase the diagnosis code.
(3) Turn the ignition switch from "LOCK" (OFF) to "ON" position.
(4) Check whether diagnosis code No. U1570 is set.
noteIf diagnosis code No. U1570 is not set, diagnosis code No. U1504 is set because the lighting control sensor is disconnected. This is normal.
(5) Turn the ignition switch to the "LOCK" (OFF) position.
Is diagnosis code No. U1570 set?
Reconnect the lighting control sensor connector, and go to Step 6.
Replace the lighting control sensor (Refer to GROUP 51 - Lighting control sensor ).
 
STEP 6. Recheck for diagnosis code.
Check again if the diagnosis code is set in the ETACS-ECU.
(1) Erase the diagnosis code.
(2) Turn the ignition switch from "LOCK" (OFF) to "ON" position.
(3) Check whether diagnosis code No. U1570 is set.
(4) Turn the ignition switch to the "LOCK" (OFF) position.
Is the diagnosis code set?
Replace the ETACS-ECU (Refer to GROUP 54A - ETACS-ECU ).
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to use Troubleshooting/inspection Service Points - How to Cope with Intermittent Malfunction ).
