Inspection Procedure 5: No door will be locked or unlocked by operating a lock/unlock switch on any door.
 
 
cautionBefore replacing the ECU, ensure that the power supply circuit, the earth circuit and the communication circuit are normal.
 
 
 
 
PROBABLE CAUSES
  • Malfunction of the CAN bus lines
  • Malfunction of the central door locking system
  • Malfunction of the keyless operation key
  • Damaged wiring harness and connectors
  • Keyless operation key not registered
  • Malfunction of the KOS&OSS-ECU
  • Malfunction of the ETACS-ECU
  • Malfunction of the outside transmission antenna assembly
  • Function setting error or no setting with a customisation
  • An ambient electric device or noise source interferes with this system when the lock/unlock switch is operated.
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III CAN bus diagnostics
Use M.U.T.-III to diagnose the CAN bus lines.
Is the check result normal?
Go to Step 2.
Repair the CAN bus line (Refer to GROUP 54C - Troubleshooting ).
 
STEP 2. M.U.T.-III diagnosis code
Check if any diagnosis code is set in the KOS&OSS-ECU.
Is the diagnosis code set?
Refer to diagnosis code chart .
Go to Step 3.
 
STEP 3. M.U.T.-III other system diagnosis code
Check if any diagnosis code is set in the ETACS-ECU.
Is the diagnosis code set?
Perform troubleshooting for the ETACS (Refer to GROUP 54A - ETACS - Troubleshooting ).
Go to Step 4.
 
STEP 4. Check the central door locking system operation
Check that the central door locking system works normally.
Is the check result normal?
Go to Step 5.
Perform troubleshooting for the central door locking system (Refer to GROUP 42A - Troubleshooting ) or the dead lock system (Refer to GROUP 42A - Troubleshooting ).
 
STEP 5. Check the customisation
Check that KOS feature is set to Both enable or Door Entry enable with a customisation function.
Is it set to Both enable or Door Entry enable?
Go to Step 6.
Set KOS feature to Both enable or Door Entry enable with a customisation function (Refer to ).
 
STEP 6. Keyless operation system communication test
Using M.U.T.-III, perform the antenna communication test to check that the outside transmission antenna assembly communicates normally (Refer to ).
  • Outside transmission antenna assembly (LH)
  • Outside transmission antenna assembly (RH)
OK: Normal values are displayed for all the items
Is the check result normal?
Go to Step 8.
Go to Step 7.
 
STEP 7. Check with another registered keyless operation key.
Using another registered keyless operation key, check whether the doors can be locked and unlocked for twice or more consecutively.
Can they be operated?
They can be operated. : Go to Step 10.
They cannot be operated. : Go to Step 8.
There is no other registered keyless operation key. : Go to Step 10.
 
STEP 8. Connection check of commercial electric products
Check whether commercial electric products are connected.
Are commercial electric products connected?
Disconnect the commercial electric products as they may be interfering with communication. Then go to Step 9.
Go to Step 9.
 
STEP 9. Check of the troubles
Operate the lock/unlock switch to check that the door can be locked and unlocked.
Is the check result normal?
The diagnosis is complete.
Replace KOS&OSS-ECU and register the ID codes (Refer to ).
 
STEP 10. Keyless operation key battery charged level check
Check whether the LED illuminates when the lock/unlock switch on the keyless operation key that does not work is pressed.
Is the check result normal?
Go to Step 11.
Replace the battery in the keyless operation key (Refer to ).
 
STEP 11. Keyless operation key check
Check whether the registered keyless operation key is used.
Is the registered keyless operation key used?
Used. : Register the key ID and keyless operation key ID of the keyless operation key. Then go to Step 12.
Not used (The keyless operation key is not registered or the keyless operation key is also used for another vehicle). : Refer to the key ID registration procedure manual to register the key ID. If the key ID cannot be registered, replace the keyless operation key and register the key ID.
 
STEP 12. Check of the trouble
Operate the lock/unlock switch to check that the door can be locked and unlocked.
Is the check result normal?
The diagnosis is complete.
Replace the keyless operation key, and register the ID codes (Refer to ).
