INPUT SHAFT SPEED SENSOR CHECK
2020 / L200/L200 SPORTERO / EU - Workshop Manual
23 -AUTOMATIC TRANSMISSION > 23C-AUTOMATIC TRANSMISSION<R6AWH, V6AWH> > ON-VEHICLE SERVICE > AUTOMATIC TRANSMISSION CONTROL COMPONENT CHECK > INPUT SHAFT SPEED SENSOR CHECK
caution
  • Do not damage the input shaft speed sensor and its terminals.
1. Remove the bolt to remove the input shaft speed sensor from the transmission case.
2. Check that the continuity exists.
 
Measurement terminal No.Normal condition (check condition)
1 - 2Continuity exists (Less than 2 Ω)
noteWhen measuring the resistance, 100 kΩ or more may be detected. However, this is not due to a malfunction.
3. Connect the input shaft speed sensor connector and the battery as shown in the illustration.
4. Wave a steel material in front of the input shaft speed sensor tip (gap of 5 mm or closer) while measuring the current.
 
SignalNormal condition (check condition)
High12.0 - 16.0 mA
Low4.0 - 8.0 mA
5. Replace the input shaft speed sensor if there is no continuity or the measured current is not normal.
6. Apply A/T fluid to a new O-ring, and then install it onto the input shaft speed sensor.
7. Install the input shaft speed sensor to the transmission case with the bolt.
8. Tighten the input shaft speed sensor mounting bolt to the specified torque.
Tightening torque: 5.4 ± 1.5 N·m
