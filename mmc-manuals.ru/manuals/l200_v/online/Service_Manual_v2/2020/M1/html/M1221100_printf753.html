Warning Indicator <Vehicles with Mono-colour Liquid Crystal Display Type Combination Meter>
When a problem occurs in the transfer or rear differential lock system, the 2WD/4WD operation indicator lamp (a) or rear differential lock lamp (b) flashes rapidly (twice per second).
When the rear differential lock lamp (b) is flashing rapidly, check the diagnosis output.
 
 
 
Warning Indicator <Vehicles with Colour Liquid Crystal Display Type Combination Meter>
When a problem occurs in the transfer, an icon (A) will be displayed. When a problem occurs in the rear differential lock system, an icon (B) will be displayed.
If any of these icons is displayed, check whether the diagnosis codes are set.

Code No. C1403: Mode switch failure <Easy select 4WD>
 
 
 
 
OPERATION
  • When the transfer-ECU measures 0 V at the drive mode-selector position which the driver selects, it will determine that the appropriate position is turned on. When the ECU measures system voltage at the other selector positions, it will determine that the other positions are turned off.
  • System voltage is applied to the transfer-ECU when the drive mode-selector is in the "2H," "4H" or "4L." The transfer-ECU judges that the drive mode-selector is in the "2H," "4H" or "4L." when the system voltage is applied.
 
 
DIAGNOSIS CODE SET CONDITIONS
In the case that the input signal from the drive mode-selector is open or shorted, the diagnosis code No. C1403 is set as the open circuit or the short circuit of the drive mode-selector system.
 
 
PROBABLE CAUSES
  • Malfunction of the drive mode-selector
  • Damaged harness wires and connectors
  • Malfunction of the transfer-ECU
