Code No. C2330: Radar TEMP. (Out of guarantee)
 
 
cautionBefore replacing the ECU, ensure that the power supply circuit, the earth circuit and the communication circuit are normal.
 
 
OPERATION
The BSW-ECU (slave) will be disabled when its internal temperature is out of the range (-40°C to 85°C). It will be enabled again when the temperature is within the range.
 
 
CHECK CONDITION
When all of the conditions below are satisfied:
  • Ignition switch is in the "ON" position.
  • BSW-ECU (slave) system voltage is at 9 to 16 V.
 
 
DIAGNOSTIC FUNCTION AND JUDGEMENT CRITERIA
The BSW-ECU (slave) will set diagnosis code No. C2330 when all of the following conditions are satisfied:
  • Internal temperature in the BSW-ECU (slave) is less than -40°C.
  • Internal temperature in the BSW-ECU (slave) is 85°C or more.
 
 
PROBABLE CAUSES
  • Internal temperature in the BSW-ECU (slave) is high or too low.
  • Malfunction of BSW-ECU (slave)
 
 
DIAGNOSIS PROCEDURE
Check whether the diagnosis code is set again.
Check again if the diagnosis code is set in the BSW-ECU (slave).
(1) Erase the diagnosis code.
(2) Turn the ignition switch to "LOCK" (OFF) position and then wait for approximately 15 to 30 minutes until the BSW-ECU (slave) remains at room temperature.
(3) Turn the ignition switch from the "LOCK" (OFF) position to the "ON" position.
(4) Check if the diagnosis code is set.
Is the diagnosis code set?
Replace the BSW-ECU (slave) (Refer to ).
This diagnosis is complete. A component protection has been triggered. This is not a malfunction.
