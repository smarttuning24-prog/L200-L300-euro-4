Inspection Procedure 16: ACC power auto cut function does not work normally.
 
 
cautionBefore replacing the ECU, ensure that the power supply circuit, the earth circuit and the communication circuit are normal.
 
 
COMMENTS ON TROUBLE SYMPTOM
The KOS&OSS-ECU will evaluate the signals from the power supply mode (ACC) and the driver's door switch to determine whether the ACC power supply is shut down or not. If this function does not work normally, the input signal or the KOS&OSS-ECU may be defective. Also, "ACC automatic cut timer (OSS)" may be set to "Disable" through customisation.
 
 
PROBABLE CAUSES
  • Malfunction of KOS&OSS-ECU
  • Malfunction of the driver's door switch
  • External noise interruption in input signal
  • Malfunction of selector lever assembly (ON seizure of P range detection switch)
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check of customisation function
Use the ETACS-ECU customisation function to check whether "ACC automatic cut timer (OSS)" is set to "30 min" or "60 min."
Is the check result normal?
Go to Step 2.
Using the ETACS-ECU customisation function, set "ACC automatic cut timer (OSS)" to "30 min" or "60 min" (Refer to ).
 
STEP 2. M.U.T.-III diagnosis code
Check if any diagnosis code is set in the KOS&OSS-ECU.
Is the diagnosis code set?
Refer to diagnosis code chart .
Go to Step 3.
 
STEP 3. Check of short to earth in PDSW line between selector lever assembly connector and KOS&OSS-ECU connector
noteBecause the conformity cannot be determined even when the digital tester is used to measure the voltage of PDSW line between the selector lever assembly connector and KOS&OSS-ECU connector, be sure to use the oscilloscope (Outputs the sawtooth waveform of 0 to 5 V with intervals of 10 ms.).
Is the check result normal?
Go to Step 4.
Repair the connector(s) or wiring harness.
 
STEP 4. Check of selector lever assembly (P range detection switch)
Disconnect the selector lever assembly connector, and check the continuity at the equipment side.
