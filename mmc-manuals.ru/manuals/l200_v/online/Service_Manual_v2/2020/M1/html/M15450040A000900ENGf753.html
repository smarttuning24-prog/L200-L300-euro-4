Code No.B2206 Chassis No. mismatch
 
 
caution
  • If diagnosis code No.B2206 is set in the ETACS-ECU, diagnose the CAN main bus line.
  • Whenever the ECU is replaced, ensure that the input and output signal circuits are normal.
 
 
JUDGEMENT CRITERIA
If the chassis number registered to ETACS-ECU and the chassis number on CAN bus lines do not match, the ETACS-ECU sets diagnosis code No. B2206.
 
 
PROBABLE CAUSES
  • Malfunction of CAN bus line
  • Malfunction of engine-ECU
  • The ETACS-ECU, which was used on another vehicle, is installed on this vehicle.
  • The ETACS-ECU improperly coding or incorrect chassis number
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III CAN bus diagnostics
Use the M.U.T.-III to diagnose the CAN bus lines.
Is the check result normal?
Go to Step 2.
Repair the CAN bus line (Refer to GROUP 54C - Troubleshooting ).
 
STEP 2. M.U.T.-III other system diagnosis code
Check if a chassis number-related diagnosis code is set in the engine-ECU.
Is the diagnosis code set?
Troubleshoot the engine-ECU (Refer to GROUP 13A - Troubleshooting ).
Go to Step 3.
 
STEP 3. Check whether the diagnosis code is set again.
Replace the ETACS-ECU with a properly coded one. Then check if a diagnosis code is set again.
(1) Replace the ETACS-ECU with a properly coded one.
(2) Erase the diagnosis code.
(3) Turn the ignition switch from "LOCK" (OFF) position to "ON" position.
(4) Check if diagnosis code is set.
Is the diagnosis code set?
Return to Step 1.
The diagnosis is complete.
