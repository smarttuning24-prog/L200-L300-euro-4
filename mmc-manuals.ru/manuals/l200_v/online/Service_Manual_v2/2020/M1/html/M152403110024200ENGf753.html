cautionIf diagnosis codes is set in the SRS-ECU, always diagnose the CAN main bus line.
 
 
TROUBLE JUDGEMENT
These diagnosis codes are set when a fault is detected in the SRS-ECU. The most likely causes for this code to be set are shown in the table below:
 
 
PROBABLE CAUSE
Malfunction of the SRS-ECU
