DISASSEMBLY SERVICE POINT
 
 
<<A>> COUNTERSHAFT/INPUT SHAFT REMOVAL
Pull out the countershaft and the input shaft together from the clutch housing.
 
 
 
REASSEMBLY SERVICE POINTS
 
 
>>A<< OIL GUIDE INSTALLATION
Using the special tool lower arm bush remover and installer (MB990971) to install the oil guide.
 
 
>>B<< OIL SEAL INSTALLATION
1. Apply the specified lubricant to the oil seal lip.
Specified lubricant: MITSUBISHI MOTORS GENUINE Part No. 0101011 or equivalent
2. Use special tool install the oil seal.
  • Installer adaptor (MB990928)
  • Installer bar (MB990938)
 
 
>>C<< COUNTERSHAFT/INPUT SHAFT INSTALLATION
Insert the countershaft and the input shaft together into the clutch housing.
 
 
>>D<< BUSHING INSTALLATION
1. Use special tool installer bar (MB990938), install the bushing.
2. Press the gearshift 1ST and 2ND fork with a press and install the bushing.
 
 
>>E<< SPRING INSTALLATION
Install the spring with the small end directed to the ball.
 

 
 
>>F<< GEARSHIFT SHAFT LEVER, GEARSHIFT REVERSE FORK, GEARSHIFT INTERLOCK PLATE, GEARSHIFT 1ST AND 2ND FORK AND GEARSHIFT REVERSE RAIL/GEARSHIFT 1ST AND 2ND RAIL/GEARSHIFT 3RD AND 4TH RAIL/GEARSHIFT 5TH AND 6TH RAIL INSTALLATION
The gearshift lugs on the gearshift rails should be in one line.
 

 
 
>>G<< BUSHING INSTALLATION
Use special tools to install the bushing.
  • Installer adaptor (MB990926)
  • Installer bar (MB990938)
 
 
>>H<< GEARSHIFT SHAFT INSTALLATION
1. Insert the gearshift shaft into the M/T case front.
2. The finger control on the gearshift shaft should be positioned as shown.
 

 
 
>>I<< COUNTERSHAFT SPACER/INPUT SHAFT SPACER INSTALLATION
1. Place pieces of solder (approximately 10 mm long and 1.0 mm thick) on the bearing outer race of the input shaft.
2. Place pieces of solder (approximately 10 mm long and 1.0 mm thick) on the bearing outer race of the countershaft.
3. Install the M/T case front, and then tighten it to the specified torque.
4. Remove the M/T case front, and remove the pieces of solder.
5. If the pieces of solder are not flattened, repeat steps 1 to 3 by using a thicker solder (approximately 10 mm long and 1.6 mm thick).
6. Use a micrometer to measure the thickness of the flattened solder. Select an appropriate spacer so that the axial play meets the standard value.
Standard value: 0.05 to 0.17 mm
 

 
 
>>J<< M/T CASE FRONT INSTALLATION
1. Completely degrease the FIPG-applied surface so that water and oil including the old sealant cannot adhere to the surface coated with the sealant.
Never touch the degreased surface by hand.
2.
cautionSqueeze the sealant out evenly to apply it in an unbroken, non-excessive quantity.
Apply the sealant to the M/T case front.
Special sealant: MITSUBISHI MOTORS GENUINE Part No. MZ100388 or equivalent
3. Install the M/T case front.
4. The groove on the finger control of the gearshift shaft must be visible from the hole on the M/T case front. If no, adjust the gearshift shaft so that the groove is visible.
5. The hole on the gearshift shaft should be positioned as shown.
 

 
 
>>K<< BOLT INSTALLATION
1. Align the bolt mounting holes on the M/T case front with those on the reverse idler gear.
2. Tighten the bolts to the specified torque.
