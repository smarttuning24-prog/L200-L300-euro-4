Code No. B1209: Test mode <Colour liquid crystal display type>
 
 
 
 
TROUBLE JUDGEMENT
When the mode is changed to the meter test mode (supplier mode) with the ignition switch at the "ON" position, the combination meter sets diagnosis code No. B1209.
 
 
PROBABLE CAUSES
Malfunction of combination meter
