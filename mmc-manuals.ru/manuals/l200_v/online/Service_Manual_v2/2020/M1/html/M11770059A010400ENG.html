Code No. U1213: ASTC Parity/Toggle Bit Error <A/T>
 
 
caution
  • If diagnosis code U1213 is set from AS&G-ECU, surely perform CAN bus line diagnosis.
  • Replace ECU after certainly confirming that the communication circuits are normal.
 
 
TROUBLE JUDGMENT
Check Conditions
  • 2 seconds has passed after the ignition switch is turn to the "ON" position or the engine has started up.
  • The battery voltage is more than 10 V and is less than 16 V.
  • Code No. U0121: ASTC-ECU Time-out has not occurred.
Judgement Criterion
  • The CAN reception signal from the ASTC-ECU is abnormal.
 
 
COMMENTS ON TROUBLE SYMPTOM
Current malfunction
  • The failure is possibly caused by malfunction of the harness and connector in CAN bus line between AS&G-ECU and ASTC-ECU, by malfunction in the power supply system of ASTC-ECU, in ASTC-ECU itself, in AS&G-ECU.
Past malfunction
  • Proceed with troubleshooting focusing on malfunction of the harness and connector in CAN bus line between AS&G-ECU and ASTC-ECU and malfunction in the power supply system of ASTC-ECU. For diagnosis procedure, refer to how to cope with past malfunctions (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Treat Past Trouble ).
    noteIn the case of past malfunctions, M.U.T.-III CAN bus line diagnosis cannot detect a malfunction even if there is a malfunction in CAN bus. Therefore, check CAN bus line as in the case of response to intermittent malfunctions (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ). It is possible to narrow down areas with a high possibility of malfunctions from diagnosis codes set in each ECU that performs CAN communication (Refer to GROUP 54C - Explanation about the M.U.T.-III CAN Bus Diagnostics ).
 
 
FAIL-SAFE AND BACKUP FUNCTION
In the normal operation, the auto stop is prohibited. If the malfunction is detected during the auto stop, the auto start is prohibited.
 
 
PROBABLE CAUSES
  • Failed harness and connector.
  • Failed hydraulic unit assembly (integrated with ASTC-ECU).
  • Failed AS&G-ECU.
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III CAN bus diagnosis
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Use M.U.T.-III to perform CAN bus line diagnosis.
Is the check result normal?
Go to Step 2.
Repair CAN bus line (Refer to GROUP 54C - Troubleshooting <CAN>, CAN Bus Diagnostics Table ). Then go to Step 5.
 
STEP 2. Check for diagnosis code of other system
Use M.U.T.-III to check if the diagnosis code is set in the ASTC-ECU.
Is any diagnosis code set in the ASTC-ECU?
Perform troubleshooting of the ASTC-ECU (Refer to GROUP 35C - Troubleshooting, Diagnosis Code Chart ).
Go to Step 3.
 
STEP 3. Diagnosis code recheck after erasing diagnosis code
Check again if the diagnosis code is set.
(1) Use M.U.T.-III to erase the diagnosis code.
(2) Turn the ignition switch to "LOCK" (OFF) position.
(3) Turn the ignition switch to "ON" position and start the engine.
(4) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. U1213 set?
Replace the hydraulic unit assembly (integrated with ASTC-ECU). Then go to Step 4.
Intermittent malfunction in CAN bus line between AS&G-ECU and ASTC-ECU (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
 
STEP 4. Diagnosis code recheck after erasing diagnosis code
Check again if the diagnosis code is set.
(1) Use M.U.T.-III to erase the diagnosis code.
(2) Turn the ignition switch to "LOCK" (OFF) position.
(3) Turn the ignition switch to "ON" position and start the engine.
(4) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. U1213 set?
Replace the AS&G-ECU. Then go to Step 5.
Intermittent malfunction in CAN bus line between AS&G-ECU and ASTC-ECU (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
 
STEP 5. Diagnosis code recheck after erasing diagnosis code
Check again if the diagnosis code is set.
(1) Use M.U.T.-III to erase the diagnosis code.
(2) Turn the ignition switch to "LOCK" (OFF) position.
(3) Turn the ignition switch to "ON" position and start the engine.
(4) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. U1213 set?
Go to Step 1.
Check end.
