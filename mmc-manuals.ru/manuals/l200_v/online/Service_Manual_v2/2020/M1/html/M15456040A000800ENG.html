Code No. B1861: e-Call Battery
 
 
cautionWhen replacing the ECU, always check that the communication circuit is normal.
 
 
DIAGNOSTIC FUNCTION
When the system determines a voltage drop in the emergency call battery, the emergency call control unit sets diagnosis code No. B1861.
noteWhen diagnosis code No. B1861 is set, the emergency call switch flashes in green.
 
 
PROBABLE CAUSES
  • Malfunction of emergency call battery
  • Malfunction of emergency call control unit
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check the M.U.T.-III Diagnosis code for other.
Check if diagnosis code No. B1860 is set in the emergency call control unit.
Is the diagnosis code set?
Trouble shooting diagnosis code No. B1860 (Refer to ).
Go to Step 2.
 
STEP 2. Replace the emergency call battery, and check whether the diagnosis code is set again.
Check again if the diagnosis code is set in the emergency call control unit.
(1) Erase the diagnosis code.
(2) Turn the ignition switch from "LOCK" (OFF) position to "ON" position.
(3) Check if diagnosis code is set.
Is the diagnosis code set?
Replace the emergency call control unit (Refer to ).
The procedure is complete.
