DIAGNOSIS FUNCTION
FCM/LDW/AHB-ECU has the following functions for easier system checks.
  • Diagnosis code set (Refer to ).
  • Data list output (Refer to ).
  • Freeze frame data output (Refer to ).
All the above items can be diagnosed using M.U.T.-III.
 
 
CHECK OF FREEZE FRAME DATA
The freeze frame data can be checked by using the M.U.T.-III.
 
 
When detecting fault and storing the diagnosis code, the ECU connected to CAN bus line obtains the data before the determination of the diagnosis code and the data when the diagnosis code is determined, and then stores the ECU status of that time. By analysing each data from M.U.T.-III, the troubleshooting can be performed more efficiently. The displayed items are as the table below.
