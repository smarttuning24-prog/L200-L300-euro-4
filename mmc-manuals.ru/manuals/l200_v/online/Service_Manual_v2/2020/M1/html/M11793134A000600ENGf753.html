Code No. P20EE: SCR Efficiency Below Threshold
 
 
 
 
FUNCTION
  • This monitor determines that the purification efficiency of the SCR has decreased when the actual integrated value of the NOx purification calculated from the exhaust muffler NOx sensor (No. 1, No. 2) input becomes lower than the integrated value estimated from the catalyst temperature, the exhaust air temperature and the AdBlue™ injection quantity.
  • The DCU sets diagnosis code No. P20EE when it determines that the purification efficiency of the SCR has decreased.
 
 
TROUBLE JUDGEMENT
Check Conditions
  • After the exhaust muffler NOx sensor (No. 1, No. 2) heater is activated.
  • During urea injection assembly.
  • During on-line dosing.
  • Not during DPF regeneration.
  • Barometric pressure is more than 750 hPa.
  • Intake air temperature is more than -12°C.
  • The ammonia is adsorbed onto the SCR sufficiently.
  • The SCR efficiency is 0.7 or more.
Judgement Criterion
  • The NOx purification efficiency is determined to be decreased.
 
 
PROBABLE CAUSES
  • Dilution or deterioration of the AdBlue™
  • Failed diesel fuel system
  • Failed No. 3 exhaust gas temperature sensor (DPF temperature)
  • Failed urea injector assembly (injection quantity is small)
  • Failed urea pump (integrated with supply pump)
  • Failed exhaust center pipe (integrated with SCR)
  • Failed No. 1 exhaust muffler NOx sensor
  • Failed No. 2 exhaust muffler NOx sensor
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check for diagnosis code of other system
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Use M.U.T.-III to check if the diagnosis code is set in the engine-ECU.
Is any diagnosis code set in the engine-ECU?
Perform the troubleshooting of diesel fuel system (Refer to GROUP 13A - Troubleshooting, Inspection Chart For Diagnosis Code ).
Go to Step 2.
 
STEP 2. Check for data list of other system
Use M.U.T.-III to check the engine-ECU data list (Refer to GROUP 13A - Troubleshooting, Data List Reference Table ).
  • Item No. 119: No. 3 exhaust gas temperature sensor (DPF temperature)
OK:
  • Engine cold state (engine stopped): At ambient temperature (atmospheric temperature) or equivalent
  • Engine hot state: The temperature changes as the engine operates (The indicated temperature rises when the engine operation is changed from "low revolution/low load" to "high revolution/high load").
Is the check result normal?
Go to Step 3.
Carry out the troubleshooting for diagnosis code No. P1427 and P1428 of the engine-ECU (Refer to GROUP 13A - Troubleshooting, Diagnosis Code Procedures No. P1427 and Diagnosis Code Procedures No. P1428 ).
 
STEP 3. M.U.T.-III diagnosis code
Use M.U.T.-III to check whether any diagnosis code related to the exhaust muffler NOx sensor (No. 1, No. 2), urea injector assembly and urea pump (supply pump) is set.
Is any diagnosis code set?
Carry out the relevant troubleshooting for the set diagnosis code (Refer to ).
Go to Step 4.
 
STEP 4. Urea pressure check
Use the M.U.T.-III to check the urea pressure (Refer to ).
Is the check result normal?
Go to Step 5.
Perform the troubleshooting Step 1 to 4 of diagnosis code No. P20E8: Urea pressure too low (Refer to ).
 
STEP 5. Urea injection assembly condition check
noteIf diagnosis code No. P249D or P249E is set, carry out the relevant troubleshooting for the set diagnosis code (Refer to or ).
Use the M.U.T.-III to check the spray state of the urea injector assembly (Refer to ).
OK: The AdBlue™ is injected from each injection port of the urea injector assembly equally.
Is the check result normal?
Go to Step 6.
Replace the urea injector assembly (Refer to ).
 
STEP 6. Check the SCR
(1) Remove the exhaust center pipe (integrated with SCR) (Refer to ).
(2) Check if there is a damage or urea crystal (white) adhered inside the catalytic converter (with mixer) and the SCR.
Is the check result normal?
Go to Step 7.
Repair or replace the catalytic converter (with mixer) and/or the exhaust center pipe (integrated with SCR). Then change all the AdBlue™ (Refer to ).
 
STEP 7. Check the No. 1 exhaust muffler NOx sensor and No. 2 exhaust muffler NOx sensor
(1) Warm up the engine until data list item No. 21: NOx sensor (UP) heater release status and item No. 22: NOx sensor (DOWN) heater release status are "ON".
(2) Display the M.U.T.-III data list below.
  • Item No. 49: Modified NOx value (UP) <No. 1 exhaust muffler NOx sensor>
  • Item No. 50: Modified NOx value (DOWN) <No. 2 exhaust muffler NOx sensor>
  • Item No. 76: Estimated ammonia load
(3) Carry out the ammonia purge to consume ammonia adsorbed onto the SCR (Refer to ).
OK:
  • Check that the amount detected at the downstream side of the SCR becomes closer to the value of the upstream side of the SCR as the amount of adsorbed ammonia decreases.
  • When the amount of adsorbed ammonia is 0 g, operate the accelerator pedal to check that the amount detected at the downstream side of the SCR changes following the amount detected at the upstream side of the SCR.
Is the check result normal?
Change all the AdBlue™ (Refer to ). Then go to Step 8.
Replace the exhaust muffler NOx sensor (No. 1 and/or No. 2) the value of which does not change even when the accelerator pedal is operated (Refer to ).
 
STEP 8. Diagnosis code recheck after erasing diagnosis code
Check again if the diagnosis code is set.
(1) Use M.U.T.-III to erase the diagnosis code.
(2) Turn the ignition switch to "LOCK" (OFF) position.
(3) Warm up the engine until data list item No. 21: NOx sensor (UP) heater release status and item No. 22: NOx sensor (DOWN) heater release status are "ON".
(4) Perform constant speed driving at approximately 60 km/h for 25 minutes or more in total.
(5) Use M.U.T.-III to recheck the diagnosis code.
noteDo not turn off the ignition switch during reconfirmation. If the engine is stopped by turning off the ignition switch, repeat the procedure from the beginning.
Is diagnosis code No. P20EE set?
Replace the exhaust center pipe (integrated with SCR) (Refer to ).
Check is end
