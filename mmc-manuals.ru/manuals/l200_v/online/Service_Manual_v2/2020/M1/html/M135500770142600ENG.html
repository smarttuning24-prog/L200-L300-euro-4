Inspection Procedure 11: Abnormality in ASTC System Operation.
 
 
caution
  • Whenever ECU is replaced, ensure that the CAN bus lines are normal.
  • When the hydraulic unit (integrated with ASTC-ECU) is replaced, always carry out the calibration of the steering wheel sensor and the G and yaw rate sensor (Refer to ).
 
 
COMMENTS ON TROUBLE SYMPTOM
In case of this trouble symptom, the ASTC system operation may be disabled. Diagnosis code may be set by the ASTC system using M.U.T.-III.
 
 
PROBABLE CAUSES
  • Low battery voltage
  • The CAN bus line is defective.
  • Engine coding data problem
  • Malfunction of the hydraulic unit assembly (integrated with ASTC-ECU)
  • Different ETACS-ECU, abnormal variant coding information
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III diagnosis code.
Use M.U.T.-III to check the diagnosis code for the ASTC system.
Is the diagnosis code set?
Carry out the diagnosis for the diagnosis code (Refer to ).
Go to Step 2.
 
STEP 2. Engine-ECU coding data check.
Using M.U.T.-III, check if any abnormality is present to the coding data below which are stored in the engine-ECU.
A.S.C.
  • OK: Available
Is the check result normal?
Go to Step 3.
Replace the engine-ECU (Refer to GROUP 13A - Engine-ECU <4N14> <4N14> or Engine-ECU <4N15> <4N15>). Then go to Step 4.
 
STEP 3. Hydraulic unit assembly (integrated with ASTC-ECU) check.
Check that the brake tube is installed to the hydraulic unit assembly (integrated with ASTC-ECU) correctly (Refer to ).
Is the check result normal?
Replace the hydraulic unit assembly (integrated with ASTC-ECU) (Refer to ). Then go to Step 4.
Connect the brake tubes correctly, and repair or replace the external brake lines of the hydraulic unit assembly (integrated with ASTC-ECU).
 
STEP 4. Retest the system.
Does ASTC system operation normally?
This diagnosis is complete.
Check the brake system related components other than the ASTC system.
