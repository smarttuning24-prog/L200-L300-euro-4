Inspection Procedure 15: Washer fluid level warning lamp does not illuminate.
 
 
caution
  • If there is any problem in the CAN bus lines, an incorrect diagnosis code may be set. Prior to this diagnosis, diagnose the CAN bus lines (Refer to GROUP 54C - CAN Bus Diagnostics Table ).
  • Whenever ECU is replaced, ensure that the CAN bus lines are normal.
  • Before replacing the combination meter, be sure to check that the power supply circuit, earth circuit, and communication circuit are normal.
 
 
 
 
COMMENTS ON TROUBLE SYMPTOM
  • ETACS-ECU sends the illumination request signal of the washer fluid level warning lamp to the combination meter via the CAN communication.
  • Combination meter illuminates washer fluid level warning lamp for approximately 3 seconds for valve check with the ignition switch turned to the "ON" position.
  • If the washer fluid level warning lamp do not illuminate, the wiring harness and connector(s) in the CAN bus lines, combination meter, ETACS-ECU or the windshield washer tank fluid level sensor may have a problem.
 
 
PROBABLE CAUSES
  • The CAN bus line is defective.
  • Malfunction of windshield washer tank fluid level sensor input signal
  • Damaged harness wires and connectors
  • Malfunction of combination meter
  • The ETACS-ECU coding data is abnormal.
  • Malfunction of ETACS-ECU
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III CAN bus diagnostics.
Use the M.U.T.-III to diagnose the CAN bus lines.
Is the check result normal?
Go to Step 2.
Repair the CAN bus line (Refer to GROUP 54C - Troubleshooting ). Then go to Step 6.
 
STEP 2. Check the washer fluid level warning lamp.
Check if the washer fluid level warning lamp illuminate normally when the ignition switch is turned to the "ON" position.
Is the check result normal?
Go to Step 3.
Check the combination meter (Refer to ). Then go to Step 6.
 
STEP 3. Use M.U.T.-III to check the data list of other system.
Check the ETACS system data list (Refer to ).
  • Item No. 112: Washer fluid switch
Is the check result normal?
Go to Step 4.
Check the windshield washer tank fluid level sensor input signal (Refer to ). Then go to Step 6.
 
STEP 4. M.U.T.-III other system diagnosis code.
Check again if the diagnosis code is set in the ETACS-ECU (Refer to ).
Is the diagnosis code set?
Carry out the diagnosis for the diagnosis code (Refer to ). Then go to Step 6.
Go to Step 5.
 
STEP 5. Retest the system.
Check the washer fluid level warning lamp is illuminated normally.
Is the check result normal?
The procedure is complete. (If no malfunctions are found in all steps, an intermittent malfunction is suspected. Refer to GROUP 00 / How to Cope with Intermittent Malfunction ).
Replace the ETACS-ECU (Refer to ). Then go to Step 6.
 
STEP 6. Retest the system.
Check the washer fluid level warning lamp is illuminated normally.
Is the check result normal?
This diagnosis is complete.
Return to Step 1.
