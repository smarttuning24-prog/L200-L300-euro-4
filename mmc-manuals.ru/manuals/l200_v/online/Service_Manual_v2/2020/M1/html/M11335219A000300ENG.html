Code No. P0513: Immobilizer Malfunction
 
 
 
 
FUNCTION
  • The engine-ECU transmits to and receives from KOS&OSS-ECU <Vehicles with KOS>, ETACS-ECU <Vehicles without KOS> a control signal to authenticate the ignition key.
    note
    • This code may be displayed due to radio interference if there are multiple registered ignition keys nearby during engine start.
    • This code may be displayed when the transponder ID code is registered.
    • This code is also displayed when the ignition key code is not recognised.
    • When this code is displayed, it is also sent to KOS&OSS-ECU <Vehicles with KOS> or ETACS-ECU <Vehicles without KOS>.
 
 
TROUBLE JUDGEMENT
Check Condition
  • Ignition switch is in "ON" position.
Judgement Criterion
  • The check result of the KOS&OSS-ECU is not suitable for that of the engine-ECU <Vehicles with KOS>.
  • The check result of the ETACS-ECU is not suitable for that of the engine-ECU <Vehicles without KOS>.
 
 
FAIL-SAFE AND BACKUP FUNCTION
The AS&G control is prohibited.
 
 
PROBABLE CAUSES
  • Failed KOS&OSS-ECU <Vehicles with KOS>
  • Failed ETACS-ECU <Vehicles without KOS>
  • Failed engine-ECU
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check for diagnosis code of other system
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Use M.U.T.-III to check if the diagnosis code is set in the KOS&OSS-ECU <Vehicles with KOS> or ETACS-ECU <Vehicles without KOS>.
Is any diagnosis code set in the KOS&OSS-ECU <Vehicles with KOS> or ETACS-ECU <Vehicles without KOS>?
Perform the troubleshooting of KOS-system (Refer to GROUP 42B - Troubleshooting, Diagnosis Code Chart <Vehicles with KOS>) or the troubleshooting of KOS-system (Refer to GROUP 54A - Immobilizer System, Troubleshooting, Diagnosis Code Chart <Vehicles without KOS>).
Go to Step 2.
 
STEP 2. Diagnosis code recheck after erasing diagnosis code
Check again if the diagnosis code is set.
(1) Use M.U.T.-III to erase the diagnosis code.
(2) Turn the ignition switch to "LOCK" (OFF) position.
(3) Turn the ignition switch to "ON" position.
(4) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. P0513 set?
Replace the KOS&OSS-ECU. After the replacement, put the each identification code into KOS&OSS-ECU (Refer to GROUP 42B - On-vehicle Service, ID Codes Registration Procedures <Vehicles with KOS>). Or replace the ETACS-ECU. After the replacement, put the each identification code into ETACS-ECU (Refer to GROUP 54A - Immobilizer System, On-vehicle Service, ID Codes Registration Procedures <Vehicles without KOS>). Then go to Step 3.
Intermittent malfunction (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
 
STEP 3. Diagnosis code recheck after erasing diagnosis code
Check again if the diagnosis code is set.
(1) Use M.U.T.-III to erase the diagnosis code.
(2) Turn the ignition switch to "LOCK" (OFF) position.
(3) Turn the ignition switch to "ON" position and.
(4) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. P0513 set?
Replace the engine-ECU.
The check is end.
