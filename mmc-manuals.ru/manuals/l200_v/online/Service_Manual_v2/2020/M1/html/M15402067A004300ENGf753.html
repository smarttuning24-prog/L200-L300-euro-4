Inspection Procedure 7: Buzzers do not sound normally.
 
 
cautionBefore replacing the combination meter, be sure to check that the power supply circuit, earth circuit, and communication circuit are normal.
 
 
 
 
COMMENTS ON TROUBLE SYMPTOM
When the following signals are received via the CAN communication or signal line, the combination meter sounds the incorporated buzzer according to the each pattern.
Adjustable speed limiter function overspeed buzzer <Vehicles with cruse control system>
  • Ignition switch ON signal
  • Vehicles speed signal
Electric steering wheel lock malfunction warning buzzer <Vehicles with Keyless Operation System (KOS)>
  • Sounding request signal from the KOS&OSS-ECU (The KOS&OSS-ECU uses the following input signals to send a sounding request signal)
    • Ignition switch signal
    • LIN bus line (local) for electrical steering lock
    • Actuator driving permission monitoring signal
OSS-related malfunction warning buzzer <Vehicles with KOS>
  • Sounding request signal from the KOS&OSS-ECU (The KOS&OSS-ECU uses the following input signals to send a sounding request signal)
    1. Ignition switch signal
    2. When the following conditions are met, and 1 second or more has elapsed (a: +B1 is detected to be ON and +B2 is detected to be OFF. b: When +B1 is OFF and +B2 is ON, KOS&OSS-ECU detects that the IOD fuse is mounted.).
Engine switch reminder buzzer <Vehicles with KOS>
  • Sounding request signal from the KOS&OSS-ECU (The KOS&OSS-ECU uses the following input signals to send a sounding request signal)
    • Engine switch ON signal
    • Driver's door switch ON signal
Keyless operation key reminder buzzer <Vehicles with KOS>
  • Sounding request signal from the KOS&OSS-ECU (The KOS&OSS-ECU uses the following input signals to send a sounding request signal)
    • Ignition switch OFF signal
    • Immobilizer antenna output signal
    • Driver's door switch ON signal
Key reminder buzzer <Vehicles without KOS>
  • Driver's door switch ON signal
  • Sounding request signal from the ETACS-ECU (The ETAC-ECU uses the following input signals to send a sounding request signal)
    • Ignition switch OFF signal
    • Key reminder switch ON signal
Lamp reminder buzzer
  • Ignition switch OFF signal
  • Sounding request signal from the ETACS-ECU (The ETAC-ECU uses the following input signals to send a sounding request signal)
    • Lighting switch ON signal
    • Driver's door switch ON signal
Door-ajar warning buzzer
  • Ignition switch ON signal
  • Any door switch ON signal
  • Vehicles speed signal
Freeze warning buzzer
  • Ignition switch ON signal
  • Ambient temperature signal
Parking brake reminder buzzer
  • Ignition switch ON signal
  • Parking brake switch
  • Vehicles speed signal
  • Engine speed
Turn-signal lamp buzzer
  • Turn-signal lamp switch ON signal
A/T shift cancel buzzer/Paddle shift cancel prohibition buzzer <A/T>
  • Sounding request signal from the A/T-ECU
ETACS-ECU function customise buzzer
  • Sounding request signal from the ETACS-ECU
Seat belt reminder function buzzer
  • Ignition switch ON signal
  • Driver’s seat belt switch signal
  • Passenger’s seat belt switch signal
  • Rear seat belt switch signal
  • Vehicle speed signal
AS&G buzzer <Vehicles with AS&G system>
  • Sounding request signal from the AS&G-ECU
Rest reminder buzzer
  • When the time to take a rest which was set on the multi-information display is reached.
Active Stability & Traction Control (ASTC) pre high temperature warning buzzer/ASTC high temperature warning buzzer
  • Sounding request signal from the ASTC-ECU
Multi-information display switch operation buzzer <Mono-colour liquid crystal display type>
  • ON signal for multi-information display switch
Rheostat illumination switch/multi-information display switch/multi-information display back switch operation buzzer <Colour liquid crystal display type>
  • ON signal for multi-information display switch
Multi-information display interrupt display buzzer <Colour liquid crystal display type>
  • Display condition signal of information display from each warning (When there is a fixed buzzer sounding pattern for each warning, that pattern has the priority.)
Rear differential lock system <4WD>/Forward Collision Mitigation (FCM) <Vehicles with FCM>/Ultrasonic misacceleration Mitigation System (UMS) <Vehicles with UMS>/Hill Descent Control (HDC) <Vehicles with HDC>/Off road mode <Vehicles with HDC> cancel buzzer
  • Sounding request signal from the transfer-ECU, FCM/LDW/AHB-ECU, ASTC-ECU, clearance warning control unit
UMS warning buzzer <Vehicles with UMS>
  • Sounding request signal from the clearance warning control unit
FCM collision warning buzzer/FCM approaching warning buzzer <Vehicles with FCM>
  • Sounding request signal from the FCM/LDW/AHB-ECU
Lane Departure Warning System (LDW) warning buzzer <Vehicles with LDW>
  • Sounding request signal from the FCM/LDW/AHB-ECU
Blind spot warning buzzer <Vehicles with Blind Spot Warning [BSW] (with Lane Change Assist)>
  • Sounding request signal from the BSW-ECU
Rear cross traffic alert buzzer <Vehicles with Blind Spot Warning [BSW] (with Lane Change Assist)>
  • Sounding request signal from the BSW-ECU
If the buzzer does not sound normally, the connector(s) and wiring harness in the CAN bus lines, or the ETACS-ECU or the combination meter may have a problem.
 
 
PROBABLE CAUSES
  • Malfunction of combination meter
  • Damaged harness wires and connectors
  • Malfunction of ECUs
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III CAN bus diagnostics
Use the M.U.T.-III to diagnose the CAN bus lines.
Is the check result normal?
Go to Step 2.
Repair the CAN bus line (Refer to GROUP 54C - Troubleshooting ).
 
STEP 2. M.U.T.-III diagnosis code
Check if diagnosis code is set in the combination meter.
Is the diagnosis code set?
Refer to .
Go to Step 3.
 
STEP 3. M.U.T.-III other system diagnosis code
Check if diagnosis code is set in the engine-ECU, A/T-ECU, ASTC-ECU, KOS&OSS-ECU, ETACS-ECU, A/C-ECU, transfer-ECU, AS&G-ECU, clearance warning control unit, FCM/LDW/AHB-ECU and BSW-ECU.
Is the diagnosis code set in the any of the above?
Set in the engine-ECU. : Troubleshoot the engine (Refer to GROUP 13A - Diagnosis Code Chart ).
Set in the A/T-ECU. : Troubleshoot the A/T (Refer to GROUP 23C - Diagnosis code chart ).
Set in the ASTC-ECU. : Troubleshoot the ASTC (Refer to GROUP 35C - Troubleshooting ).
Set in the KOS&OSS-ECU. : Troubleshoot the KOS (Refer to GROUP 42B - Troubleshooting ).
Set in the ETACS-ECU. : Troubleshoot the ETACS (Refer to ).
Set in the A/C-ECU. : Troubleshoot the A/C (Refer to GROUP 55B - Troubleshooting ).
Set in the transfer-ECU. : Troubleshoot the 4WD system (Refer to GROUP 22A - Diagnosis code chart ).
Set in the AS&G-ECU. : Troubleshoot the AS&G system (Refer to GROUP 17 - Diagnosis Code Chart <M/T> or <A/T>).
Set in the clearance warning control unit. : Troubleshoot the Clearance warning control unit (Refer to ).
Set in the FCM/LDW/AHB-ECU. : Troubleshoot the FCM and LDW (Refer to ).
Set in the BSW-ECU. : Troubleshoot the Blind Spot Warning [BSW] (with Lane Change Assist) (Refer to ).
The diagnosis code is not set. : Go to Step 4.
 
STEP 4. Check by M.U.T.-III "Special Function"
Using M.U.T.-III, select "Test" from the special function of the combination meter. Execute the following item to check the buzzer.
  • Item 3: Buzzer (AUTO)
Is the check result normal?
Go to Step 5.
Replace the combination meter (Refer to ).
 
STEP 5. Retest the system.
Check that the buzzers sound normally.
Is the check result normal?
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to Cope with Intermittent Malfunction ).
Replace the combination meter (Refer to ).
