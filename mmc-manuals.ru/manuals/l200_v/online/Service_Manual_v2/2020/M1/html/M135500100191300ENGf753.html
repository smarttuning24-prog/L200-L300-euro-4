DIAGNOSTIC FUNCTION
ASTC-ECU has the following functions for easier system checks.
  • Diagnosis code set (Refer to ).
  • Data list output (Refer to ).
  • Actuator test (Refer to ).
  • Freeze frame data output (Refer to ).
All the above items can be diagnosed using M.U.T.-III.
 
 
Check of Freeze Frame Data
The freeze frame data can be checked by using the M.U.T.-III.
When detecting fault and setting the diagnosis code, the ECU connected to CAN bus line obtains the data before the determination of the diagnosis code and the data when the diagnosis code is determined, and then stores the ECU status of that time. By analysing each data from M.U.T.-III, the troubleshooting can be performed more efficiently. The displayed items are as the table below.

Fail-safe Functions
When a malfunction is determined by the diagnostic function, ASTC-ECU illuminates the brake warning lamp, ABS warning lamp, ASTC indicator lamp and ASTC OFF indicator lamp. At the same time, ASTC-ECU prohibits the ABS, EBD, BA, ATC, ASC, HSA, TSA, Emergency Stop Signal system (ESS), FCM, HDC and AS&G BH.
noteThe brake warning lamp is used as the EBD control warning lamp.
