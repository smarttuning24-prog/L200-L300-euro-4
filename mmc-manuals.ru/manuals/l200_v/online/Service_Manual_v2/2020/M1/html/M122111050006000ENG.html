Code No. C1430: Transfer position SW <Super select 4WD>
 
 
 
 
OPERATION
System voltage is applied to the each transfer actuator from the transfer-ECU.
 
 
DIAGNOSIS CODE SET CONDITIONS
  • When the transfer changeover is not completed while driving, diagnosis code No. C1430 is set for abnormality of each transfer actuator.
  • When the transfer changeover is not completed while the vehicle is stopped, diagnosis code No. C1430 is set for abnormality of each transfer actuator.
  • When the abnormality of each transfer actuator is detected all the time, diagnosis code No. C1430 is set.
 
 
PROBABLE CAUSES
  • Damaged harness wires and connectors
  • Malfunction of the transfer actuator
  • Malfunction of the transfer-ECU

STEP 31. Transfer actuator check.
(1) Disconnect the transfer actuator connector, and measure at the transfer actuator side.

ON: Continuity exists (2 Ω or less)
OFF: No continuity
(3) Measure the resistance between the transfer actuator connector (transfer limit switch 2 terminal) and transfer actuator connector (earth terminal).
ON: Continuity exists (2 Ω or less)
OFF: No continuity

ON: Continuity exists (2 Ω or less)
OFF: No continuity
