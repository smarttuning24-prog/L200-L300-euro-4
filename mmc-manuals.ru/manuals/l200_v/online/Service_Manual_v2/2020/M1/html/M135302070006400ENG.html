Code No. C1369: G and Yaw Rate Sensor (Bank Run Record)
 
 
caution
  • If there is any problem in the CAN bus lines, an incorrect diagnosis code may be set. Prior to this diagnosis, diagnose the CAN bus lines (Refer to GROUP 54C - CAN Bus Diagnostics Table ).
  • Whenever ECU is replaced, ensure that the CAN bus lines are normal.
 
 
OPERATION
The G and yaw rate sensor detects longitudinal acceleration, lateral acceleration, and yaw rate for the vehicle, and sends them to ASTC-ECU via the CAN bus line.
 
 
DIAGNOSIS CODE SET CONDITIONS
This diagnosis code is set when the vehicle is driven under special conditions (laterally inclined road, etc).
noteThe ASTC indicator lamp turns OFF when the ignition switch is in the "lock" (OFF) position, or by the normal driving at the vehicle speed of 10 km/h or more.
 
 
PROBABLE CAUSES
  • Malfunction of the hydraulic unit assembly (integrated with ASTC-ECU)
