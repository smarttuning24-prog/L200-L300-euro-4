DISASSEMBLY
 
 
1. LOW/REVERSE BRAKE PISTON REMOVAL
1.
cautionBe careful not to shorten the return spring too much.
Set the following special tools on the low/reverse brake return spring sub-assembly, tighten the special tools and compress the low/reverse brake return spring sub-assembly.
  • Spring compressor body (MD998903)
  • Spring compressor (MD998924)
2.
cautionBe careful not to expand the snap ring too much.
Using snap ring pliers, remove the snap ring.
3. Remove the low/reverse brake return spring sub-assembly from the transmission case.
4.
cautionBe careful not to damage the low/reverse brake piston.
While pushing the low/reverse brake piston by hand, apply compressed air (300 kPa) into the oil passage of the transmission case as shown in the illustration and remove the low/reverse brake piston.
5. Remove the O-ring from the low/reverse brake piston.
noteIn some cases, the low/reverse brake piston may be removed with the brake reaction sleeve and the inner brake piston No. 4 attached.
 

 
 
2. BRAKE REACTION SLEEVE REMOVAL
1. Remove the brake reaction sleeve from the transmission case.
2. Remove the 2 O-rings from the brake reaction sleeve.
 

 
 
3. INNER BRAKE PISTON NO. 4 REMOVAL
1. Remove the inner brake piston No. 4 from the transmission case.
2. Remove the 2 O-rings from the inner brake piston No. 4.
 

 
 
4. ONE-WAY CLUTCH INNER RACE REMOVAL
Remove the one-way clutch inner race from the one-way clutch assembly No. 3.
 
 
 
 
REASSEMBLY
 
 
INSPECTION BEFORE REASSEMBLY
Refer to Low-reverse Brake Inspection .
  • Check One-way Clutch Assembly No. 3
  • Inspect Brake Disc Set No. 4 (B4)
  • Check Low/reverse Brake Return Spring Sub-assembly
  • Check Pack Clearance of Brake Disc Set No. 4 (B4)

 
 
1. ONE-WAY CLUTCH INNER RACE INSTALLATION
1. Apply ATF to the one-way clutch inner race and the one-way clutch assembly No. 3.
2. Install one-way clutch inner race to the one-way clutch assembly No. 3.
noteRotate the one-way clutch inner race clockwise to install it.
 

 
 
2. O-RING (INNER BRAKE PISTON NO. 4) INSTALLATION
1. Apply ATF to 2 new O-rings and the inner brake piston No. 4.
2. Install the 2 O-rings to the inner brake piston No. 4.
O-ring size:
 
Inner diameter (mm)Thickness (mm)
67.352.62
105.552.62
 

 
 
3. O-RING (BRAKE REACTION SLEEVE) INSTALLATION
1. Apply ATF to 2 new O-rings and the brake reaction sleeve.
2. Install the 2 O-rings to the brake reaction sleeve.
O-ring size:
 
Inner diameter (mm)Thickness (mm)
79.12.62
112.72.62
 

 
 
4. BRAKE REACTION SLEEVE WITH INNER BRAKE PISTON NO. 4 INSTALLATION
1.
cautionBe careful not to damage the O-rings.
Install the inner brake piston No. 4 to the brake reaction sleeve.
2.
cautionBe careful not to damage the O-rings and the sliding surface of the transmission case.
With the inner brake piston No. 4 side down, install the brake reaction sleeve and the inner brake piston No. 4 to the transmission case.
 

 
 
5. LOW/REVERSE BRAKE PISTON INSTALLATION
1. Apply ATF to a new O-ring and the low/reverse brake piston.
2. Install the O-ring to the low/reverse brake piston.
O-ring size:
 
Inner diameter (mm)Thickness (mm)
131.62.62
3.
cautionBe careful not to damage the O-ring.
Install the low/reverse brake piston to the transmission case while making sure that the protrusion of the low/reverse brake piston is positioned as shown in the illustration.
 

 
 
6. LOW/REVERSE BRAKE RETURN SPRING SUB-ASSEMBLY INSTALLATION
1. Install the low/reverse brake return spring sub-assembly to the transmission case.
notePlace the low/reverse brake return spring sub-assembly onto the low/reverse brake piston.
2.
cautionBe careful not to shorten the return spring too much.
Set the following special tools on the low/reverse brake return spring sub-assembly, tighten the special tools and compress the low/reverse brake return spring sub-assembly.
  • Spring compressor body (MD998903)
  • Spring compressor (MD998924)
3.
cautionBe careful not to expand the snap ring too much.
Using snap ring pliers, install the snap ring in the groove.
