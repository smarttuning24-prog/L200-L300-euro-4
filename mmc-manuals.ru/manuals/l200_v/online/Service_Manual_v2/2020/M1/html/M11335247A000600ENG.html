Code No. P1497: DPF System Malfunction
 
 
 
 
FUNCTION
The engine-ECU monitors the input signals from sensors to check if the DPF regeneration is executed normally.
 
 
TROUBLE JUDGEMENT
Check Condition
  • On DPF regeneration
Judgement Criterion
  • The DPF regeneration status continues for the predetermined time or more.
 
 
FAIL-SAFE AND BACKUP FUNCTION
The EGR control is stopped (Fully closed).
 
 
PROBABLE CAUSES
  • Failed No. 2 exhaust gas temperature sensor (catalyst temperature)
  • Failed No. 3 exhaust gas temperature sensor (DPF temperature)
  • Failed exhaust differential pressure sensor
  • Catalytic converter inactive
  • The engine oil-containing ash is loaded to the DPF inside.
  • The small injection quantity improperly learned
 
 
DIAGNOSIS PROCEDURE
STEP 1. M.U.T.-III data list
cautionTo prevent damage to the M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting the M.U.T.-III.
Use M.U.T.-III to check the data list (Refer to ).
  • Item No. 118: No. 2 exhaust gas temperature sensor (catalyst temperature)
  • Item No. 119: No. 3 exhaust gas temperature sensor (DPF temperature)
  • Item No. 120: Exhaust differential pressure sensor
Are the check results normal?
Go to Step 2.
Perform the diagnosis code classified check procedure for the sensor that has shown an abnormal data value (Refer to Inspection Chart for Diagnosis Code ).
 
STEP 2. M.U.T.-III data list
Use M.U.T.-III to check the data list (Refer to ).
  • Item No. 208: Short interval DPF regeneration count
OK: 10 or more
Is the short interval DPF regeneration counting 10 or more?
Go to Step 4.
Go to Step 3.
 
STEP 3. Diagnosis code recheck after forcible DPF regeneration execution and small injection quantity learning procedure
After forcible DPF regeneration execution and small injection quantity learning procedure, check again if the diagnosis code is set.
(1) Initialise the DPF-related parts, and then erase the diagnosis code (Refer to GROUP 00 - Precautions before Service, Initialisation Procedure for Learning Value in Common Rail Engine ).
  • Item No. 42: DPF system malfunction service
(2) Carry out the forcible DPF regeneration (Refer to GROUP 17 - Emission Control <DIESEL>, Diesel Particulate Filter (DPF) System, Forcible DPF Regeneration <4N14> or <4N15>).
(3) Carry out the small injection quantity learning procedure (Refer to GROUP 00 - Precautions Before Service, Small Injection Quantity Learning Procedure ).
(4) Turn the ignition switch to "LOCK" (OFF) position.
(5) Engine: Idle operation after warm-up
(6) Use M.U.T.-III to recheck the diagnosis code.
Is diagnosis code No. P0420 or No. P1497 set?
Replace the catalytic converter. When the catalytic converter is replaced, initialise the learning value (Refer to GROUP 00 - Precautions before Service, Initialisation Procedure for Learning Value in Common Rail Engine ). On completion, carry out the forcible DPF regeneration (Refer to GROUP 17 - Emission Control <DIESEL>, Diesel Particulate Filter (DPF) System - Forcible DPF Regeneration <4N14> or <4N15>).
Replace the engine-ECU.
 
STEP 4. M.U.T.-III data list check after forcible DPF regeneration execution and small injection quantity learning procedure
After forcible DPF regeneration execution and small injection quantity learning procedure, check the data list.
(1) Initialise the DPF-related parts, and then erase the diagnosis code (Refer to GROUP 00 - Precautions before Service, Initialisation Procedure for Learning Value in Common Rail Engine ).
  • Item No. 42: DPF system malfunction service
(2) Carry out the forcible DPF regeneration (Refer to GROUP 17 - Emission Control <DIESEL>, Diesel Particulate Filter (DPF) System, Forcible DPF Regeneration <4N14> or <4N15>).
(3) Carry out the small injection quantity learning procedure (Refer to GROUP 00 - Precautions Before Service, Small Injection Quantity Learning Procedure ).
(4) Use M.U.T.-III to check the data list (Refer to ).
  • Item No. 156: Soot loading by differential pressure
OK: 3 g/L or less
Is the check result normal?
Proceed to troubleshooting for black smoke (Refer to Inspection Procedure 13 <4N14> or <4N15>).
Replace the DPF.
