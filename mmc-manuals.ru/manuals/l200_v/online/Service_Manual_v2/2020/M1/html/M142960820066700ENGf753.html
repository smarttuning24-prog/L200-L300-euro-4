Inspection Procedure 10: Multimode keyless entry function does not work at all.
 
 
cautionBefore replacing the ECU, ensure that the power supply circuit, the earth circuit and the communication circuit are normal.
 
 
 
 
OPERATION
The ETACS-ECU may be defective when the electric-folding door mirrors <Vehicles with electric retractable remote controlled door mirrors> works normally. Alternatively, the multimode keyless entry function may be deactivated by using the customisation function.
 
 
PROBABLE CAUSES
  • Malfunction of the keyless operation key
  • Function setting error or no setting with a customisation
  • Malfunction of ETACS-ECU
 
 
DIAGNOSIS PROCEDURE
STEP 1. Check the keyless entry function operation.
Check that the keyless entry system works normally.
Is the check result normal?
Go to Step 2.
Refer to Inspection Procedure 7: keyless entry function does not work .
 
STEP 2. Check the operation of the electric-folding door mirror.
Check that the electric folding door mirrors work normally.
Is the check result normal?
Go to Step 3.
Refer to GROUP 51 - Door Mirror .
 
STEP 3. M.U.T.-III diagnosis code
Check if any diagnosis code is set in the KOS&OSS-ECU.
Is the diagnosis code set?
Refer to diagnosis code chart .
Go to Step 4.
 
STEP 4. M.U.T.-III other system diagnosis code
Check if any diagnosis code is set in the ETACS-ECU.
Is the diagnosis code set?
Perform troubleshooting for the ETACS (Refer to GROUP 54A - ETACS, Troubleshooting ).
Go to Step 5.
 
STEP 5. Check the customisation
Check whether "Multi mode RKE" is set as "D-spec" on the customisation function.
Is it set to other than disabled?
Go to Step 6.
Set "Multi mode RKE" to "D-spec" by using the customisation function (Refer to ).
 
STEP 6. Check of the troubles
Check that the multimode keyless entry function works normally.
Is the check result normal?
Intermittent malfunction is suspected (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points - How to Deal with Intermittent Malfunction ).
Replace ETACS-ECU (Refer to GROUP 54A - ETACS, ETACS-ECU ).
