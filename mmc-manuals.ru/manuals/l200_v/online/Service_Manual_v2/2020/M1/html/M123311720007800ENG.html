DISASSEMBLY
 
 
1. INHIBITOR SWITCH REMOVAL
1. Using a screwdriver, pry off the washer.
2. Remove the nut to remove the washer from the inhibitor switch.
3.
caution
  • Do not touch the inhibitor switch connector terminals.
  • Be careful not to damage the inhibitor switch.
Remove the bolt (seal bolt) to remove the inhibitor switch from the transmission case.
Bolt size:
M8 × 1.25 × 30 mm (Seal bolt)
 

 
 
2. OUTPUT SHAFT SPEED SENSOR REMOVAL
cautionBe careful not to damage the output shaft speed sensor.
1. Remove the bolt and the output shaft speed sensor from the transmission case.
Bolt size:
M6 × 1.0 × 14 mm
2. Remove the O-ring from the output shaft speed sensor.
 

 
 
3. INPUT SHAFT SPEED SENSOR REMOVAL
cautionBe careful not to damage the input shaft speed sensor.
1. Remove the bolt and the input shaft speed sensor from the transmission case.
Bolt size:
M6 × 1.0 × 14 mm
2. Remove the O-ring from the input shaft speed sensor.
 

 
 
4. BREATHER HOSE REMOVAL
1. Remove the breather hose from the breather plug.
2. Remove the breather plug and the O-ring from the transmission case.
 

 
 
5. TRANSMISSION CONTROL SHAFT LEVER LH REMOVAL
1. Remove the nut and spring washer.
2. Remove the transmission control shaft lever LH from the manual valve lever shaft.
 

 
 
6. FILLER PLUG REMOVAL
1. Remove the filler plug from the extension housing. <2WD>
2. Remove the O-ring from the filler plug. <2WD>
3. Remove the filler plug from the transmission case adapter. <4WD>
4. Remove the O-ring from the filler plug. <4WD>
 

 
 
7. UNION REMOVAL
1. Remove the union from the transmission case.
2. Remove the O-ring from the union.
 

 
 
8. A/T HARNESS BRACKET AND A/T CASE BRACKET REMOVAL
Remove the bolts to remove the A/T harness brackets and A/T case bracket from the transmission case.
Bolt size:
M6 × 1.0 × 12 mm
 
 
 
 
REASSEMBLY
 
 
1. A/T HARNESS BRACKET AND A/T CASE BRACKET INSTALLATION
1. Install the A/T harness brackets and A/T case bracket to the transmission case with the bolts.
2. Tighten the A/T harness brackets and A/T case bracket mounting bolts to the specified torque.
Tightening torque: 11 ± 1 N·m
Bolt size:
M6 × 1.0 × 12 mm
 

 
 
2. UNION INSTALLATION
1. Apply ATF to a new O-ring, and then install it onto the union.
O-ring size:
 
Inner diameter (mm)Thickness (mm)
11.681.98
2. Install the union to the transmission case by tightening it to the specified torque.
Tightening torque: 30 ± 5 N·m
 

 
 
3. FILLER PLUG INSTALLATION
1. Apply ATF to a new O-ring, and then install it onto the filler plug.
O-ring size:
 
Inner diameter (mm)Thickness (mm)
15.412.21
2. Install the filler plug to the extension housing by tightening it to the specified torque. <2WD>
Tightening torque: 39 ± 15 N·m
3. Install the filler plug to the transmission case adapter by tightening it to the specified torque. <4WD>
Tightening torque: 39 ± 15 N·m
 

 
 
4. TRANSMISSION CONTROL SHAFT LEVER LH INSTALLATION
1. Install the transmission control shaft lever LH using the spring washer and the nut.
2. Tighten the nut to the specified torque.
Tightening torque: 16 ± 2 N·m
 

 
 
5. BREATHER PLUG INSTALLATION
1. Apply ATF to a new O-ring, and then install it onto the breather plug.
O-ring size:
 
Inner diameter (mm)Thickness (mm)
8.21.9
2. Install the breather hose to the breather plug.
3. Install the breather plug with the breather hose to the transmission case.
 

 
 
6. INPUT SHAFT SPEED SENSOR INSTALLATION
1. Apply ATF to a new O-ring, and then install it onto the input shaft speed sensor.
O-ring size:
 
Inner diameter (mm)Thickness (mm)
11.81.9
2.
cautionBe careful not to damage the input shaft speed sensor.
Install the input shaft speed sensor to the transmission case with the bolt.
3. Tighten the input shaft speed sensor mounting bolt to the specified torque.
Tightening torque: 5.4 ± 1.5 N·m
Bolt size:
M6 × 1.0 × 14 mm
 

 
 
7. OUTPUT SHAFT SPEED SENSOR INSTALLATION
1. Apply ATF to a new O-ring, and then install it onto the output shaft speed sensor.
O-ring size:
 
Inner diameter (mm)Thickness (mm)
11.81.9
2.
cautionBe careful not to damage the output shaft speed sensor.
Install the output shaft speed sensor to the transmission case with the bolt.
3. Tighten the output shaft speed sensor mounting bolt to the specified torque.
Tightening torque: 5.4 ± 1.5 N·m
Bolt size:
M6 × 1.0 × 14 mm
 

 
 
8. INHIBITOR SWITCH INSTALLATION
1. Temporarily install the inhibitor switch with a new bolt (seal bolt).
Bolt size:
M8 × 1.25 × 30 mm (Seal bolt)
noteTighten the bolt after adjusting the inhibitor switch to the "N" position.
2. Install a new washer with the nut.
3. Tighten the nut to the specified torque.
Tightening torque: 6.9 ± 0.9 N·m
4. Using a screwdriver, stake the washer.
5. As shown in the illustration, align the reference line of the inhibitor switch with the manual valve lever shaft surface.
6. Tighten the inhibitor switch mounting bolt to the specified torque.
Tightening torque: 13 ± 2 N·m
7. Adjust the manual valve lever shaft to the "P" position.
