Code No. B1161 Chassis No. mismatch
 
 
caution
  • When diagnosis code No.B1161 is set, be sure to diagnose the CAN bus line.
  • When replacing the ECU, always check that the communication circuit is normal.
 
 
DIAGNOSTIC FUNCTION
When the chassis number registered in KOS&OSS-ECU is different from the chassis number transmitted by engine-ECU on the CAN bus line, KOS&OSS-ECU sets diagnosis code No.B1161.
 
 
JUDGEMENT CRITERIA
Check Conditions
  • When all the following conditions are met.
    1. Power supply mode: "ON"
    2. Chassis number written in KOS&OSS-ECU
Judgement Criterion
  • The chassis number received via CAN communication is different from the chassis number written in KOS&OSS-ECU.
 
 
PROBABLE CAUSES
  • Malfunction of the CAN bus line
  • Writing error of chassis number to engine-ECU (engine-ECU has been interchanged between two vehicles, or chassis number has not been written after engine-ECU replacement)
  • Malfunction of KOS&OSS-ECU
 
 
DIAGNOSIS PROCEDURE
1. M.U.T.-III CAN bus diagnosis
2. Check of chassis number in engine-ECU
3. KOS&OSS-ECU diagnosis code recheck
